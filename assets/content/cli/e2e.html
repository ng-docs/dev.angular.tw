CLI

e2e

Command
ng e2e
ng e

构建并启动 Angular 应用，然后运行端到端测试。

参数

project

要构建的项目名称。可以是应用或库。

Value Typestring

选项

configuration
Aliasc

一个或多个命名构建器配置，以逗号分隔的列表形式，如 angular.json 中的「configurations」部分所指定。构建器使用命名配置来运行给定的目标。有关更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

Value Typestring
help

在控制台显示关于本命令的帮助信息。

Value Typeboolean
