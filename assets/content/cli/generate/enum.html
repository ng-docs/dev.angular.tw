CLI

生成列舉

指令
ng generate enum
ng generate e

在你的專案中建立一個新的列舉。列舉（enumerations）是一種定義一組命名常數的方式，使你的程式碼更具可讀性和可維護性。此原理圖使用指定的名稱和型別生成一個新的列舉。

引數

name

新列舉的名稱。這將用於建立列舉檔案（例如，my-enum.enum.ts）。

值型別string

選項

project

要在其中建立列舉的專案名稱。如果未指定，CLI 將從當前目錄確定專案。

值型別string
type

向檔案名新增自定義類型，允許你建立更具描述性的列舉名稱。例如，如果將型別設定為 status，則檔案名將為 my-enum.status.ts。

值型別string
