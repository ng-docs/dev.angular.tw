CLI

generate pipe

Command
ng generate pipe
ng generate p

在给定的项目中创建一个新的通用管道定义。

参数

name

管道的名称。

Value Typestring

选项

export

声明的 NgModule 导出此管道。

Value TypebooleanDefaultfalse
flat

如果为 true（默认值），则在项目的顶层创建文件。

Value TypebooleanDefaulttrue
module
Aliasm

声明的 NgModule。

Value Typestring
project

项目的名称。

Value Typestring
skip-import

不要将此管道导入到所属的 NgModule 中。

Value TypebooleanDefaultfalse
skip-tests

不要为新的管道创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
standalone

生成的管道是否是独立的。

Value TypebooleanDefaulttrue
