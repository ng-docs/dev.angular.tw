CLI

generate guard

Command
ng generate guard
ng generate g

在给定的项目中生成一个新的、通用的路由守卫定义。

参数

name

新路由守卫的名称。

Value Typestring

选项

flat

如果为 true（默认值），则在当前项目的顶层创建新文件。

Value TypebooleanDefaulttrue
functional

指定是否将守卫生成为一个函数。

Value TypebooleanDefaulttrue
implements
AliasguardType

指定要创建的守卫类型。

Value Typearray
project

项目的名称。

Value Typestring
skip-tests

不要为新的守卫创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
