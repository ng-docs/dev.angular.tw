CLI

generate directive

Command
ng generate directive
ng generate d

在给定项目中创建一个新的通用指令定义。

参数

name

新指令的名称。

Value Typestring

选项

export

声明此指令的 NgModule 导出该指令。

Value TypebooleanDefaultfalse
flat

如果为 true（默认值），则在当前项目的顶层创建新文件。

Value TypebooleanDefaulttrue
module
Aliasm

声明的 NgModule。

Value Typestring
prefix
Aliasp

应用于生成的选择器的前缀。

Value Typestring
project

项目的名称。

Value Typestring
selector

此指令要使用的 HTML 选择器。

Value Typestring
skip-import

不要将此指令导入到所属的 NgModule 中。

Value TypebooleanDefaultfalse
skip-tests

不要为该类创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
standalone

生成的指令是否为独立的。

Value TypebooleanDefaulttrue
