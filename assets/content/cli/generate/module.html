CLI

generate module

Command
ng generate module
ng generate m

在给定的项目中创建一个新的通用 NgModule 定义。

参数

name

NgModule 的名称。

Value Typestring

选项

flat

在当前项目根目录的顶层创建新文件。

Value TypebooleanDefaultfalse
module
Aliasm

声明的 NgModule。

Value Typestring
project

项目的名称。

Value Typestring
route

延迟加载模块的路由路径。当提供时，会在新模块中创建一个组件，并在 --module 选项中提供的模块中声明的 Routes 数组中添加该组件的路由。

Value Typestring
routing

创建一个路由模块。

Value TypebooleanDefaultfalse
routing-scope

新路由模块的作用域。

Value TypestringDefaultChild
