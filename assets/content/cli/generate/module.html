CLI

generate module

指令
ng generate module
ng generate m

在給定的專案中建立一個新的通用 NgModule 定義。

引數

name

NgModule 的名稱。

值型別string

選項

flat

在當前專案根目錄的頂層建立新檔案。

值型別boolean預設值false
module
別名m

宣告它的 NgModule。

值型別string
project

專案的名稱。

值型別string
route

延遲載入模組的路由路徑。當提供時，將在新模組中建立一個元件，並將該組件的路由新增到 --module 選項中提供的模組裡宣告的 Routes 陣列中。

值型別string
routing

建立一個路由模組。

值型別boolean預設值false
routing-scope

新路由模組的作用域。

值型別string預設值Child
