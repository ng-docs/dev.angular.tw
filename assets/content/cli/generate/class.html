CLI

生成類別

指令
ng generate class
ng generate cl

在你的專案中建立一個新類別。類別是 TypeScript 中物件導向程式設計的基礎基礎元素。它們為建立具有屬性和方法的物件提供了一個藍圖。這個原理圖可以幫助你生成一個具有基本結構和可選測試檔案的新類別。

引數

name

新類別的名稱。這將用於建立類別檔案（例如，my-class.ts），如果啟用，還將建立相應的測試檔案 my-class.spec.ts。

值型別string

選項

project

應該在其中新增此類別的專案的名稱。如果未指定，CLI 將從當前目錄確定專案。

值型別string
skip-tests

跳過為新類別生成單元測試檔案 spec.ts。

值型別boolean預設值false
type

向檔案名新增自定義類型，允許你建立更具描述性的類別名稱。例如，如果將型別設定為 helper，則檔案名將為 my-class.helper.ts。

值型別string
