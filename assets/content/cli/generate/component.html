CLI

generate component

Command
ng generate component
ng generate c

在给定项目中创建一个新的通用组件定义。

参数

name

组件的名称。

Value Typestring

选项

change-detection
Aliasc

在新组件中使用的变更检测策略。

Value TypestringDefaultDefault
display-block
Aliasb

指定样式是否包含 :host { display: block; }。

Value TypebooleanDefaultfalse
export

声明此组件的 NgModule 导出该组件。

Value TypebooleanDefaultfalse
flat

在当前项目的顶层创建新文件。

Value TypebooleanDefaultfalse
inline-style
Aliass

在 component.ts 文件中内联包含样式。只能内联包含 CSS 样式。默认情况下，创建一个外部样式文件并在 component.ts 文件中引用。

Value TypebooleanDefaultfalse
inline-template
Aliast

在 component.ts 文件中内联包含模板。默认情况下，创建一个外部模板文件并在 component.ts 文件中引用。

Value TypebooleanDefaultfalse
module
Aliasm

声明的 NgModule。

Value Typestring
prefix
Aliasp

应用于生成的组件选择器的前缀。

Value Typestring
project

项目的名称。

Value Typestring
selector

使用此组件的 HTML 选择器。

Value Typestring
skip-import

不要将此组件导入到所属的 NgModule 中。

Value TypebooleanDefaultfalse
skip-selector

指定组件是否应具有选择器。

Value TypebooleanDefaultfalse
skip-tests

不要为新组件创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
standalone

指定生成的组件是否是独立的。

Value TypebooleanDefaulttrue
style

用于样式文件的文件扩展名或预处理器，或 'none' 以跳过生成样式文件。

Value TypestringDefaultcss
type

向文件名添加一个开发者定义的类型，格式为 "name.type.ts"。

Value TypestringDefaultComponent
view-encapsulation
Aliasv

在新组件中使用的视图封装策略。

Value Typestring
