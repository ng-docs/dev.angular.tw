CLI

生成 interface

指令
ng generate interface
ng generate i

在你的專案中建立一個新的 interface。Interface 定義了 TypeScript 中物件的結構，確保了型別安全和程式碼清晰度。這個原理圖會生成一個具有指定名稱和型別的新 interface。

引數

name

新 interface 的名稱。這將用於建立 interface 檔案 (例如，my-interface.interface.ts)。

值型別string
type

向檔案名新增自定義類型，允許你建立更具描述性的 interface 名稱。例如，如果你將型別設定為 data，則檔案名將為 my-interface.data.ts。

值型別string

選項

prefix

要新增到 interface 名稱的字首。這通常不用於 interface，因為它們沒有像元件或指令那樣的選擇器。

值型別string
project

要在其中建立 interface 的專案名稱。如果未指定，CLI 將從當前目錄確定專案。

值型別string
