CLI

生成 web-worker

指令
ng generate web-worker

在你的專案中建立一個新的 web worker。Web worker 允許你在後台執行 JavaScript 程式碼，透過解除安裝計算密集型任務來提高應用的效能和回應性。這個原理圖為新的 web worker 生成必要的檔案，並提供一個可選的程式碼片段來示範其用法。

引數

name

新 web worker 的名稱。這將用於建立 worker 檔案（例如，my-worker.worker.ts）。

值型別string

選項

project

應該在其中建立 web worker 的專案名稱。如果未指定，CLI 將從當前目錄確定專案。

值型別string
snippet

生成一個程式碼片段，示範如何建立和使用新的 web worker。

值型別boolean預設值true
