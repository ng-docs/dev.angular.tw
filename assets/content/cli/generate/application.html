CLI

generate application

Command
ng generate application
ng generate app

在工作区的 "projects" 子文件夹中生成一个新的基础应用定义。

参数

name

新应用的名称。

Value Typestring

选项

inline-style
Aliass

将样式内联到根 component.ts 文件中。只有 CSS 样式可以内联。默认值为 false，意味着会创建一个外部样式文件，并在根 component.ts 文件中引用。

Value Typeboolean
inline-template
Aliast

将模板内联到根 component.ts 文件中。默认值为 false，意味着会创建一个外部模板文件，并在根 component.ts 文件中引用。

Value Typeboolean
minimal

创建一个没有任何测试框架的基础项目。（仅用于学习目的。）

Value TypebooleanDefaultfalse
prefix
Aliasp

应用于生成的选择器的前缀。

Value TypestringDefaultapp
project-root

新应用的根目录。

Value Typestring
routing

创建一个启用了路由功能的应用。

Value TypebooleanDefaulttrue
skip-install

跳过安装依赖包。

Value TypebooleanDefaultfalse
skip-package-json

不要将依赖项添加到 "package.json" 文件中。

Value TypebooleanDefaultfalse
skip-tests
AliasS

不要为该应用创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
ssr

创建一个启用了服务器端渲染（SSR）和静态站点生成（SSG/预渲染）的应用。

Value TypebooleanDefaultfalse
standalone

创建一个基于独立 API 的应用，不包含 NgModules。

Value TypebooleanDefaulttrue
strict

创建一个具有更严格捆绑包预算设置的应用。

Value TypebooleanDefaulttrue
style

用于样式文件的文件扩展名或预处理器。

Value TypestringDefaultcss
view-encapsulation

在新应用中使用的视图封装策略。

Value Typestring
