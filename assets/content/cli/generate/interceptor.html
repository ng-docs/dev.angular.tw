CLI

generate interceptor

Command
ng generate interceptor

在给定的项目中创建一个新的、通用的拦截器定义。

参数

name

拦截器的名称。

Value Typestring

选项

flat

当为 true（默认）时，在项目的顶层创建文件。

Value TypebooleanDefaulttrue
functional

将拦截器创建为 HttpInterceptorFn。

Value TypebooleanDefaulttrue
project

项目的名称。

Value Typestring
skip-tests

不要为新的拦截器创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
