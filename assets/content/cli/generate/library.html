CLI

generate library

Command
ng generate library
ng generate lib

在当前工作区中创建一个新的通用库项目。

参数

name

库的名称。

Value Typestring

选项

entry-file

创建库的公共 API 文件的路径（相对于工作区根目录）。

Value TypestringDefaultpublic-api
prefix
Aliasp

应用于生成的选择器的前缀。

Value TypestringDefaultlib
project-root

新库的根目录。

Value Typestring
skip-install

不要安装依赖包。

Value TypebooleanDefaultfalse
skip-package-json

不要将依赖项添加到 "package.json" 文件中。

Value TypebooleanDefaultfalse
skip-ts-config

不要更新 "tsconfig.json" 以添加新库的路径映射。路径映射在应用中使用该库时是需要的，但可以在这里禁用以简化开发。

Value TypebooleanDefaultfalse
standalone

基于独立 API 创建一个库，不含 NgModules。

Value TypebooleanDefaulttrue
