CLI

generate resolver

Command
ng generate resolver
ng generate r

在指定的项目中生成一个新的通用求解器定义。

参数

name

新求解器的名称。

Value Typestring

选项

flat

如果为 true（默认值），则在当前项目的顶层创建新文件。

Value TypebooleanDefaulttrue
functional

将求解器创建为 ResolveFn。

Value TypebooleanDefaulttrue
project

项目的名称。

Value Typestring
skip-tests

不要为新的求解器创建 "spec.ts" 测试文件。

Value TypebooleanDefaultfalse
