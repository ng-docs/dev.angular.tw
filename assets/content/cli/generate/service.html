CLI

生成服務

指令
ng generate service
ng generate s

在你的專案中建立一個新的服務。服務用於封裝可複用的邏輯，例如資料訪問、API 呼叫或工具函式。這個原理圖簡化了生成包含必要檔案和樣板程式碼的新服務的流程。

引數

name

新服務的名稱。這將用於建立服務的類別檔案和 spec 檔案（例如，my-service.service.ts 和 my-service.service.spec.ts）。

值型別string

選項

flat

在專案頂層或給定路徑下建立檔案。如果設定為 false，將建立一個以服務名稱命名的新檔案夾來包含這些檔案。

值型別boolean預設值true
project

應該新增服務的專案名稱。如果未指定，CLI 將從當前目錄確定專案。

值型別string
skip-tests

跳過為服務生成單元測試檔案 spec.ts。

值型別boolean預設值false
