CLI

run

Command
ng run

Architect 是 CLI 用于根据提供的配置执行复杂任务（如编译）的工具。 CLI 命令运行 Architect 目标，如 build、 serve、 test 和 lint。 每个命名目标都有一个默认配置，由 options 对象指定，并且在 configurations 对象中有一组可选的命名备用配置。

例如，新生成的应用程序的 serve目标具有预定义的备用配置，名称为 production。

你可以在 angular.json文件的 architect部分中定义新的目标及其配置选项，然后可以使用 ng run命令从命令行运行它们。

参数

target

提供的 Architect 目标以以下格式运行： project:target[:configuration]。
