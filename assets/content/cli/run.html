CLI

run

指令
ng run

Architect 是 CLI 使用的工具，用於根據提供的設定執行諸如編譯等複雜任務。 CLI 指令執行 Architect 目標，例如 build、serve、test 和 lint。 每個具名目標都有一個預設設定（由 options 物件指定），以及 configurations 物件中一組可選的具名備用設定。

例如，新生成的應用的 serve 目標具有一個名為 production 的預定義備用設定。

你可以在 angular.json 檔案的 architect 部分定義新的目標及其設定選項，然後可以使用 ng run 指令從命令行執行它們。

引數

target

要執行的 Architect 目標，以下列格式提供：project:target[:configuration]。

值型別string

選項

help

在控制台中顯示此指令的幫助訊息。

值型別boolean
