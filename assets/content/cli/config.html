設定

指令
ng config

工作區在頂層有一個 CLI 設定檔案 angular.json。
projects 物件包含工作區中每個專案的設定物件。

你可以直接在程式碼編輯器中編輯設定，
也可以使用此指令在指令行中間接編輯。

可設定的屬性名稱與指令選項名稱對應，
不同之處在於設定檔案中，所有名稱都必須使用 camelCase，
而在指令行選項中可以使用 dash-case。

更多詳情，請參閱工作區設定。

有關 CLI 使用情況分析的設定，請參閱 ng analytics。

引數

json-path

要設定或查詢的設定鍵，以 JSON 路徑格式表示。例如：『a[3].foo.bar[2]』。如果未提供新值，則回傳此鍵的當前值。

值型別string
value

如果提供，則為給定設定鍵的新值。

值型別string

選項

global
別名g

訪問呼叫者主目錄中的全域設定。

值型別boolean預設值false
help

在控制台中顯示此指令的幫助訊息。

值型別boolean
