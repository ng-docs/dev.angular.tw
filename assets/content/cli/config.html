CLI

config

Command
ng config

一个工作区有一个顶层的 CLI 配置文件 angular.json。 projects 对象包含工作区中每个项目的配置对象。

你可以直接在代码编辑器中编辑配置，或者在命令行中使用此命令间接编辑配置。

可配置的属性名称与命令选项名称相匹配，只是配置文件中的所有名称必须使用 camelCase，而命令行选项可以使用 dash-case。

有关详细信息，请参阅 工作区配置。

关于 CLI 使用分析的配置，请参阅 ng analytics。

参数

json-path

要设置或查询的配置键，采用 JSON 路径格式。例如："a[3].foo.bar[2]"。如果没有提供新值，则返回此键的当前值。

Value Typestring
value

如果提供了，则为给定配置键的新值。
