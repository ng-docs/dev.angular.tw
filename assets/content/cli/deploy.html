CLI

deploy

Command
ng deploy

该命令接受一个可选的项目名称，如 angular.json 工作区配置文件的 projects 部分中所指定的。 当未提供项目名称时，执行默认项目的 deploy 构建器。

要使用 ng deploy 命令，请使用 ng add 添加一个实现部署能力的包到你喜欢的平台。 添加包会自动更新你的工作区配置，添加一个部署 CLI 构建器。 例如：

          
            "projects": {  "my-project": {    ...    "architect": {      ...      "deploy": {        "builder": "@angular/fire:deploy",        "options": {}      }    }  }}

参数

project

要构建的项目名称。可以是应用或库。

Value Typestring

选项

configuration
Aliasc

一个或多个命名构建器配置，以逗号分隔的列表形式，如 angular.json 中的「configurations」部分所指定。构建器使用命名配置来运行给定的目标。有关更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

Value Typestring
help

在控制台显示关于本命令的帮助信息。

Value Typeboolean
