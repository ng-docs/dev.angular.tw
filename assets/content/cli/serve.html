CLI

serve

Command
ng serve
ng dev
ng s

构建并提供你的应用程序，在文件更改时重新构建。

参数

project

要构建的项目名称。可以是应用或库。

Value Typestring

选项

allowed-hosts

允许访问开发服务器的主机列表。当使用“应用程序”或其他基于 esbuild 的构建器时，此选项无效。

Value Typearray
@deprecated
browser-target

浏览器构建器目标以 project:target[:configuration]格式提供。你还可以将多个配置名称作为逗号分隔的列表传入。示例： project:target:production,staging。

Value Typestring
build-target

构建构建器目标以 project:target[:configuration]格式提供。你还可以将多个配置名称作为逗号分隔的列表传入。示例： project:target:production,staging。

Value Typestring
configuration
Aliasc

一个或多个命名构建器配置，以逗号分隔的列表形式，如 angular.json 中的「configurations」部分所指定。构建器使用命名配置来运行给定的目标。有关更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

Value Typestring
disable-host-check

不验证连接的客户端是否属于允许的主机。当使用“应用程序”或其他基于 esbuild 的构建器时，此选项无效。

Value TypebooleanDefaultfalse
force-esbuild

强制开发服务器在构建时使用“browser-esbuild”构建器。这是 esbuild 构建系统的开发者预览选项。

Value TypebooleanDefaultfalse
help

在控制台显示关于本命令的帮助信息。

Value Typeboolean
hmr

启用热模块替换。

Value TypebooleanDefaultfalse
host

要监听的主机。

Value TypestringDefaultlocalhost
live-reload

是否在更改时重新加载页面，使用实时重新加载。

Value TypebooleanDefaulttrue
open
Aliaso

在默认浏览器中打开URL。

Value TypebooleanDefaultfalse
poll

启用并定义文件监视的轮询时间周期（以毫秒为单位）。

Value Typenumber
port

要监听的端口。

Value TypenumberDefault4200
prebundle

启用和控制 Vite-based 开发服务器的预捆绑功能。要启用预捆绑，还必须启用 Angular CLI 缓存。当使用“browser”或其他基于 Webpack 的构建器时，此选项无效。

Value Typeboolean
proxy-config

代理配置文件。有关更多信息，请参阅 https://angular.dev/tools/cli/serve#proxying-to-a-backend-server。

Value Typestring
public-host

浏览器客户端（或启用时的实时重新加载客户端）应使用的连接到开发服务器的URL。用于复杂的开发服务器设置，例如具有反向代理的设置。当使用“应用程序”或其他基于 esbuild 的构建器时，此选项无效。

Value Typestring
serve-path

应用程序将提供的路径名。

Value Typestring
ssl

使用 HTTPS 进行提供。

Value TypebooleanDefaultfalse
ssl-cert

用于提供 HTTPS 服务的 SSL 证书。

Value Typestring
ssl-key

用于提供 HTTPS 服务的 SSL 密钥。

Value Typestring
verbose

在输出日志中添加更多详细信息。

Value Typeboolean
watch

在更改时重新构建。

Value TypebooleanDefaulttrue
