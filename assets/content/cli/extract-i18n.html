CLI

extract-i18n

Command
ng extract-i18n

从源代码中提取 i18n 消息。

参数

project

要构建的项目名称。可以是应用或库。

Value Typestring

选项

@deprecated
browser-target

一个浏览器构建器目标，用于提取 i18n 消息，格式为 project:target[:configuration]。你也可以传递多个配置名称作为逗号分隔的列表。例如： project:target:production,staging。

Value Typestring
build-target

一个构建器目标，用于提取 i18n 消息，格式为 project:target[:configuration]。你也可以传递多个配置名称作为逗号分隔的列表。例如： project:target:production,staging。

Value Typestring
configuration
Aliasc

一个或多个命名构建器配置，以逗号分隔的列表形式，如 angular.json 中的「configurations」部分所指定。构建器使用命名配置来运行给定的目标。有关更多信息，请参阅 https://angular.dev/reference/configs/workspace-config#alternate-build-configurations。

Value Typestring
format

生成文件的输出格式。

Value TypestringDefaultxlf
help

Value Typeboolean
out-file

Value Typestring

Value Typestring
