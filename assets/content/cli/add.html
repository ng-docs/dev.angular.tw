CLI

add

Command
ng add

将某个已发布库的 npm 包添加到你的工作区，并根据该库的原理图配置当前工作目录中的项目。例如，添加 @angular/pwa 会为你的项目配置 PWA 支持：

          
            ng add @angular/pwa

参数

collection

要添加的包。
