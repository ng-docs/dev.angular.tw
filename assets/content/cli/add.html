CLI

add

指令
ng add

將已發布函式庫的 npm 套件新增到你的工作區，並設定當前工作目錄中的專案以使用該函式庫，具體設定由該函式庫的原理圖指定。 例如，新增 @angular/pwa 會將你的專案設定為支援 PWA：

          
ng add @angular/pwa

引數

collection

要新增的套件。

值型別string

選項

defaults

停用帶預設值的選項的互動式輸入提示。

值型別boolean預設值false
dry-run
別名d

貫穿執行並報告活動，但不寫出結果。

值型別boolean預設值false
force

強制覆蓋現有檔案。

值型別boolean預設值false
help

在控制台中顯示此指令的幫助訊息。

值型別boolean
interactive

啟用互動式輸入提示。

值型別boolean預設值true
registry

要使用的 NPM 登錄檔。

值型別string
skip-confirmation

跳過在安裝和執行套件之前顯示確認提示。請確保套件名稱在使用此選項之前是正確的。

值型別boolean預設值false
verbose

顯示有關執行期間內部操作的其他詳細訊息。

值型別boolean預設值false
