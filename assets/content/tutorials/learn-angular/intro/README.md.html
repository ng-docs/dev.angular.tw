如何使用本教學

您需要具備 HTML、CSS 和 JavaScript 的基本知識，才能了解 Angular。

每個步驟都代表 Angular 中的一個概念。您可以執行其中一個，或所有步驟。

如果您受困，請按一下頂端的「顯示答案」。
