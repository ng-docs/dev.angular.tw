组件中的控制流 - @if

edit

在应用开发中，为用户决定在屏幕上显示什么是一个常见任务。许多时候，决定是通过条件以编程方式做出的。

为了在模板中表示条件显示，Angular 使用 @if 模板语法。

在这个活动中，你将学习如何在模板中使用条件语句。

在模板中启用条件显示元素的语法是 @if。

下面是一个在组件中使用 @if 语法的示例：

      
@Component({
  ...
  template: `
    @if (isLoggedIn) {
      <p>Welcome back, Friend!</p>
    }
  `,
})
class AppComponent {
  isLoggedIn = true;
}

需要注意两点：

  • if 前面有一个 @ 前缀，因为它是一种特殊的语法，称为 Angular 模板语法
  • 对于使用 v16 及以下版本的应用，请参考 Angular NgIf 文档 了解更多信息。

  1. 创建一个名为 isServerRunning 的属性

    在 AppComponent 类中，添加一个名为 isServerRunning 的 boolean 属性，并将初始值设置为 true。

  2. 在模板中使用 @if

    更新模板，如果 isServerRunning 的值为 true，则显示消息 Yes, the server is running。

  3. 在模板中使用 @else

    现在 Angular 支持使用 @else 语法来定义 else 情况。更新模板，在 else 情况下显示消息 No, the server is not running。

    例子如下：

          
    template: `
      @if (isServerRunning) { ... }
      @else { ... }
    `;

    添加你的代码以补全缺少的标记。

这种类型的功能称为条件控制流。接下来你将学习如何在模板中重复某些条目。
