元件中的控制流 - @if

edit

決定在螢幕上為使用者顯示什麼是應用開發中的常見任務。很多時候，這個決定是透過程式設計方式使用條件來做出的。

為了在範本中表達條件顯示，Angular 使用 @if 範本語法。

在本活動中，你將學習如何在範本中使用條件。

在範本中啟用元素條件顯示的語法是 @if。

下面是如何在元件中使用 @if 語法的範例：

      
@Component({  ...  template: `    @if (isLoggedIn) {      <p>Welcome back, Friend!</p>    }  `,})class AppComponent {  isLoggedIn = true;}

有兩件事需要注意：

  • if 有一個 @ 字首，因為它是一種特殊的語法，稱為 Angular 範本語法
  • 對於使用 v16 及更早版本的應用，請參閱 關於 NgIf 的 Angular 文件 以獲取更多訊息。

  1. 建立一個名為 isServerRunning 的屬性

    在 AppComponent 類別中，新增一個名為 isServerRunning 的 boolean 屬性（Property），並將初始值設定為 true。

  2. 在範本中使用 @if

    更新範本以顯示訊息 Yes, the server is running （如果 isServerRunning 的值為 true）。

  3. 在範本中使用 @else

    現在，Angular 支援使用原生範本語法透過 @else 語法定義 else 分支。更新範本以顯示訊息 No, the server is not running 作為 else 分支。

    這是一個範例：

          
    template: `  @if (isServerRunning) { ... }  @else { ... }`;

    新增你的程式碼以填充缺失的標記。

這種型別的功能稱為條件控制流。接下來，你將學習如何在範本中重複條目。
