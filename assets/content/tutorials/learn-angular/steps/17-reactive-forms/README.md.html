回應式表單

edit

當你想以程式設計方式管理表單，而不是僅僅依賴樣板時，回應式表單就是答案。

在本次活動中，你將學習如何建立回應式表單。

  1. 匯入 ReactiveForms 模組

    在 app.component.ts 中，從 @angular/forms 匯入 ReactiveFormsModule 並將其新增到元件的 imports 陣列中。

          
    import { ReactiveFormsModule } from '@angular/forms';@Component({  selector: 'app-root',  template: `    <form>      <label>Name        <input type="text" />      </label>      <label>Email        <input type="email" />      </label>      <button type="submit">Submit</button>    </form>  `,  imports: [ReactiveFormsModule],})

  2. 使用 FormControl 建立 FormGroup 物件

    回應式表單使用 FormControl 類來表示表單控制元件（例如，輸入框）。Angular 提供了 FormGroup 類，用作表單控制元件的分組，形成一個有用的物件，使開發者可以更方便地處理大型表單。

    將 FormControl 和 FormGroup 新增到從 @angular/forms 的匯入中，以便你可以為每個表單建立一個 FormGroup，並將屬性 name 和 email 作為 FormControl。

          
    import {ReactiveFormsModule, FormControl, FormGroup } from '@angular/forms';...export class AppComponent {  profileForm = new FormGroup({    name: new FormControl(''),    email: new FormControl(''),  });}

  3. 將 FormGroup 和 FormControl 連結到表單

    每個 FormGroup 都應該使用 [formGroup] 指令附著到一個表單。

    此外，每個 FormControl 都可以使用 formControlName 指令附著，並賦值給相應的屬性（Property）。使用以下表單程式碼更新樣板：

          
    <form [formGroup]="profileForm">  <label>    Name    <input type="text" formControlName="name" />  </label>  <label>    Email    <input type="email" formControlName="email" />  </label>  <button type="submit">Submit</button></form>

  4. 處理表單更新

    當你想從 FormGroup 訪問資料時，可以透過訪問 FormGroup 的值來完成。更新 template 以顯示錶單值：

          
    ...<h2>Profile Form</h2><p>Name: {{ profileForm.value.name }}</p><p>Email: {{ profileForm.value.email }}</p>

  5. 訪問 FormGroup 值

    向元件類別新增一個名為 handleSubmit 的新方法，稍後你將使用它來處理表單送出。 此方法將顯示錶單中的值，你可以從 FormGroup 訪問這些值。

    在元件類別中，新增 handleSubmit() 方法來處理表單送出。

          
    handleSubmit() {  alert(    this.profileForm.value.name + ' | ' + this.profileForm.value.email  );}

  6. 將 ngSubmit 新增到表單

    你已經可以訪問表單值了，現在是時候處理送出事件並使用 handleSubmit 方法了。 Angular 為此特定目的提供了一個名為 ngSubmit 的事件處理程式。更新 form 元素，以便在送出表單時調用 handleSubmit 方法。

          
    <form  [formGroup]="profileForm"  (ngSubmit)="handleSubmit()">

就像這樣，你就知道如何在 Angular 中使用回應式表單了。
