基於建構式函式的依賴注入

edit

在之前的活動中，你使用 inject() 函式來提供資源，也就是將資源『提供』給你的元件。inject() 函式是一種模式，但瞭解還有另一種注入資源的模式也很有用，這種模式叫做基於建構式函式的依賴注入。

你可以將資源指定為元件的 constructor 函式的引數。Angular 將使這些資源可用於你的元件。

在本活動中，你將學習如何使用基於建構式函式的依賴注入。

要將服務或其他可注入的資源注入到你的元件中，請使用以下語法：

      
@Component({...})class PetCarDashboardComponent {    constructor(private petCareService: PetCareService) {        ...    }}

這裡有幾件事需要注意：

  • 使用 private 關鍵字
  • petCareService 變成了你可以在類別中使用的屬性（Property）
  • PetCareService 類是注入的類

好的，現在你來試試看：

  1. 更新程式碼以使用基於建構式函式的依賴注入

    在 app.component.ts 中，更新建構式函式程式碼以對應以下程式碼：

    提示： 請記住，如果你遇到困難，請參考此活動頁面上的範例。

          
    constructor(private carService: CarService) {    this.display = this.carService.getCars().join(' ⭐️ ');}

祝賀你完成此活動。範例程式碼的工作方式與使用 inject 函式相同。雖然這兩種方法在很大程度上是相同的，但仍有一些細微的差異超出了本教程的範圍。
