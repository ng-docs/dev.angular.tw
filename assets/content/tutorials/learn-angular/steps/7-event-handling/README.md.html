事件處理

edit

事件處理讓 Web 應用具備了互動屬性。它使你作為開發人員能夠回應使用者的操作，例如按鈕按下、表單送出等等。

在本次活動中，你將學習如何新增事件處理器。

在 Angular 中，你要使用圓括號語法 () 繫結到事件。在給定的元素上，用圓括號包裹你要繫結的事件，並設定一個事件處理器。考慮一下這個 button 範例：

      
@Component({    ...    template: `<button (click)="greet()">`})class AppComponent {    greet() {        console.log('Hello, there 👋');    }}

在這個例子中，每次單擊按鈕時，greet() 函式都會執行。請注意，greet() 語法包括了尾部的圓括號。

好了，輪到你來嘗試一下了：

  1. 新增一個事件處理器

    在 AppComponent 類別中新增 onMouseOver 事件處理器函式。使用以下程式碼作為實現：

          
    onMouseOver() {    this.message = 'Way to go 🚀';}

  2. 繫結到範本事件

    更新 app.component.ts 中的範本程式碼，以繫結到 section 元素的 mouseover 事件。

          
    <section (mouseover)="onMouseOver()">

透過程式碼中的幾個步驟，你已經在 Angular 中建立了你的第一個事件處理器。看來你在這方面做得相當不錯，請繼續保持。
