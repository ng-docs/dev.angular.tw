事件处理

edit

事件处理使 web 应用拥有交互特性。它让开发者能够响应用户操作，比如按钮按下、表单提交等。

在这个活动中，你将学习如何添加事件处理器。

在 Angular 中，你使用圆括号语法 () 绑定事件。在给定的元素上，将你想绑定的事件包裹在圆括号中并设置事件处理器。考虑以下 button 示例：

      
@Component({
    ...
    template: `<button (click)="greet()">`
})
class AppComponent {
    greet() {
        console.log('Hello, there 👋');
    }
}

在这个示例中，每次点击按钮时都会运行 greet() 函数。注意 greet() 语法包含尾随的圆括号。

好了，现在轮到你试一试了：

  1. 添加事件处理器

    在 AppComponent 类中添加 onMouseOver 事件处理器函数。使用以下代码作为实现：

          
    onMouseOver() {
        this.message = 'Way to go 🚀';
    }

  2. 绑定模板事件

    更新 app.component.ts 中的模板代码，将 section 元素的 mouseover 事件绑定上。

          
    <section (mouseover)="onMouseOver()">

通过在代码中执行几个步骤，你已经创建了你的第一个 Angular 事件处理器。看起来你做得很好，继续保持。
