定义一个路由

edit

既然你已经设置了应用使用 Angular 路由器，你需要定义一些路由。

在这个活动中，你将学习如何为你的应用添加和配置路由。

  1. 在 app.routes.ts 中定义一个路由

    在你的应用中，有两个页面需要显示：(1) 首页 和 (2) 用户页面。

    要定义一个路由，在 app.routes.ts 文件中的 routes 数组中添加一个包含以下内容的路由对象：

    • 该路由的 path（默认从根路径开始，即 /）
    • 你想让该路由显示的 component
          
    import {Routes} from '@angular/router';
    import {HomeComponent} from './home/home.component';
    export const routes: Routes = [
      {
        path: '',
        component: HomeComponent,
      },
    ];

    上面的代码是将 HomeComponent 添加为路由的一个示例。现在继续在 playground 中实现这个以及 UserComponent。

    使用 'user' 作为 UserComponent 的路径。

  2. 为路由定义添加标题

    除了正确定义路由，Angular 路由器还允许你通过为每个路由添加 title 属性来在用户导航时设置页面标题。

    在 app.routes.ts 中，为默认路由（ path: ''）和 user 路由添加 title 属性。以下是一个示例：

          
    import {Routes} from '@angular/router';
    import {HomeComponent} from './home/home.component';
    export const routes: Routes = [
    {
    path: '',
    title: 'App Home Page',
    component: HomeComponent,
    },
    ];
