定義路由

edit

既然你已經建立好應用來使用 Angular 路由，現在你需要定義路由。

在本活動中，你將學習如何在你的應用中新增和設定路由。

  1. 在 app.routes.ts 中定義路由

    在你的應用中，有兩個頁面要顯示：（1）首頁和（2）使用者頁面。

    要定義路由，請將一個路由物件新增到 app.routes.ts 中的 routes 陣列，其中包含：

    • 路由的 path（它會自動從根路徑（即 /）開始）
    • 你希望路由顯示的 component
          
    import {Routes} from '@angular/router';import {HomeComponent} from './home/home.component';export const routes: Routes = [  {    path: '',    component: HomeComponent,  },];

    上面的程式碼是如何將 HomeComponent 新增為路由的範例。現在繼續在 playground 中實現它以及 UserComponent。

    為 UserComponent 的路徑使用 'user'。

  2. 向路由定義新增標題

    除了正確定義路由之外，Angular 路由還允許你透過向每個路由新增 title 屬性，在使用者導向時設定頁面標題。

    在 app.routes.ts 中，將 title 屬性新增到預設路由 (path: '') 和 user 路由。 這是一個例子：

          
    import {Routes} from '@angular/router';import {HomeComponent} from './home/home.component';export const routes: Routes = [  {    path: '',    title: 'App Home Page',    component: HomeComponent,  },];

在本活動中，你已經學習瞭如何在 Angular 應用中定義和設定路由。做得好。🙌

在你的應用中完全啟用路由的旅程即將完成，繼續前進。
