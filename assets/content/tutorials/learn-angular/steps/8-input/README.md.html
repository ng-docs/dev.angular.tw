使用 @Input 进行组件通信

edit

有时，应用开发需要你向组件传递数据。这些数据可以用来定制组件，或将信息从父组件传递到子组件。

Angular 使用一个称为 Input 的概念。这类似于其他框架中的 props。要创建一个 Input 属性，使用 @Input 装饰器。

在这个活动中，你将学习如何使用 @Input 装饰器向组件传递信息。

要创建一个 Input 属性，将 @Input 装饰器添加到组件类的某个属性上：

user.component.ts

      
class UserComponent {
  @Input() occupation = '';
}

当你准备好通过 Input 传入一个值时，可以在模板中使用属性语法设置值。以下是示例：

app.component.ts

      
@Component({
  ...
  template: `<app-user occupation="Angular Developer"><app-user/>`
})
class AppComponent {}

确保你在 UserComponent 中绑定属性 occupation。

user.component.ts

      
@Component({
  ...
  template: `<p>The user's name is {{occupation}}</p>`
})

  1. 定义一个 @Input 属性

    更新 user.component.ts 中的代码，在 UserComponent 上定义一个名为 name 的 Input 属性。目前，将初始值设置为 空字符串。务必更新模板，在句子的末尾插值 name 属性。

  2. 向 @Input 属性传递一个值

    更新 app.component.ts 中的代码，传入值为 "Simran" 的 name 属性。

    当代码成功更新后，应用将显示 The user's name is Simran。
