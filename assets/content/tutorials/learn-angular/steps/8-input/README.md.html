使用 @Input 的元件通訊

edit

有時應用開發需要你將資料傳送到元件中。這些資料可以用來定製元件，或者從父元件向子元件傳送訊息。

Angular 使用一個叫做 Input 的概念。這類似於其他框架中的 props。要建立一個 Input 屬性（Property），請使用 @Input 裝飾器。

在本活動中，你將學習如何使用 @Input 裝飾器向元件傳送訊息。

要建立一個 Input 屬性（Property），請將 @Input 裝飾器新增到元件類別的屬性（property）上：

user.component.ts

      
class UserComponent {  @Input() occupation = '';}

當你準備好透過 Input 傳入一個值時，可以使用 Attribute 語法在樣板中設定值。這是一個範例：

app.component.ts

      
@Component({  ...  template: `<app-user occupation="Angular Developer"></app-user>`})class AppComponent {}

請確保在你的 UserComponent 中繫結屬性（property） occupation。

user.component.ts

      
@Component({  ...  template: `<p>The user's occupation is {{occupation}}</p>`})

  1. 定義一個 @Input 屬性（Property）

    更新 user.component.ts 中的程式碼，在 UserComponent 上定義一個名為 name 的 Input 屬性（Property）。現在，將初始值設定為 empty string。請務必更新樣板以在句子末尾內嵌 name 屬性（property）。

  2. 向 @Input 屬性（Property）傳遞一個值

    更新 app.component.ts 中的程式碼，傳入 name 屬性（property），其值為 "Simran"。

    當代碼成功更新後，應用將顯示 The user's name is Simran。
