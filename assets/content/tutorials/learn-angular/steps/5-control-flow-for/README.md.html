组件中的控制流 - @for

edit

在构建 Web 应用时，你常常需要重复某些代码多次 —— 例如，给定一个名字数组，你可能希望在每个 <p> 标签中显示每个名字。

在此活动中，你将学习如何使用 @for 在模板中重复元素。

启用在模板中重复元素的语法是 @for。

以下是如何在组件中使用 @for 语法的示例：

      
@Component({
  ...
  template: `
    @for (os of operatingSystems; track os.id) {
      {{ os.name }}
    }
  `,
})
export class AppComponent {
  operatingSystems = [{id: 'win', name: 'Windows'}, {id: 'osx', name: 'MacOS'}, {id: 'linux', name: 'Linux'}];
}

需要注意两点：

  • @ 前缀在 for 前是因为它是一种特殊语法，称为 Angular 模板语法。
  • 对于使用 v16 及更早版本的应用，请参阅 Angular 文档中的 NgFor

  1. 添加 users 属性

    在 AppComponent 类中，添加一个名为 users 的属性，其中包含用户及其姓名。

          
    [{id: 0, name: 'Sarah'}, {id: 1, name: 'Amy'}, {id: 2, name: 'Rachel'}, {id: 3, name: 'Jessica'}, {id: 4, name: 'Poornima'}]

  2. 更新模板

    更新模板，使用 @for 模板语法在 p 元素中显示每个用户名。

          
    @for (user of users; track user.id) {
      <p>{{ user.name }}</p>
    }

    注意：需要使用 track，你可以使用 id 或其他唯一标识符。

这种功能称为控制流。接下来，你将学习如何自定义和与组件通信 —— 顺便说一句，你做得很棒。
