元件中的控制流 - @for

edit

通常在建立 Web 應用時，你需要重複一段程式碼特定的次數 - 例如，給定一個名稱陣列，你可能希望在 <p> 標籤中顯示每個名稱。

在本活動中，你將學習如何使用 @for 在樣板中重複條目。

在樣板中啟用條目重複的語法是 @for。

這是一個關於如何在元件中使用 @for 語法的範例：

      
@Component({  ...  template: `    @for (os of operatingSystems; track os.id) {      {{ os.name }}    }  `,})export class AppComponent {  operatingSystems = [{id: 'win', name: 'Windows'}, {id: 'osx', name: 'MacOS'}, {id: 'linux', name: 'Linux'}];}

注意看以下兩點：

  • for 帶有 @ 字首，因為它是一種稱為 Angular 樣板語法 的特殊語法
  • 對於使用 v16 及更老版本的應用，請參考 關於 NgFor 的 Angular 文件

  1. 新增 users 屬性（Property）

    在 AppComponent 類別中，新增一個名為 users 的屬性（Property），其中包含使用者及其姓名。

          
    [{id: 0, name: 'Sarah'}, {id: 1, name: 'Amy'}, {id: 2, name: 'Rachel'}, {id: 3, name: 'Jessica'}, {id: 4, name: 'Poornima'}]

  2. 更新樣板

    更新樣板以使用 @for 樣板語法在 p 元素中顯示每個使用者名。

          
    @for (user of users; track user.id) {  <p>{{ user.name }}</p>}

    注意：必須使用 track，你可以使用 id 或其他唯一識別符號。

這種型別的功能稱為控制流。接下來，你將學習如何自定義元件以及與元件通訊 - 順便說一句，到目前為止你做得非常棒。
