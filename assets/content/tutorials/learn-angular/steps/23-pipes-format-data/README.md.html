使用管道格式化数据

edit

你可以通过配置管道进一步提升对管道的使用。可以通过传递选项来配置管道。

在此活动中，你将使用某些管道和管道参数。

要向管道传递参数，请使用 : 语法，后跟参数值。这里有一个示例：

      
template: `{{ date | date:'medium' }}`;

输出结果为 Jun 15, 2015, 9:43:11 PM。

是时候自定义一些管道输出了：

  1. 使用 DecimalPipe 格式化数字

    在 app.component.ts 中，更新模板以包含 decimal 管道的参数。

          
    template: `
        ...
        <li>Number with "decimal" {{ num | number:"3.2-2" }}</li>
    `

    注意：那是什么格式？ DecimalPipe 的参数称为 digitsInfo，此参数使用的格式为： {minIntegerDigits}.{minFractionDigits}-{maxFractionDigits}

  2. 使用 DatePipe 格式化日期

    现在，更新模板以使用 date 管道。

          
    template: `
        ...
        <li>Date with "date" {{ birthday | date: 'medium' }}</li>
    `

    为了更有趣，尝试一些不同的 date 参数。更多信息可以在 Angular 文档 中找到。

  3. 使用 CurrencyPipe 格式化货币

    你的最后一个任务是，更新模板以使用 currency 管道。

          
    template: `
        ...
        <li>Currency with "currency" {{ cost | currency }}</li>
    `

    你也可以尝试不同的 currency 参数。更多信息可以在 Angular 文档 中找到。

使用管道做得很棒。你已经取得了一些不错的进展。

还有更多内置管道可以在你的应用中使用。你可以在 Angular 文档 中找到列表。

如果内置管道不能满足你的需求，你也可以创建自定义管道。查看下一课以了解更多信息。
