使用通道格式化資料

edit

你可以透過設定通道來進一步使用它們。可以透過向通道傳遞選項來設定通道。

在本活動中，你將使用一些通道和通道引數。

要向通道傳遞引數，請使用 : 語法，後跟引數值。這是一個範例：

      
template: `{{ date | date:'medium' }}`;

輸出是 Jun 15, 2015, 9:43:11 PM。

是時候自定義一些通道輸出了：

  1. 使用 DecimalPipe 格式化數字

    在 app.component.ts 中，更新樣板以包含 decimal 通道的引數。

          
    template: `    ...    <li>Number with "decimal" {{ num | number:"3.2-2" }}</li>`

    注意：那是什麼格式？ DecimalPipe 的引數名為 digitsInfo，此引數使用以下格式：{minIntegerDigits}.{minFractionDigits}-{maxFractionDigits}

  2. 使用 DatePipe 格式化日期

    現在，更新樣板以使用 date 通道。

          
    template: `    ...    <li>Date with "date" {{ birthday | date: 'medium' }}</li>`

    為了增加樂趣，請嘗試 date 的一些不同引數。更多訊息可以在 Angular 文件 中找到。

  3. 使用 CurrencyPipe 格式化貨幣

    對於你的最後一個任務，更新樣板以使用 currency 通道。

          
    template: `    ...    <li>Currency with "currency" {{ cost | currency }}</li>`

    你也可以嘗試 currency 的不同引數。更多訊息可以在 Angular 文件 中找到。

通道用得真棒。到目前為止，你已經取得了很大的進步。

還有更多內建通道可以在你的應用中使用。你可以在 Angular 文件 中找到清單。

如果內建通道不能滿足你的需求，你還可以建立一個自定義通道。檢視下一課以瞭解更多訊息。
