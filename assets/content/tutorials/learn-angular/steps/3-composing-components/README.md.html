组合组件

edit

你已经学会了更新组件模板、组件逻辑和组件样式，但如何在你的应用中使用组件呢？

组件配置的 selector 属性为你提供了在另一个模板中引用组件时使用的名称。你像使用 HTML 标签一样使用 selector，例如 app-user 会在模板中写成 <app-user />。

在这个活动中，你将学习如何组合组件。

在这个示例中，有两个组件 UserComponent 和 AppComponent。

  1. 添加对 UserComponent 的引用

    更新 AppComponent 模板以包含对使用 app-user 选择器的 UserComponent 的引用。确保将 UserComponent 添加到 AppComponent 的 imports 数组中，这使得它可以在 AppComponent 模板中使用。

          
    template: `<app-user />`,
    imports: [UserComponent]

    组件现在显示消息 Username: youngTech。你可以更新模板代码以包含更多标记。

  2. 添加更多标记

    因为你可以在模板中使用任何 HTML 标记，试着更新 AppComponent 的模板以包含更多 HTML 元素。这个示例将添加一个 <section> 元素作为 <app-user> 元素的父元素。

          
    template: `<section><app-user /></section>`,

你可以使用尽可能多的 HTML 标记和组件来实现你的应用创意。你甚至可以在同一页面上拥有多个组件副本。
