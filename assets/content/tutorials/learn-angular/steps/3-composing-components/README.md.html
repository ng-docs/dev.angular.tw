組合元件

edit

你已經學會了更新元件範本、元件邏輯和元件樣式，但是如何在你的應用中使用元件呢？

元件設定的 selector 屬性為你提供了一個名稱，以便在另一個範本中引用該組件。你可以像使用 HTML 標籤一樣使用 selector，例如 app-user 在範本中會是 <app-user />。

在本活動中，你將學習如何組合元件。

在這個範例中，有兩個元件 UserComponent 和 AppComponent。

  1. 新增對 UserComponent 的引用

    更新 AppComponent 範本，以包含對使用選擇器 app-user 的 UserComponent 的引用。請務必將 UserComponent 新增到 AppComponent 的 imports 陣列中，這將使其可在 AppComponent 範本中使用。

          
    template: `<app-user />`,imports: [UserComponent]

    現在，該組件顯示訊息 Username: youngTech。你可以更新範本程式碼以包含更多標記。

  2. 新增更多標記

    因為你可以在範本中使用任何你想要的 HTML 標記，所以嘗試更新 AppComponent 的範本，使其也包含更多 HTML 元素。此範例將新增一個 <section> 元素作為 <app-user> 元素的父元素。

          
    template: `<section><app-user /></section>`,

你可以使用任意數量的 HTML 標記和元件，以將你的應用想法變為現實。你甚至可以在同一頁面上擁有元件的多個副本。
