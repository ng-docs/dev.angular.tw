更新组件类

edit

在 Angular 中，组件的逻辑和行为在组件的 TypeScript 类中定义。

在此活动中，你将学习如何更新组件类以及如何使用 插值。

  1. 添加一个名为 city 的属性

    通过向 AppComponent 类添加一个名为 city 的属性来更新组件类。

          
    export class AppComponent {
      city = 'San Francisco';
    }

    虽然 city 属性是 string 类型，但你可以省略类型，因为有 TypeScript 的类型推断。 city 属性可以在 AppComponent 类中使用，并且可以在组件模板中引用。


    要在模板中使用类属性，你必须使用 {{ }} 语法。

  2. 更新组件模板

    更新 template 属性以匹配以下 HTML：

          
    template: `Hello {{ city }}`,

    这是插值的一个示例，是 Angular 模板语法的一部分。它不仅可以让你在模板中放置动态文本，还可以使用此语法调用函数、编写表达式等。

  3. 更多插值练习

    试试这个 —— 添加另一组 {{ }}，内容为 1 + 1：

          
    template: `Hello {{ city }}, {{ 1 + 1 }}`,

    Angular 会求值 {{ }} 的内容，并在模板中渲染输出。

这只是 Angular 模板可能性的开始，继续学习以了解更多。
