更新元件類別

edit

在 Angular 中，元件的邏輯和行為是在元件的 TypeScript 類別中定義的。

在本活動中，你將學習如何更新元件類別以及如何使用內嵌。

  1. 新增一個名為 city 的屬性

    透過向 AppComponent 類新增一個名為 city 的屬性來更新元件類別。

          
    export class AppComponent {  city = 'San Francisco';}

    city 屬性的型別為 string，但由於 TypeScript 中的型別推斷，你可以省略型別。city 屬性可以在 AppComponent 類別中使用，也可以在元件樣板中引用。


    要在樣板中使用類別屬性，你必須使用 {{ }} 語法。

  2. 更新元件樣板

    更新 template 屬性以對應以下 HTML：

          
    template: `Hello {{ city }}`,

    這是內嵌的一個範例，也是 Angular 樣板語法的一部分。它使你能夠做更多的事情，而不僅僅是在樣板中放置動態文字。你還可以使用此語法來呼叫函式、編寫表示式等等。

  3. 更多內嵌實踐

    試試這個 - 新增另一組 {{ }}，內容為 1 + 1：

          
    template: `Hello {{ city }}, {{ 1 + 1 }}`,

    Angular 會求值 {{ }} 的內容，並在樣板中渲染輸出。

這僅僅是 Angular 樣板可能實現的冰山一角，繼續學習以瞭解更多訊息。
