Angular 中的屬性（Property）繫結

edit

Angular 中的屬性（Property）繫結使你能夠為 HTML 元素、Angular 元件等的屬性（Property）設定值。

使用屬性（Property）繫結來動態地為屬性（Property）和 Attribute 設定值。你可以執行諸如切換按鈕屬性、以程式設計方式設定圖片路徑以及在元件之間共享值等操作。

在本活動中，你將學習如何在樣板中使用屬性（Property）繫結。

要繫結到元素的 Attribute，請將 Attribute 名稱放在方括號中。這是一個範例：

      
<img alt="photo" [src]="imageURL">

在本範例中，src Attribute 的值將繫結到類別屬性（Property） imageURL。無論 imageURL 具有什麼值，都將設定為 img 標籤的 src Attribute。

  1. 新增一個名為 isEditable 的屬性

    透過將一個名為 isEditable 的屬性（Property）新增到 AppComponent 類，並將初始值設定為 true，來更新 app.component.ts 中的程式碼。

          
    export class AppComponent {    isEditable = true;}

  2. 繫結到 contentEditable

    接下來，透過使用 [] 語法，將 div 的 contentEditable Attribute 繫結到 isEditable 屬性（Property）。

          
    @Component({    ...    template: `<div [contentEditable]="isEditable"></div>`,})

現在這個 div 是可編輯的了。幹得漂亮 👍

屬性（Property）繫結是 Angular 眾多強大的屬性之一。如果你想了解更多訊息，請檢視Angular 文件。
