Angular 中的属性绑定

edit

Angular 中的属性绑定使你可以为 HTML 元素、Angular 组件等的属性设置值。

使用属性绑定可以动态设置属性和属性的值。你可以做到诸如切换按钮特性、以编程方式设置图像路径以及在组件之间共享值等操作。

在此活动中，你将学习如何在模板中使用属性绑定。

要绑定到元素的属性，请将属性名称用方括号括起来。以下是一个示例：

      
<img alt="photo" [src]="imageURL">

在此示例中， src 属性的值将绑定到类属性 imageURL。 imageURL 的任何值都将设置为 img 标签的 src 属性。

  1. 添加一个名为 isEditable 的属性

    通过在 app.component.ts 中添加一个名为 isEditable 的属性，并将初始值设置为 true，来更新 AppComponent 类中的代码。

          
    export class AppComponent {
        isEditable = true;
    }

  2. 绑定到 contentEditable

    接下来，使用 [] 语法将 div 的 contentEditable 属性绑定到 isEditable 属性。

          
    @Component({
        ...
        template: `<div [contentEditable]="isEditable"></div>`,
    })

此 div 现在是可编辑的。干得好 👍

属性绑定是 Angular 的众多强大特性之一。如果你想了解更多，请查看 Angular 文档。
