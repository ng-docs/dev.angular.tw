路由概覽

edit

對於大多數應用來說，都會出現應用需要多個頁面的情況。當這種情況不可避免地發生時，路由就會成為關乎使用者體驗效能的重要部分。

在本活動中，你將學習如何建立和設定你的應用以使用 Angular 路由器。

  1. 建立一個 app.routes.ts 檔案

    在 app.routes.ts 檔案中，進行如下更改：

    1. 從 @angular/router 套件中匯入 Routes。
    2. 匯出一個名為 routes 的常數，其型別為 Routes，並賦予它 [] 作為值。
          
    import {Routes} from '@angular/router';export const routes: Routes = [];

  2. 將路由新增到提供者

    在 app.config.ts 中，透過以下步驟將應用設定為使用 Angular 路由器：

    1. 從 @angular/router 中匯入 provideRouter 函式。
    2. 從 ./app.routes.ts 中匯入 routes。
    3. 在 providers 陣列中呼叫 provideRouter 函式，並將 routes 作為引數傳入。
          
    import {ApplicationConfig} from '@angular/core';import {provideRouter} from '@angular/router';import {routes} from './app.routes';export const appConfig: ApplicationConfig = {  providers: [provideRouter(routes)],};

  3. 在元件中匯入 RouterOutlet

    最後，為了確保你的應用已準備好使用 Angular 路由器，你需要告訴應用你期望路由器在何處顯示所需內容。透過使用 @angular/router 中的 RouterOutlet 指令來完成此操作。

    透過新增 <router-outlet /> 來更新 AppComponent 的樣板

          
    import {RouterOutlet} from '@angular/router';@Component({  ...  template: `    <nav>      <a href="/">Home</a>      |      <a href="/user">User</a>    </nav>    <router-outlet />  `,  imports: [RouterOutlet],})export class AppComponent {}

你的應用現在已設定為使用 Angular 路由器。幹得漂亮！🙌
