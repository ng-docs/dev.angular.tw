路由概述

edit

对于大多数应用，总有一天需要超过一个页面。当那一天不可避免地到来时，路由将成为用户性能故事中的重要部分。

在这个活动中，你将学习如何设置和配置应用以使用 Angular 路由器。

  1. 创建 app.route.ts 文件

    在 app.routes.ts 中，做以下更改：

    1. 从 @angular/router 包导入 Routes。
    2. 导出一个名为 routes 的常量，类型为 Routes，赋值为 []。
          
    import {Routes} from '@angular/router';
    export const routes: Routes = [];

  2. 将路由添加到提供者

    在 app.config.ts 中，按以下步骤配置应用以使用 Angular 路由器：

    1. 从 @angular/router 导入 provideRouter 函数。
    2. 从 ./app.routes.ts 导入 routes。
    3. 在 providers 数组中调用 provideRouter 函数，并传入 routes 作为参数。
          
    import {ApplicationConfig} from '@angular/core';
    import {provideRouter} from '@angular/router';
    import {routes} from './app.routes';
    export const appConfig: ApplicationConfig = {
      providers: [provideRouter(routes)],
    };

  3. 在组件中导入 RouterOutlet

    最后，为了确保你的应用可以使用 Angular 路由，你需要告诉应用在何处显示所需的内容。通过使用 @angular/router 中的 RouterOutlet 指令来实现。

    通过添加 <router-outlet /> 更新 AppComponent 的模板。

          
    import {RouterOutlet} from '@angular/router';
    @Component({
      ...
      template: `
        <nav>
          <a href="/">Home</a>
          |
          <a href="/user">User</a>
        </nav>
        <router-outlet />
      `,
      standalone: true,
      imports: [RouterOutlet],
    })
    export class AppComponent {}

你的应用现在已设置为使用 Angular 路由器。做得好！🙌
