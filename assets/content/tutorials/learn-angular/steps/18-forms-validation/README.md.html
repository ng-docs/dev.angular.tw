验证表单

edit

在处理表单时的另一个常见情景是需要验证输入，以确保提交正确的数据。

在这个活动中，你将学习如何使用响应式表单验证表单。

  1. 导入验证器

    Angular提供了一套验证工具。要使用这些工具，首先更新组件，从 @angular/forms导入 Validators。

          
    import {ReactiveFormsModule, Validators} from '@angular/forms';
    @Component({...})
    export class AppComponent {}

  2. 向表单添加验证

    每个 FormControl可以传入你想要用于验证 FormControl值的 Validators。例如，如果你想要使 profileForm中的 name字段为必填，则使用 Validators.required。 对于我们Angular表单中的 email字段，我们希望确保它不为空并遵循有效的电子邮件地址结构。我们可以通过将 Validators.required和 Validators.email验证器组合在一个数组中来实现这一点。 更新 name和 email的 FormControl：

          
    profileForm = new FormGroup({
      name: new FormControl('', Validators.required),
      email: new FormControl('', [Validators.required, Validators.email]),
    });

  3. 检查模板中的表单验证

    要确定表单是否有效， FormGroup类具有一个 valid属性。你可以使用这个属性动态绑定属性。根据表单的有效性更新提交 button以启用它。

          
    <button type="submit" [disabled]="!profileForm.valid">Submit</button>

你现在已经了解了使用响应式表单进行验证的基础知识。

学习 Angular 中与表单相关的核心概念做得很好。如果你想了解更多，请务必参考 Angular 表单文档。
