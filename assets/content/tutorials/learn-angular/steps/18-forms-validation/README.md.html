驗證表單

edit

使用表單時，另一個常見的場景是需要驗證輸入，以確保送出的資料正確。

在本活動中，你將學習如何使用回應式表單驗證表單。

  1. 匯入 Validators

    Angular 提供了一組驗證工具。要使用它們，首先更新元件以從 @angular/forms 匯入 Validators。

          
    import {ReactiveFormsModule, Validators} from '@angular/forms';@Component({...})export class AppComponent {}

  2. 為表單新增驗證

    每個 FormControl 都可以傳入你想要用於驗證 FormControl 值的 Validators。例如，如果你想讓 profileForm 中的 name 欄位成為必要欄位，則可以使用 Validators.required。 對於 Angular 表單中的 email 欄位，我們希望確保它不為空，並且遵循有效的電子郵件地址結構。我們可以透過在一個陣列中組合 Validators.required 和 Validators.email 驗證器來實現這一點。 更新 name 和 email FormControl：

          
    profileForm = new FormGroup({  name: new FormControl('', Validators.required),  email: new FormControl('', [Validators.required, Validators.email]),});

  3. 在範本中檢查表單驗證

    要確定表單是否有效，FormGroup 類別具有 valid 屬性。 你可以使用此屬性來動態繫結 Attribute。更新送出 button，使其根據表單的有效性啟用。

          
    <button type="submit" [disabled]="!profileForm.valid">Submit</button>

現在你已經瞭解了關於驗證如何與回應式表單一起工作的基礎知識。
