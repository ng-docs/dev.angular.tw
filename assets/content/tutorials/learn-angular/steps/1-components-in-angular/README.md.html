Angular 中的元件

edit

元件是任何 Angular 應用的基礎基礎元素。每個元件都由三個部分組成：

  • TypeScript 類別
  • HTML 範本
  • CSS 樣式

在本活動中，你將學習如何更新元件的範本和樣式。

這是一個開始使用 Angular 的絕佳機會。

  1. 更新元件範本

    更新 template 屬性，使其讀取 Hello Universe

          
    template: `  Hello Universe`,

    當你更改 HTML 範本後，預覽會使用你的訊息進行更新。 讓我們更進一步：更改文字的顏色。

  2. 更新元件樣式

    更新 styles 值並將 color 屬性從 blue 更改為 #a144eb。

          
    styles: `  :host {    color: #a144eb;  }`,

    當你檢視預覽時，你會發現文字顏色將被更改。

在 Angular 中，你可以使用所有瀏覽器支援的 CSS 和 HTML。 如果你願意，可以將範本和樣式儲存在單獨的檔案中。
