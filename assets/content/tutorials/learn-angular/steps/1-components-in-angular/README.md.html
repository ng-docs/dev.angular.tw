Angular 中的组件

edit

组件是任何 Angular 应用程序的基础构建块。每个组件包括三个部分：

  • TypeScript 类
  • HTML 模板
  • CSS 样式

在这个活动中，你将学习如何更新组件的模板和样式。

这是一个很好的机会让你开始使用 Angular。

  1. 更新组件模板

    将 template属性更新为 Hello Universe

          
    template: `
      Hello Universe
    `,

    当你改变了 HTML 模板后，预览会显示你的消息。让我们再进一步：改变文本的颜色。

  2. 更新组件样式

    更新样式值并将 color属性从 blue更改为 #a144eb。

          
    styles: `
      :host {
        color: #a144eb;
      }
    `,

    当你检查预览时，你会发现文本颜色已经改变了。

在 Angular 中，你可以使用所有浏览器支持的 CSS 和 HTML。如果愿意，你可以将模板和样式存储在分开的文件中。
