表单概览

edit

表单是许多应用的重要组成部分，因为它们使你的应用能够接受用户输入。让我们学习如何在 Angular 中处理表单。

在 Angular 中，有两种类型的表单：模板驱动的表单和响应式表单。你将在接下来的几个活动中学习这两种表单。

在此活动中，你将学习如何使用模板驱动的方法建立表单。

  1. 创建一个输入字段

    在 user.component.ts 中，通过添加一个 id 设置为 framework，类型设置为 text 的文本输入来更新模板。

          
    <label for="framework">
      Favorite Framework:
      <input id="framework" type="text" />
    </label>

  2. 导入 FormsModule

    为了让此表单使用 Angular 的特性实现数据绑定，需要导入 FormsModule。

    从 @angular/forms 导入 FormsModule 并将其添加到 UserComponent 的 imports 数组中。

          
    import {Component} from '@angular/core';
    import {FormsModule} from '@angular/forms';
    @Component({
      ...
      standalone: true,
      imports: [FormsModule],
    })
    export class UserComponent {}

  3. 为输入的值添加绑定

    FormsModule 有一个名为 ngModel 的指令，它将输入的值绑定到类中的某个属性。

    更新输入以使用 ngModel 指令，具体语法为 [(ngModel)]="favoriteFramework"，绑定到 favoriteFramework 属性。

          
    <label for="framework">
      Favorite Framework:
      <input id="framework" type="text" [(ngModel)]="favoriteFramework" />
    </label>

    在你做出更改后，尝试在输入字段中输入一个值。注意它如何在屏幕上更新（是的，非常酷）。

    注意：语法 [()] 被称为「banana in a box」，表示双向绑定：属性绑定和事件绑定。可以在 Angular 关于双向数据绑定的文档 中了解更多。

你已经迈出了使用 Angular 构建表单的重要第一步。
