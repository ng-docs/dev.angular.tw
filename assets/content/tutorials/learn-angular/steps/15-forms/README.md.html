表單概覽

edit

表單是許多應用的重要組成部分，因為它們讓你的應用能夠接受使用者輸入。讓我們來學習一下 Angular 中是如何處理表單的。

在 Angular 中，有兩種型別的表單：樣板驅動型和回應式。在接下來的幾個活動中，你將學習這兩種表單。

在這個活動中，你將學習如何使用樣板驅動型方法建立表單。

  1. 建立一個輸入欄位

    在 user.component.ts 中，更新樣板，新增一個文字輸入框，其中 id 設定為 framework，型別設定為 text。

          
    <label for="framework">  Favorite Framework:  <input id="framework" type="text" /></label>

  2. 匯入 FormsModule

    為了讓這個表單使用 Angular 的屬性來實現與表單的資料繫結，你需要匯入 FormsModule。

    從 @angular/forms 匯入 FormsModule，並將其新增到 UserComponent 的 imports 陣列中。

          
    import {Component} from '@angular/core';import {FormsModule} from '@angular/forms';@Component({  ...  imports: [FormsModule],})export class UserComponent {}

  3. 為輸入框的值新增繫結

    FormsModule 有一個名為 ngModel 的指令，它可以將輸入框的值繫結到你的類別中的一個屬性（Property）。

    更新輸入框以使用 ngModel 指令，特別是使用以下語法 [(ngModel)]="favoriteFramework" 來繫結到 favoriteFramework 屬性（Property）。

          
    <label for="framework">  Favorite Framework:  <input id="framework" type="text" [(ngModel)]="favoriteFramework" /></label>

    在你做出更改後，嘗試在輸入欄位中輸入一個值。注意它如何在螢幕上更新（是的，非常酷）。

    注意：語法 [()] 被稱為 『香蕉套盒』，但它代表雙向繫結：屬性（Property）繫結和事件繫結。 在 關於雙向資料繫結的 Angular 文件 中瞭解更多訊息。

你現在已經朝著使用 Angular 建立表單邁出了重要的第一步。
