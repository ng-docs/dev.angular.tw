通道

edit

通道是在範本中轉換資料時使用的函式。通常，通道是不會引起副作用的『純』函式。Angular 有許多有用的內建通道，你可以匯入並在元件中使用。你也可以建立自定義通道。

在本活動中，你將匯入一個通道並在範本中使用它。

要在範本中使用通道，請將其包含在內嵌表示式中。請檢視以下範例：

      
import {UpperCasePipe} from '@angular/common';@Component({    ...    template: `{{ loudMessage | uppercase }}`,    imports: [UpperCasePipe],})class AppComponent {    loudMessage = 'we think you are doing great!'}

現在，輪到你來嘗試一下了：

  1. 匯入 LowerCase 通道

    首先，更新 app.component.ts，透過從 @angular/common 新增檔案級匯入 LowerCasePipe。

          
    import { LowerCasePipe } from '@angular/common';

  2. 將通道新增到範本匯入

    接下來，更新 @Component() 裝飾器的 imports 以包含對 LowerCasePipe 的引用

          
    @Component({    ...    imports: [LowerCasePipe]})

  3. 將通道新增到範本

    最後，在 app.component.ts 中，更新範本以包含 lowercase 通道：

          
    template: `{{username | lowercase }}`

通道也可以接受引數，這些引數可用於設定其輸出。在下一個活動中瞭解更多訊息。
