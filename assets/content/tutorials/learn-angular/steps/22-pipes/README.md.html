管道

edit

管道是用于在模板中转换数据的函数。一般来说，管道是不会产生副作用的「纯」函数。Angular 提供了一些有用的内置管道，你可以导入并在组件中使用。你也可以创建自定义管道。

在此活动中，你将导入一个管道并在模板中使用它。

要在模板中使用管道，将其包含在一个插值表达式中。请看这个示例：

      
import {UpperCasePipe} from '@angular/common';
@Component({
    ...
    template: `{{ loudMessage | uppercase }}`,
    imports: [UpperCasePipe],
})
class AppComponent {
    loudMessage = 'we think you are doing great!'
}

现在，轮到你试一试了：

  1. 导入 LowerCase 管道

    首先，通过从 @angular/common 添加文件级别的 LowerCasePipe 导入来更新 app.component.ts。

          
    import { LowerCasePipe } from '@angular/common';

  2. 将管道添加到模板导入中

    接下来，更新 @Component() 装饰器中的 imports，以包含对 LowerCasePipe 的引用。

          
    @Component({
        ...
        imports: [LowerCasePipe]
    })

  3. 将管道添加到模板中

    最后，在 app.component.ts 中更新模板以包含 lowercase 管道：

          
    template: `{{username | lowercase }}`

管道还可以接受参数，这些参数可用于配置其输出。在下一个活动中了解更多。
