將搜尋屬性新增到你的應用

edit

本教程課程示範了如何向你的 Angular 應用新增搜尋功能。

該應用將使用戶能夠搜尋你的應用所提供的資料，並且僅顯示與輸入的術語對應的結果。

重要提示： 我們建議在本教程的此步驟中使用你的本地環境。

你將學到什麼

  • 你的應用將使用表單中的資料來搜尋對應的房屋位置
  • 你的應用將僅顯示對應的房屋位置

  1. 更新 home 元件的屬性（Property）

    在此步驟中，你將更新 HomeComponent 類，以將資料儲存在一個新的陣列屬性（Property）中，你將使用該屬性進行過濾。

    1. 在 src/app/home/home.component.ts 中，向該類別新增名為 filteredLocationList 的新屬性（Property）。

      新增已過濾結果屬性

            
      import {Component, inject} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocationComponent} from '../housing-location/housing-location.component';import {HousingLocation} from '../housinglocation';import {HousingService} from '../housing.service';@Component({  selector: 'app-home',  imports: [CommonModule, HousingLocationComponent],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" #filter />        <button class="primary" type="button" (click)="filterResults(filter.value)">Search</button>      </form>    </section>    <section class="results">      <app-housing-location        *ngFor="let housingLocation of filteredLocationList"        [housingLocation]="housingLocation"      ></app-housing-location>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {  housingLocationList: HousingLocation[] = [];  housingService: HousingService = inject(HousingService);  filteredLocationList: HousingLocation[] = [];  constructor() {    this.housingLocationList = this.housingService.getAllHousingLocations();    this.filteredLocationList = this.housingLocationList;  }  filterResults(text: string) {    if (!text) {      this.filteredLocationList = this.housingLocationList;      return;    }    this.filteredLocationList = this.housingLocationList.filter((housingLocation) =>      housingLocation?.city.toLowerCase().includes(text.toLowerCase()),    );  }}

      filteredLocationList 儲存與使用者輸入的搜尋條件對應的值。

    2. 預設情況下，當頁面載入時，filteredLocationList 應包含房屋位置值的總集合。更新 HomeComponent 的 constructor 以設定該值。

    設定 filteredLocationList 的值

          
    import {Component, inject} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocationComponent} from '../housing-location/housing-location.component';import {HousingLocation} from '../housinglocation';import {HousingService} from '../housing.service';@Component({  selector: 'app-home',  imports: [CommonModule, HousingLocationComponent],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" #filter />        <button class="primary" type="button" (click)="filterResults(filter.value)">Search</button>      </form>    </section>    <section class="results">      <app-housing-location        *ngFor="let housingLocation of filteredLocationList"        [housingLocation]="housingLocation"      ></app-housing-location>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {  housingLocationList: HousingLocation[] = [];  housingService: HousingService = inject(HousingService);  filteredLocationList: HousingLocation[] = [];  constructor() {    this.housingLocationList = this.housingService.getAllHousingLocations();    this.filteredLocationList = this.housingLocationList;  }  filterResults(text: string) {    if (!text) {      this.filteredLocationList = this.housingLocationList;      return;    }    this.filteredLocationList = this.housingLocationList.filter((housingLocation) =>      housingLocation?.city.toLowerCase().includes(text.toLowerCase()),    );  }}

  2. 更新 home 元件樣板

    HomeComponent 已經包含一個輸入欄位，你將使用它來捕獲使用者的輸入。該字串文字將用於過濾結果。

    1. 更新 HomeComponent 樣板，以在名為 #filter 的 input 元素中包含樣板變數。

      向 HomeComponent 的樣板新增一個樣板變數

            
      <input type="text" placeholder="Filter by city" #filter>

      此範例使用樣板引用變數來訪問 input 元素及其值。

    2. 接下來，更新元件樣板，以將事件處理器附著到『Search』按鈕。

      繫結 click 事件

            
      <button class="primary" type="button" (click)="filterResults(filter.value)">Search</button>

      透過繫結到 button 元素上的 click 事件，你能夠呼叫 filterResults 函式。該函式的引數是 filter 樣板變數的 value 屬性（Property）。具體來說，是來自 input HTML 元素的 .value 屬性（Property）。

    3. 最後一個樣板更新是針對 ngFor 指令。更新 ngFor 值以迭代來自 filteredLocationList 陣列的值。

    更新 ngFor 指令值

          
    <app-housing-location *ngFor="let housingLocation of filteredLocationList" [housingLocation]="housingLocation"></app-housing-location>

  3. 實現事件處理器函式

    樣板已更新，以將 filterResults 函式繫結到 click 事件。接下來，你的任務是在 HomeComponent 類別中實現 filterResults 函式。

    1. 更新 HomeComponent 類，以包含 filterResults 函式的實現。

      新增 filterResults 函式實現

            
      import {Component, inject} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocationComponent} from '../housing-location/housing-location.component';import {HousingLocation} from '../housinglocation';import {HousingService} from '../housing.service';@Component({  selector: 'app-home',  imports: [CommonModule, HousingLocationComponent],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" #filter />        <button class="primary" type="button" (click)="filterResults(filter.value)">Search</button>      </form>    </section>    <section class="results">      <app-housing-location        *ngFor="let housingLocation of filteredLocationList"        [housingLocation]="housingLocation"      ></app-housing-location>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {  housingLocationList: HousingLocation[] = [];  housingService: HousingService = inject(HousingService);  filteredLocationList: HousingLocation[] = [];  constructor() {    this.housingLocationList = this.housingService.getAllHousingLocations();    this.filteredLocationList = this.housingLocationList;  }  filterResults(text: string) {    if (!text) {      this.filteredLocationList = this.housingLocationList;      return;    }    this.filteredLocationList = this.housingLocationList.filter((housingLocation) =>      housingLocation?.city.toLowerCase().includes(text.toLowerCase()),    );  }}

      此函式使用 String filter 函式將 text 引數的值與 housingLocation.city 屬性（Property）進行比較。你可以更新此函式以對應任何屬性（Property）或多個屬性（Property），這是一個有趣的練習。

    2. 儲存你的程式碼。

    3. 重新整理瀏覽器並確認，在輸入文字後單擊『Search』按鈕時，你可以按城市搜尋房屋位置資料。

摘要： 在本課程中，你更新了你的應用，以使用樣板變數與樣板值進行互動，並使用事件繫結和數組函式新增了搜尋功能。
