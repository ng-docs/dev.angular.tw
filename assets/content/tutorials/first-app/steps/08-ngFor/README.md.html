使用 *ngFor* 在元件中列出物件

edit

本教程課程示範了如何在 Angular 樣板中使用 ngFor 指令，以便在樣板中顯示動態重複的資料。

你將學到什麼

  • 你將向此應用新增一個資料集
  • 你的應用將使用 ngFor 顯示來自新資料集的元素清單

ngFor 的概念預覽

在 Angular 中，ngFor 是一種特定型別的指令，用於在樣板中動態重複資料。在純 JavaScript 中，你會使用 for 迴圈 - ngFor 為 Angular 樣板提供了類似的功能。

你可以利用 ngFor 迭代陣列，甚至非同步值。在本課程中，你將新增一個新的資料陣列以進行迭代。

有關更深入的解釋，請參閱內建指令指南。
