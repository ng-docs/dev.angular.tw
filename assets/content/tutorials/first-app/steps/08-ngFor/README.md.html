使用*ngFor在组件中列出对象

edit

本教程演示了如何在Angular模板中使用 ngFor指令，以便在模板中动态显示重复的数据。

你将学到什么

  • 你将向应用程序添加一个数据集
  • 你的应用将使用 ngFor从新数据集中显示元素列表

ngFor 的概念性预览

在 Angular 中， ngFor 是一种特定类型的 指令，用于在模板中动态重复数据。在纯 JavaScript 中，你会使用 for 循环 — ngFor 为 Angular 模板提供了类似的功能。

你可以利用 ngFor 迭代数组，甚至是异步值。在本课程中，你将添加一个新数据数组来进行迭代。

有关更深入的解释，请参阅 内置指令指南。
