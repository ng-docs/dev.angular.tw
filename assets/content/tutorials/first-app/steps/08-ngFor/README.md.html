使用 *ngFor 在元件中列出物件

edit

以下教學課程示範如何在 Angular 範本中使用 ngFor 指令，以便在範本中顯示動態重複的資料。

你將會學到

  • 您會已將資料集加入 app
  • 您的 app 將使用 ngFor 顯示來自新資料集的元素清單

ngFor 的概念預覽

在 Angular 中，ngFor 是一種特定類型的 指令，用於動態重複範本中的資料。在純 JavaScript 中，您會使用 for 迴圈 - ngFor 為 Angular 範本提供類似的功能。

你可以使用 ngFor 來叠代陣列，甚至非同步值。在本課程中，你將新增一個要叠代的資料陣列。

如需更深入的說明，請參閱 內建指令 指南。
