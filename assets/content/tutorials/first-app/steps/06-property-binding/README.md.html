向组件模板添加一个属性绑定

edit

这节课演示了如何将属性绑定添加到模板中并通过它将动态数据传给组件。

你将学到什么

  • 应用程序在 HomeComponent 模板中具有一些数据绑定。
  • 应用程序将数据从 HomeComponent 发送到 HousingLocationComponent。

输入属性的概念性预览

在这节课中，你将继续通过在模板中使用属性绑定将数据从父组件共享到子组件的过程。

属性绑定使你能够将一个变量连接到 Angular 模板中的一个 Input。然后，数据会动态绑定到 Input 中。

欲了解更详细的说明，请参阅 属性绑定 指南。

  1. 更新 HomeComponent 模板

    此步骤将属性绑定添加到 <app-housing-location> 标记上。

    在代码编辑器中：

    1. 导航到 src/app/home/home.component.ts

    2. 在 @Component 装饰器的模板属性中，更新代码以匹配下面的代码：

      添加 housingLocation 属性绑定

            
      import {Component} from '@angular/core';
      import {CommonModule} from '@angular/common';
      import {HousingLocationComponent} from '../housing-location/housing-location.component';
      import {HousingLocation} from '../housinglocation';
      @Component({
        selector: 'app-home',
        standalone: true,
        imports: [CommonModule, HousingLocationComponent],
        template: `
          <section>
            <form>
              <input type="text" placeholder="Filter by city" />
              <button class="primary" type="button">Search</button>
            </form>
          </section>
          <section class="results">
            <app-housing-location [housingLocation]="housingLocation"></app-housing-location>
          </section>
        `,
        styleUrls: ['./home.component.css'],
      })
      export class HomeComponent {
        readonly baseUrl = 'https://angular.dev/assets/tutorials/common';
        housingLocation: HousingLocation = {
          id: 9999,
          name: 'Test Home',
          city: 'Test city',
          state: 'ST',
          photo: `${this.baseUrl}/example-house.jpg`,
          availableUnits: 99,
          wifi: true,
          laundry: false,
        };
      }

      当向组件标签添加属性绑定时，我们使用 [attribute] = "value" 语法来通知 Angular 所赋的值应该视为来自组件类的属性，而不是字符串值。

      右侧的值是从 HomeComponent中属性的名称。

  2. 确认代码仍在运行

    1. 保存你的更改并确认应用程序没有任何错误。
    2. 在继续下一步之前更正所有错误。

总结：在这节课中，你添加了一个新的属性绑定，并传递了一个类属性的引用。现在， HousingLocationComponent可以访问数据，用于自定义组件的显示。
