向元件的範本新增內嵌

edit

本教學課程課程示範了如何向 Angular 範本新增內嵌，以便在範本中顯示動態資料。

你將學到什麼

  • 你的應用將在 HousingLocationComponent 範本中顯示內嵌結果。
  • 你的應用會將房屋位置資料渲染到瀏覽器。

內嵌的概念預覽

在此步驟中，你將使用內嵌在範本中顯示值（屬性（Property）和 Input 值）。

在 Angular 範本中使用 {{ expression }}，你可以渲染來自屬性（Property）、Inputs 和有效的 JavaScript 表示式的值。

要獲得更深入的解釋，請參閱使用內嵌顯示值指南。

  1. 更新 HousingLocationComponent 範本以包含內嵌結果

    此步驟在 HousingLocationComponent 範本中新增了新的 HTML 結構和內嵌結果。

    在程式碼編輯器中：

    1. 導向到 src/app/housing-location/housing-location.component.ts
    2. 在 @Component 裝飾器的範本屬性中，將現有的 HTML 標記替換為以下程式碼：

    更新 HousingLocationComponent 範本

          
    import {Component, Input} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocation} from '../housinglocation';@Component({  selector: 'app-housing-location',  imports: [CommonModule],  template: `    <section class="listing">      <img        class="listing-photo"        [src]="housingLocation.photo"        alt="Exterior photo of {{ housingLocation.name }}"        crossorigin      />      <h2 class="listing-heading">{{ housingLocation.name }}</h2>      <p class="listing-location">{{ housingLocation.city }}, {{ housingLocation.state }}</p>    </section>  `,  styleUrls: ['./housing-location.component.css'],})export class HousingLocationComponent {  @Input() housingLocation!: HousingLocation;}

    在這個更新後的範本程式碼中，你使用了屬性繫結將 housingLocation.photo 繫結到 src Attribute。alt Attribute 使用內嵌來為影象的 alt 文字提供更多上下文。

    你使用內嵌來包含 housingLocation 屬性的 name、city 和 state 的值。

  2. 確認更改在瀏覽器中渲染

    1. 儲存所有更改。
    2. 開啟瀏覽器並確認應用渲染了照片、城市和州範例資料。

摘要： 在本課程中，你新增了一個新的 HTML 結構，並使用 Angular 範本語法在 HousingLocation 範本中渲染值。

現在，你掌握了兩項重要的技能：

  • 將資料傳遞給元件
  • 將值內嵌到範本中

有了這些技能，你的應用現在可以共享資料並在瀏覽器中顯示動態值了。目前為止做得很好。
