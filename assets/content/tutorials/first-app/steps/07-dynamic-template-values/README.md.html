向组件模板添加插值

edit

本教程示范了如何向 Angular 模板中添加插值，以便在模板中显示动态数据。

你将学到什么

  • 应用程序将在 HousingLocationComponent 模板中显示插值。
  • 应用程序将向浏览器渲染出房屋位置数据。

插值的概念性预览

在这一步中，你将使用插值在模板中显示值（属性和 Input值）。

在 Angular 模板中使用 {{ expression }}，可以从属性、 Inputs 和有效的 JavaScript 表达式中渲染值。

有关更详细的解释，请参考 使用插值显示值指南。

  1. 更新 HousingLocationComponent 模板以包含插值结果

    此步骤在 HousingLocationComponent 模板中添加新的 HTML 结构和插值。

    在代码编辑器中：

    1. 导航到 src/app/housing-location/housing-location.component.ts

    2. 在 @Component装饰器的模板属性中，用以下代码替换现有的HTML标记：

      更新HousingLocationComponent模板

            
      import {Component, Input} from '@angular/core';
      import {CommonModule} from '@angular/common';
      import {HousingLocation} from '../housinglocation';
      @Component({
        selector: 'app-housing-location',
        standalone: true,
        imports: [CommonModule],
        template: `
          <section class="listing">
            <img
              class="listing-photo"
              [src]="housingLocation.photo"
              alt="Exterior photo of {{ housingLocation.name }}"
              crossorigin
            />
            <h2 class="listing-heading">{{ housingLocation.name }}</h2>
            <p class="listing-location">{{ housingLocation.city }}, {{ housingLocation.state }}</p>
          </section>
        `,
        styleUrls: ['./housing-location.component.css'],
      })
      export class HousingLocationComponent {
        @Input() housingLocation!: HousingLocation;
      }

      在此更新过的模板代码中，你已使用属性绑定将 housingLocation.photo 绑定到 src 属性。而 alt 属性通过插值为图像的替代文本提供更多上下文。

      你使用插值来包含 housingLocation属性的 name、 city和 state的值。

  2. 确认更改在浏览器中渲染

    1. 保存所有更改。
    2. 打开浏览器，确认应用程序呈现照片、城市和州的示例数据。

总结：在这节课中，你添加了一个新的HTML结构，并使用Angular模板语法在 HousingLocation模板中呈现值。

现在，你掌握了两项重要技能：

  • 将数据传给组件
  • 将值插入模板

有了这些技能，应用程序现在就可以共享数据并在浏览器中显示动态值。到目前为止这工作得很好。
