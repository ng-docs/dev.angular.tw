Hello world

edit

以第一课作为起点，本教程中的每一课都会添加一些新功能以构建出完整的 Angular 应用程序。在这一课，我们将更新应用程序以显示著名的文本 “Hello World”。

你将学到什么

在这节课后更新你的应用程序，以确认你本人以及你的 IDE 已经准备好了创建 Angular 应用程序。

注意：如果你正在使用嵌入式编辑器，则跳转至 第三步。 在浏览器演练场中工作时，你无需运行 ng serve来运行应用程序。其他命令如 ng generate可以在右侧的控制台窗口中执行。

  1. 下载默认应用程序

    首先点击代码编辑器右上角的"下载"图标。这将下载一个包含本教程源代码的 .zip文件。在本地终端和IDE中打开此文件，然后继续测试默认应用程序。

    在教程的任何步骤中，你都可以点击这个图标下载该步骤的源代码并从那里开始。

  2. 测试默认应用

    在此步骤中，当下载完默认启动的应用程序后，就可以开始构建默认的 Angular 应用程序了。这可以确保你的开发环境具有继续本教程所需的内容。

    在你的 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令以安装运行应用程序所需的依赖项。

            
      npm install

    3. 运行此命令以构建和为默认应用程序启动开发服务器。

            
      ng serve

      该应用程序应该能没有任何错误地构建成功。

    4. 在开发机上的 Web 浏览器中，打开 http://localhost:4200。

    5. 确认浏览器中出现了默认网站。

    6. 当你完成下一步时，可以让 ng serve 继续运行。

  3. 审查项目中的文件

    在这一步中，你将认识那些构成默认 Angular 应用程序的文件。

    在 IDE 的浏览窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。
    2. Open the src directory to see these files.
      1. In the file explorer, find the Angular app files (/src).
        1. index.html 是应用程序的顶级 HTML 模板。
        2. style.css 是应用程序的顶级样式表。
        3. main.ts 是应用程序开始运行的地方。
        4. favicon.ico 是应用的图标，就像你在任何网站中找到的一样。
      2. In the file explorer, find the Angular app's component files (/app).

        1. app.component.ts 是描述 app-root 组件的源文件。这是应用中顶级的 Angular 组件。 组件描述包括组件的代码、HTML 模板和样式，这些可以在这个文件中描述，也可以在单独的文件中描述。

          在此应用程序中，样式位于单独的文件中，而组件的代码和 HTML 模板位于此文件中。

        2. app.component.css 是这个组件的样式表。

        3. 新组件会添加到此目录中。

      3. 在文件浏览器中，找到包含应用程序使用的图像的图像目录（ /assets）。
      4. In the file explorer, find the files and directories that an Angular app needs to build and run, but they are not files that you normally interact with.
        1. .angular 包含构建 Angular 应用程序时所需的文件。
        2. .e2e 包含用于测试应用程序的文件。
        3. .node_modules 包含应用程序使用的 node.js 包。
        4. angular.json 将向应用程序构建工具描述 Angular 应用程序。
        5. package.json 被 npm（Node 包管理器）用来运行已完成的应用程序。
        6. tsconfig.* 是向 TypeScript 编译器描述应用程序配置的文件。

    复查构成 Angular 应用程序项目的文件后，继续下一步。

  4. 创建 Hello World

    在此步骤中，你将更新 Angular 项目文件以更改显示的内容。

    在你的 IDE 中：

    1. 打开 first-app/src/index.html。 注意：这一步和接下来的步骤仅适用于你的本地环境！

    2. 在 index.html 中，用此代码替换 <title> 元素以更新应用程序的标题。

      在 src/index.html 中替换

            
      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="utf-8">
        <title>Homes</title>
        <base href="/">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link rel="icon" type="image/x-icon" href="favicon.ico">
        <link href="/assets/css/be-vietnam-pro.css" rel="stylesheet">
      </head>
      <body>
        <app-root></app-root>
      </body>
      </html>

      然后，保存你刚刚对 index.html 所做的更改。

    3. 接下来，打开 first-app/src/app/app.component.ts。

    4. 在 app.component.ts 中，在 @Component 定义中，用以下代码替换 template 一行，以更改应用组件中的文本。

      在 src/app/app.component.ts 中替换

            
      import {Component} from '@angular/core';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [],
        template: `
          <h1>Hello world!</h1>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'homes';
      }

    5. 在 app.component.ts 的 AppComponent 类定义中，用如下代码替换 title 行以更改组件标题。

      在 src/app/app.component.ts 中替换

            
      import {Component} from '@angular/core';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [],
        template: `
          <h1>Hello world!</h1>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'homes';
      }

      然后，保存对 app.component.ts 所做的更改。

    6. 如果你在第 1 步曾停止了 ng serve 命令，请在 IDE 的终端窗口中再次运行 ng serve。

    7. 打开你的浏览器并导航至 localhost:4200 并确认该应用程序构建无误并在应用程序的标题和正文中显示出了Hello world：

总结： 在这节课中，你更新了默认的 Angular 应用程序以显示 Hello world。 在这个过程中，你了解了 ng serve 命令用于在本地测试你的应用程序。
