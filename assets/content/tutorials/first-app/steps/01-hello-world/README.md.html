Hello world

edit

本節課是本教學課程的起點，本教學課程的後續課程都會在本節課的基礎上增加新屬性，以建立一個完整的 Angular 應用。在本節課中，我們將更新這個應用，以顯示著名的文字 『Hello World』。

你將學到什麼

本節課之後，更新後的應用會確認你和你的 IDE 已準備就緒，可以開始建立 Angular 應用了。

注意： 如果你正在使用嵌入式編輯器，請跳到第三步。 在瀏覽器 Playground 中工作時，你不需要使用 ng serve 來執行應用。其他指令（如 ng generate）可以在右側的控制台視窗中完成。

  1. 下載預設應用

    首先，單擊程式碼編輯器右上窗格中的 『下載』 圖示。這將下載一個 .zip 檔案，其中包含本教學課程的原始碼。在本地終端和 IDE 中開啟它，然後繼續測試預設應用。

    在本教學課程的任何步驟中，你都可以單擊此圖示來下載該步驟的原始碼並從那裡開始。

  2. 測試預設應用

    在此步驟中，下載預設起始應用後，你將建立預設的 Angular 應用。 這確認你的開發環境具備繼續本教學課程所需的條件。

    在 IDE 的 終端 窗格中：

    1. 在你的專案目錄中，導向到 first-app 目錄。

    2. 執行此指令以安裝執行應用所需的依賴項。

            
      npm install

    3. 執行此指令以建立併為預設應用啟動伺服器。

            
      ng serve

      應用應該可以成功建立，且不報錯。

    4. 在開發電腦的 Web 瀏覽器中，開啟 http://localhost:4200。

    5. 確認預設網站出現在瀏覽器中。

    6. 在完成後續步驟時，你可以保持 ng serve 執行。

  3. 審閱專案中的檔案

    在此步驟中，你將瞭解構成預設 Angular 應用的檔案。

    在 IDE 的 資源管理器 窗格中：

    1. 在你的專案目錄中，導向到 first-app 目錄。
    2. 開啟 src 目錄以檢視這些檔案。
      1. 在檔案資源管理器中，找到 Angular 應用文件 (/src)。
        1. index.html 是應用的頂層 HTML 範本。
        2. styles.css 是應用的頂層樣式表。
        3. main.ts 是應用開始執行的地方。
        4. favicon.ico 是應用的圖示，就像你在任何網站中找到的那樣。
      2. 在檔案資源管理器中，找到 Angular 應用的元件檔案 (/app)。

        1. app.component.ts 是描述 app-root 元件的源檔案。 這是應用中的頂層 Angular 元件。元件是 Angular 應用的基本元素。 元件描述包括元件的程式碼、HTML 範本和樣式，這些可以在此檔案中描述，也可以在單獨的檔案中描述。

          在本應用中，樣式位於單獨的檔案中，而元件的程式碼和 HTML 範本則位於此檔案中。

        2. app.component.css 是此元件的樣式表。

        3. 新元件會新增到此目錄中。

      3. 在檔案資源管理器中，找到影象目錄 (/assets)，其中包含應用使用的影象。
      4. 在檔案資源管理器中，找到 Angular 應用建立和執行所需的檔案和目錄，但它們不是你通常會互動的檔案。
        1. .angular 包含建立 Angular 應用所需的檔案。
        2. .e2e 包含用於測試應用的檔案。
        3. .node_modules 包含應用使用的 node.js 套件。
        4. angular.json 嚮應用建立工具描述 Angular 應用。
        5. package.json 供 npm（node 套件管理器）用來執行已完成的應用。
        6. tsconfig.* 是向 TypeScript 編譯器描述應用設定的檔案。

    審閱構成 Angular 應用專案的檔案後，繼續下一步。

  4. 建立 Hello World

    在此步驟中，你將更新 Angular 專案檔案以更改顯示的內容。

    在你的 IDE 中：

    1. 開啟 first-app/src/index.html。 注意：此步驟和下一步僅適用於你的本地環境！

    2. 在 index.html 中，將 <title> 元素替換為以下程式碼以更新應用的標題。

      在 src/index.html 中替換

            
      <!DOCTYPE html><html lang="en"><head>  <meta charset="utf-8">  <title>Homes</title>  <base href="/">  <meta name="viewport" content="width=device-width, initial-scale=1">  <link rel="icon" type="image/x-icon" href="favicon.ico">  <link href="/assets/css/be-vietnam-pro.css" rel="stylesheet"></head><body>  <app-root></app-root></body></html>

      然後，儲存你剛剛對 index.html 所做的更改。

    3. 接下來，開啟 first-app/src/app/app.component.ts。

    4. 在 app.component.ts 的 @Component 定義中，將 template 行替換為以下程式碼以更改應用元件中的文字。

      在 src/app/app.component.ts 中替換

            
      import {Component} from '@angular/core';@Component({  selector: 'app-root',  imports: [],  template: `    <h1>Hello world!</h1>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    5. 在 app.component.ts 的 AppComponent 類別定義中，將 title 行替換為以下程式碼以更改元件標題。

      在 src/app/app.component.ts 中替換

            
      import {Component} from '@angular/core';@Component({  selector: 'app-root',  imports: [],  template: `    <h1>Hello world!</h1>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

      然後，儲存你對 app.component.ts 所做的更改。

    6. 如果你在步驟 1 中停止了 ng serve 指令，在你的 IDE 的 Terminal 視窗中，再次執行 ng serve。

    7. 開啟你的瀏覽器並導向到 localhost:4200，確認該應用建立時沒有錯誤，並在應用的標題和正文中顯示 Hello world：

總結： 在本課程中，你更新了一個預設的 Angular 應用以顯示 Hello world。 在此過程中，你學習了使用 ng serve 指令在本地為你的應用啟動伺服器以進行測試。
