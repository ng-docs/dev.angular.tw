這個第一課是起點，本教學課程中的每一課都從此處開始，新增功能以建構完整的 Angular 應用程式。在這一課中，我們會更新應用程式以顯示著名的文字「Hello World」。

你將會學到

完成此課程後您所更新的應用程式確認您和您的 IDE 已準備好開始建立 Angular 應用程式。

注意：如果您使用的是嵌入式編輯器，請跳至 步驟三。 在瀏覽器遊樂場中工作時，您不需要 ng serve 來運行應用程式。其他命令，例如 ng generate 可以直接在您右邊的主控台視窗中執行。

  1. 下載預設應用程式

    先點選程式碼編輯器右上角的「下載」圖示。這會下載一個包含本教學程式碼的 .zip 檔案。在你的本地端終端機和 IDE 中開啟此檔案，然後繼續測試預設應用程式。

    在教學課程的任何步驟中，您可以點擊此圖示下載該步驟的原始程式碼，然後從該處開始。

  2. 測試預設應用程式

    在這個步驟，下載預設起始應用程式後，建立預設 Angular 應用程式。 這確認您的開發環境有繼續本教學所需的內容。

    在您的 IDE 的 Terminal 窗格中：

    1. 在您的專案目錄中，導航至 first-app 目錄。

    2. 執行此命令，以安裝執行應用程式所需的相依性。

            
      npm install

    3. 執行此命令，以建構並提供預設應用程式。

            
      ng serve

      應用程式應會在沒有錯誤的情況下建構。

    4. 在您的開發電腦上的網路瀏覽器中，開啟 http://localhost:4200。

    5. 確認預設網站出現在瀏覽器中。

    6. 您可以在完成後續步驟時，讓 ng serve 持續執行。

  3. 檢閱專案中的檔案

    在這個步驟中，您必須了解組成預設 Angular 應用程式的檔案。

    在您的 IDE 的 瀏覽器 窗格中：

    1. 在您的專案目錄中，導覽至 first-app 目錄。
    2. 開啟 src 目錄以查看這些檔案。
      1. 在檔案總管中，找到 Angular 應用程式檔案 (/src)。
        1. index.html 是應用程式的頂層 HTML 範本。
        2. style.css 是應用程式的頂層樣式表。
        3. main.ts 是應用程式開始執行的位置。
        4. favicon.ico 是應用程式的圖示，就像您在任何網站中找到的一樣。
      2. 在檔案總管中，找到 Angular 應用程式的元件檔案 (/app)。

        1. app.component.ts 是描述 app-root 元件的原始檔。 這是應用程式中的頂層 Angular 元件。元件是 Angular 應用程式的基本建構區塊。 元件說明包括元件的程式碼、HTML 範本和樣式，可以在此檔案中或在個別檔案中描述。

          在此應用程式中，樣式在個別檔案中，而元件的程式碼和 HTML 範本在此檔案中。

        2. app.component.css 是此元件的樣式表。

        3. 新的元件會新增到此目錄。

      3. 在檔案總管中，找到包含應用程式所使用影像的影像目錄 (/assets)。
      4. 在檔案總管中，找到 Angular 應用程式需要建置和執行，但您通常不會與之互動的檔案和目錄。
        1. .angular 有建置 Angular 應用程式所需的檔案。
        2. .e2e 有用於測試應用程式的檔案。
        3. .node_modules 有應用程式使用的 node.js 套件。
        4. angular.json 向應用程式建置工具說明 Angular 應用程式。
        5. package.json 由 npm (node 套件管理員) 用於執行已完成的應用程式。
        6. tsconfig.* 是向 TypeScript 編譯器說明應用程式組態的檔案。

    當您已檢閱組成 Angular 應用程式專案的檔案後，繼續至下一步。

  4. 建立 Hello World

    在此步驟中，您會更新 Angular 專案檔案，以變更顯示的內容。

    在您的 IDE 中：

    1. 開啟 first-app/src/index.html。 注意：此步驟和下一個步驟僅適用於您的本地環境！

    2. 在 index.html 中，以這個程式碼取代 <title> 元素，以更新應用程式的標題。

      在 src/index.html 中取代

            
      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="utf-8">
        <title>Default</title>
        <base href="/">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <link rel="icon" type="image/x-icon" href="favicon.ico">
        <link href="https://fonts.googleapis.com/css2?family=Be+Vietnam+Pro:ital,wght@0,400;0,700;1,400;1,700&display=swap" rel="stylesheet">
      </head>
      <body>
        <app-root></app-root>
      </body>
      </html>

      然後，儲存您剛才對 index.html 所做的變更。

    3. 接下來，開啟 first-app/src/app/app.component.ts。

    4. 在 app.component.ts 中，在 @Component 定義中，以這個程式碼取代 template 列，以變更應用程式元件中的文字。

      在 src/app/app.component.ts 中取代

            
      import {Component} from '@angular/core';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [],
        template: `
          <h1>Default</h1>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'default';
      }

    5. 在 app.component.ts 中，在 AppComponent 類別定義中，以這個程式碼取代 title 列，以變更元件標題。

      在 src/app/app.component.ts 中取代

            
      import {Component} from '@angular/core';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [],
        template: `
          <h1>Default</h1>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'default';
      }

      然後，儲存您對 app.component.ts 所做的變更。

    6. 如果您已在步驟 1 中停止 ng serve 指令，請在 IDE 的 終端機 視窗中，再次執行 ng serve。

    7. 開啟您的瀏覽器並導航至 localhost:4200，並確認應用程式在沒有錯誤的情況下建置，並在應用程式的標題和內文中顯示 Hello world：

摘要：在本課程中，您更新了預設的 Angular 應用程式，以顯示 Hello world。 在此過程中，您瞭解了 ng serve 指令，可在本地端為您的應用程式提供測試服務。
