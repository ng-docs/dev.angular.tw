向你的 Angular 應用新增表單

edit

本教程課程示範了如何向 Angular 應用新增一個收集使用者資料的表單。 本課程從一個功能正常的 Angular 應用開始，並展示如何向其新增表單。

表單收集的資料僅傳送到應用的 Service，Service 會將其寫入瀏覽器的控制台。 本課程不介紹如何使用 REST API 傳送和接收表單資料。

重要提示： 我們建議在本教程的此步驟中使用你的本地環境。

你將學到什麼

  • 你的應用有一個表單，使用者可以在其中輸入資料，這些資料將傳送到你的應用的 Service。
  • Service 將表單中的資料寫入瀏覽器的控制台日誌。

  4. 測試你的應用的新表單

    此步驟測試新表單，以檢視當表單資料送出到應用時，表單資料是否出現在控制台日誌中。

    1. 在 IDE 的 終端 窗格中，執行 ng serve（如果尚未執行）。
    2. 在瀏覽器中，在 http://localhost:4200 開啟你的應用。
    3. 在瀏覽器中右鍵單擊應用，然後從上下文選單中選擇 檢查。
    4. 在開發者工具視窗中，選擇 控制台 頁籤。 確保開發者工具視窗在後續步驟中可見
    5. 在你的應用中：
      1. 選擇一個房屋地點，然後單擊 瞭解更多，以檢視有關該房屋的詳細訊息。
      2. 在該房屋的詳情頁面中，捲動到底部以找到新表單。
      3. 在表單的欄位中輸入資料 - 任何資料都可以。
      4. 選擇 立即申請 以送出資料。
    6. 在開發者工具視窗中，檢視日誌輸出以查詢你的表單資料。
