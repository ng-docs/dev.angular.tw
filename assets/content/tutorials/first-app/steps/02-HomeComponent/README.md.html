建立 Home 元件

edit

本教程課程示範了如何為你的 Angular 應用建立一個新的元件。

你將學到什麼

你的應用有了一個新元件：HomeComponent。

Angular 元件的概念預覽

Angular 應用是圍繞元件建立的，元件是 Angular 的建立塊。 元件包含程式碼、HTML 佈局和 CSS 樣式訊息，這些訊息提供了應用中元素的功能和外觀。 在 Angular 中，元件可以包含其他元件。應用的各項功能和外觀可以劃分並分割成元件。

在 Angular 中，元件具有元資料來定義其屬性（Property）。 當你建立 HomeComponent 時，你將使用以下屬性（Property）：

  • selector：用於描述 Angular 如何在樣板中引用元件。
  • standalone：用於描述元件是否需要 NgModule。
  • imports：用於描述元件的依賴項。
  • template：用於描述元件的 HTML 標記和佈局。
  • styleUrls：用於在陣列中列出元件使用的 CSS 檔案的 URL。

  1. 建立 HomeComponent

    在此步驟中，你將為你的應用建立一個新元件。

    在 IDE 的 終端 窗格中：

    1. 在你的專案目錄中，導向到 first-app 目錄。

    2. 執行此指令以建立一個新的 HomeComponent

            
      ng generate component home

    3. 執行此指令以建立併為你的應用啟動伺服器。

      注意：此步驟僅適用於你的本地環境！

            
      ng serve

    4. 開啟瀏覽器並導向到 http://localhost:4200 以查詢該應用。

    5. 確認應用建立時沒有錯誤。

      提示：它應該渲染得和上一課中一樣，因為即使你新增了一個新元件，你還沒有將其包含在應用的任何樣板中。

    6. 在完成後續步驟時，保持 ng serve 執行。

  2. 將新元件新增到應用的佈局中

    在此步驟中，你將新元件 HomeComponent 新增到應用的根元件 AppComponent 中，以便它顯示在應用的佈局中。

    在 IDE 的 編輯 窗格中：

    1. 在編輯器中開啟 app.component.ts。

    2. 在 app.component.ts 中，透過將此行新增到檔案級匯入來匯入 HomeComponent。

      在 src/app/app.component.ts 中匯入 HomeComponent

            
      import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';@Component({  selector: 'app-root',  imports: [HomeComponent],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    3. 在 app.component.ts 的 @Component 中，更新 imports 陣列屬性並新增 HomeComponent。

      在 src/app/app.component.ts 中替換

            
      import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';@Component({  selector: 'app-root',  imports: [HomeComponent],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    4. 在 app.component.ts 的 @Component 中，更新 template 屬性以包含以下 HTML 程式碼。

      在 src/app/app.component.ts 中替換

            
      import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';@Component({  selector: 'app-root',  imports: [HomeComponent],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    5. 將更改儲存到 app.component.ts。

    6. 如果 ng serve 正在執行，應用應該會更新。 如果 ng serve 沒有執行，請重新啟動它。 應用中的 Hello world 應該會從 HomeComponent 更改為 home works!。

    7. 在瀏覽器中檢查正在執行的應用，並確認應用已更新。

  3. 新增屬性到 HomeComponent

    在此步驟中，你將為 HomeComponent 新增屬性。

    在上一步中，你將預設的 HomeComponent 新增到應用的樣板中，因此其預設 HTML 出現在應用中。 在此步驟中，你將新增一個搜尋過濾器和按鈕，它們將在後麵的課程中使用。 目前，這就是 HomeComponent 的全部內容。 請注意，此步驟僅將搜尋元素新增到佈局中，但尚不具備任何功能。

    在 IDE 的 編輯 窗格中：

    1. 在 first-app 目錄中，在編輯器中開啟 home.component.ts。

    2. 在 home.component.ts 的 @Component 中，使用此程式碼更新 template 屬性。

      在 src/app/home/home.component.ts 中替換

            
      import {Component} from '@angular/core';import {CommonModule} from '@angular/common';@Component({  selector: 'app-home',  imports: [CommonModule],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {}

    3. 接下來，在編輯器中開啟 home.component.css 並使用這些樣式更新內容。

      注意：在瀏覽器中，這些可以放在 src/app/home/home.component.ts 的 styles 陣列中。

      在 src/app/home/home.component.css 中替換

            
      .results {  display: grid;  column-gap: 14px;  row-gap: 14px;  grid-template-columns: repeat(auto-fill, minmax(400px, 400px));  margin-top: 50px;  justify-content: space-around;}input[type="text"] {  border: solid 1px var(--primary-color);  padding: 10px;  border-radius: 8px;  margin-right: 4px;  display: inline-block;  width: 30%;}button {  padding: 10px;  border: solid 1px var(--primary-color);  background: var(--primary-color);  color: white;  border-radius: 8px;}@media (min-width: 500px) and (max-width: 768px) {  .results {      grid-template-columns: repeat(2, 1fr);  }  input[type="text"] {      width: 70%;  }   }@media (max-width: 499px) {  .results {      grid-template-columns: 1fr;  }    }

    4. 確認應用建立時沒有錯誤。 你應該在你的應用中找到過濾器查詢框和按鈕，並且它們應該已設定樣式。 在繼續下一步之前，請糾正任何錯誤。

摘要： 在本課程中，你為你的應用建立了一個新元件，併為其新增了一個過濾器編輯控制元件和按鈕。
