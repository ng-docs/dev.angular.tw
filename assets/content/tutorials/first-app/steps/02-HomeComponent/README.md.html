创建主页组件

edit

本教程演示了如何为你的 Angular 应用程序创建一个新的 组件。

你将学到什么

应用程序有了一个新组件：HomeComponent。

Angular 组件的概念性预览

Angular 应用程序围绕组件构建，这些组件是 Angular 的构建块。 组件包含提供应用程序中元素的功能和外观的代码、HTML 布局和 CSS 样式信息。 在 Angular 中，组件可以包含其他组件。应用程序的功能和外观可以被划分和分隔成组件。

在 Angular 中，组件具有一些用来定义其属性的元数据。当你创建 HomeComponent 时，会用到这些属性：

  • selector：描述了 Angular 要如何在模板中引用本组件。
  • standalone：描述组件是否需要一个 NgModule。
  • imports：描述组件的依赖关系。
  • template：描述组件的 HTML 标记和布局。
  • styleUrls：以数组形式列出组件使用的 CSS 文件的 URL。

  1. 创建 HomeComponent

    在此步骤中，你将为应用程序创建一个新组件。

    在你的 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令以创建一个新的 HomeComponent

            
      ng generate component home

    3. 运行此命令以构建并启动应用开发服务器。

      注意：此步骤仅适用于你的本地环境！

            
      ng serve

    4. 打开浏览器并导航到 http://localhost:4200 以找到该应用程序。

    5. 确认应用构建时没有错误。

      提示：它应该与上一课中的渲染结果相同，因为即使你添加了一个新组件，但尚未将其包含在任何应用模板中。

    6. 在完成后续步骤时让 ng serve 保持运行。

  2. 将新组件添加到你应用的布局

    在此步骤中，你要将新组件 HomeComponent 添加到应用的根组件 AppComponent 中，以便它能显示在应用的布局中。

    在 IDE 的编辑窗格中：

    1. 在编辑器中打开 app.component.ts。

    2. 在 app.component.ts 中，通过将此行添加到文件顶部来导入 HomeComponent。

      在 src/app/app.component.ts 中导入 HomeComponent

            
      import {Component} from '@angular/core';
      import {HomeComponent} from './home/home.component';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [HomeComponent],
        template: `
          <main>
            <header class="brand-name">
              <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />
            </header>
            <section class="content">
              <app-home></app-home>
            </section>
          </main>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'homes';
      }

    3. 在 app.component.ts中，在 @Component中，更新 imports数组属性，并添加 HomeComponent。

      在 src/app/app.component.ts 中替换

            
      import {Component} from '@angular/core';
      import {HomeComponent} from './home/home.component';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [HomeComponent],
        template: `
          <main>
            <header class="brand-name">
              <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />
            </header>
            <section class="content">
              <app-home></app-home>
            </section>
          </main>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'homes';
      }

    4. 在 app.component.ts中，在 @Component中，更新 template属性以包含以下 HTML 代码。

      在 src/app/app.component.ts 中替换

            
      import {Component} from '@angular/core';
      import {HomeComponent} from './home/home.component';
      @Component({
        selector: 'app-root',
        standalone: true,
        imports: [HomeComponent],
        template: `
          <main>
            <header class="brand-name">
              <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />
            </header>
            <section class="content">
              <app-home></app-home>
            </section>
          </main>
        `,
        styleUrls: ['./app.component.css'],
      })
      export class AppComponent {
        title = 'homes';
      }

    5. 保存对 app.component.ts 的更改。

    6. 如果 ng serve正在运行，则应用程序应更新。 如果 ng serve未运行，请重新启动。 你应用中的 Hello world应该从 HomeComponent更改为 home works!。

    7. 在浏览器中检查正在运行的应用程序并确认该应用程序已更新。

  3. 为 HomeComponent 添加特性

    在此步骤中，你将向 HomeComponent 添加一些特性。

    在前一步中，你向你的应用模板添加了默认的 HomeComponent，因此其默认 HTML 出现在应用中。 在本步骤中，你将添加一个搜索过滤器和按钮，该过滤器和按钮在后续课程中将被使用。 目前， HomeComponent仅具有这些内容。 请注意，此步骤仅将搜索元素添加到布局中，尚未具备任何功能。

    在 IDE 的编辑窗格中：

    1. 在 first-app 目录下，在编辑器中打开 home.component.ts。

    2. 在 home.component.ts中，在 @Component中，使用此代码更新 template属性。

      在 src/app/home/home.component.ts 中替换

            
      import {Component} from '@angular/core';
      import {CommonModule} from '@angular/common';
      @Component({
        selector: 'app-home',
        standalone: true,
        imports: [CommonModule],
        template: `
          <section>
            <form>
              <input type="text" placeholder="Filter by city" />
              <button class="primary" type="button">Search</button>
            </form>
          </section>
        `,
        styleUrls: ['./home.component.css'],
      })
      export class HomeComponent {}

    3. 接下来，在编辑器中打开 home.component.css 并使用如下样式更新其内容。

      注意：在浏览器中，这些内容可以放在 src/app/home/home.component.ts中的 styles数组中。

      在 src/app/home/home.component.css 中替换

            
      .results {
        display: grid;
        column-gap: 14px;
        row-gap: 14px;
        grid-template-columns: repeat(auto-fill, minmax(400px, 400px));
        margin-top: 50px;
        justify-content: space-around;
      }
      input[type="text"] {
        border: solid 1px var(--primary-color);
        padding: 10px;
        border-radius: 8px;
        margin-right: 4px;
        display: inline-block;
        width: 30%;
      }
      button {
        padding: 10px;
        border: solid 1px var(--primary-color);
        background: var(--primary-color);
        color: white;
        border-radius: 8px;
      }
      @media (min-width: 500px) and (max-width: 768px) {
        .results {
            grid-template-columns: repeat(2, 1fr);
        }
        input[type="text"] {
            width: 70%;
        }   
      }
      @media (max-width: 499px) {
        .results {
            grid-template-columns: 1fr;
        }    
      }

    4. 确认应用程序构建无误。 你应该在应用中找到过滤查询框和按钮，并且它们应该有样式。 在继续下一步之前纠正任何错误。

总结：在本课中，你为你的应用创建了一个新组件，并为其添加了一个过滤编辑控件和按钮。
