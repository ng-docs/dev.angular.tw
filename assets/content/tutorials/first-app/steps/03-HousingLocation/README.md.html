建立應用的 HousingLocation 元件

edit

本教學課程課程示範了如何將 HousingLocation 元件新增到你的 Angular 應用中。

你將學到什麼

  • 你的應用有了一個新元件：HousingLocationComponent，並且它顯示一條訊息，確認該組件已新增到你的應用中。

  1. 建立 HousingLocationComponent

    在此步驟中，你將為你的應用建立一個新元件。

    在你的 IDE 的 終端 窗格中：

    1. 在你的專案目錄中，導向到 first-app 目錄。

    2. 執行此指令以建立一個新的 HousingLocationComponent

            
      ng generate component housingLocation

    3. 執行此指令以建立併為你的應用啟動伺服器。

            
      ng serve

      注意：此步驟僅適用於你的本地環境！

    4. 開啟瀏覽器並導向到 http://localhost:4200 以查詢該應用。

    5. 確認該應用建立時沒有錯誤。

      提示：它應該渲染得和上一課中一樣，因為即使你新增了一個新元件，你還沒有把它包含在應用的任何範本中。

    6. 在完成後續步驟時，保持 ng serve 執行。

  2. 將新元件新增到應用的佈局中

    在此步驟中，你將新的元件 HousingLocationComponent 新增到應用的 HomeComponent 中，以便它顯示在應用的佈局中。

    在你的 IDE 的 編輯 窗格中：

    1. 在編輯器中開啟 home.component.ts。

    2. 在 home.component.ts 中，透過將此行新增到檔案級匯入來匯入 HousingLocationComponent。

      在 src/app/home/home.component.ts 中匯入 HousingLocationComponent

            
      import {Component} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocationComponent} from '../housing-location/housing-location.component';@Component({  selector: 'app-home',  imports: [CommonModule, HousingLocationComponent],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {}

    3. 接下來，透過將 HousingLocationComponent 新增到陣列中，來更新 @Component 元資料的 imports 屬性。

      將 HousingLocationComponent 新增到 src/app/home/home.component.ts 中的 imports 陣列

            
      import {Component} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocationComponent} from '../housing-location/housing-location.component';@Component({  selector: 'app-home',  imports: [CommonModule, HousingLocationComponent],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {}

    4. 現在，該組件已準備好在 HomeComponent 的範本中使用。更新 @Component 元資料的 template 屬性，以包含對 <app-housing-location> 標籤的引用。

      將房屋位置新增到 src/app/home/home.component.ts 中的元件範本

            
      import {Component} from '@angular/core';import {CommonModule} from '@angular/common';import {HousingLocationComponent} from '../housing-location/housing-location.component';@Component({  selector: 'app-home',  imports: [CommonModule, HousingLocationComponent],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>    <section class="results">      <app-housing-location></app-housing-location>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {}

  3. 為元件新增樣式

    在此步驟中，你將把預先寫好的 HousingLocationComponent 的樣式複製到你的應用中，以便該應用能正確渲染。

    1. 開啟 src/app/housing-location/housing-location.component.css，並將以下樣式貼上到檔案中：

      注意：在瀏覽器中，這些可以放在 src/app/housing-location/housing-location.component.ts 的 styles 陣列中。

      將 CSS 樣式新增到 src/app/housing-location/housing-location.component.css 中元件的房屋位置

            
      .listing {  background: var(--accent-color);  border-radius: 30px;  padding-bottom: 30px;}.listing-heading {  color: var(--primary-color);  padding: 10px 20px 0 20px;}.listing-photo {  height: 250px;  width: 100%;  object-fit: cover;  border-radius: 30px 30px 0 0;}.listing-location {  padding: 10px 20px 20px 20px;}.listing-location::before {  content: url("/assets/location-pin.svg") / "";}section.listing a {  padding-left: 20px;  text-decoration: none;  color: var(--primary-color);}section.listing a::after {  content: "\203A";  margin-left: 5px;}

    2. 儲存你的程式碼，回傳到瀏覽器並確認該應用建立時沒有錯誤。你應該看到訊息 『housing-location works!』 渲染到螢幕上。在繼續下一步之前，請糾正任何錯誤。

摘要： 在本課程中，你為你的應用建立了一個新元件，並將其新增到應用的佈局中。
