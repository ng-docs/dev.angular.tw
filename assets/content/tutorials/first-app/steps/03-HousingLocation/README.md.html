创建应用的 HousingLocation 组件

edit

本教程示范了如何将 HousingLocation 组件添加到 Angular 应用程序中。

你将学到什么

  • 应用程序有一个新组件：HousingLocationComponent，它会显示一条消息，以确认该组件成功添加到了应用程序中。

  1. 创建 HousingLocationComponent

    在此步骤中，你将为应用程序创建一个新组件。

    在你的 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令以创建一个新的 HousingLocationComponent

            
      ng generate component housingLocation

    3. 运行此命令以构建并启动应用开发服务器。

            
      ng serve

      注意：此步骤仅适用于你的本地环境！

    4. 打开浏览器并导航到 http://localhost:4200 以找到该应用程序。

    5. 确认应用构建时没有错误。

      提示：它应该与上一课中的渲染结果相同，因为即使你添加了一个新组件，但尚未将其包含在任何应用模板中。

    6. 在完成后续步骤时让 ng serve 保持运行。

  2. 将新组件添加到你应用的布局

    在此步骤中，你将新组件 HousingLocationComponent 添加到应用的 HomeComponent 中，以便让它显示在应用的布局中。

    在 IDE 的编辑窗格中：

    1. 在编辑器中打开 home.component.ts。

    2. 在 home.component.ts 中，通过将此行添加到文件顶部来导入 HousingLocationComponent。

      在 src/app/home/home.component.ts 中导入 HousingLocationComponent

            
      import {Component} from '@angular/core';
      import {CommonModule} from '@angular/common';
      import {HousingLocationComponent} from '../housing-location/housing-location.component';
      @Component({
        selector: 'app-home',
        standalone: true,
        imports: [CommonModule, HousingLocationComponent],
        template: `
          <section>
            <form>
              <input type="text" placeholder="Filter by city" />
              <button class="primary" type="button">Search</button>
            </form>
          </section>
          <section class="results">
            <app-housing-location></app-housing-location>
          </section>
        `,
        styleUrls: ['./home.component.css'],
      })
      export class HomeComponent {}

    3. 接下来，通过将 HousingLocationComponent 添加到数组中，更新 @Component 元数据的 imports 属性。

      在 src/app/home/home.component.ts 中的 imports 数组中添加 HousingLocationComponent。

            
      import {Component} from '@angular/core';
      import {CommonModule} from '@angular/common';
      import {HousingLocationComponent} from '../housing-location/housing-location.component';
      @Component({
        selector: 'app-home',
        standalone: true,
        imports: [CommonModule, HousingLocationComponent],
        template: `
          <section>
            <form>
              <input type="text" placeholder="Filter by city" />
              <button class="primary" type="button">Search</button>
            </form>
          </section>
          <section class="results">
            <app-housing-location></app-housing-location>
          </section>
        `,
        styleUrls: ['./home.component.css'],
      })
      export class HomeComponent {}

    4. 现在，该组件已经准备好在 HomeComponent 的模板中使用了。更新 @Component 元数据的 template 属性，包含对 <app-housing-location> 标签的引用。

      在 src/app/home/home.component.ts 中的组件模板中添加 housing location。

            
      import {Component} from '@angular/core';
      import {CommonModule} from '@angular/common';
      import {HousingLocationComponent} from '../housing-location/housing-location.component';
      @Component({
        selector: 'app-home',
        standalone: true,
        imports: [CommonModule, HousingLocationComponent],
        template: `
          <section>
            <form>
              <input type="text" placeholder="Filter by city" />
              <button class="primary" type="button">Search</button>
            </form>
          </section>
          <section class="results">
            <app-housing-location></app-housing-location>
          </section>
        `,
        styleUrls: ['./home.component.css'],
      })
      export class HomeComponent {}

  3. 为组件添加样式

    在此步骤中，你会将 HousingLocationComponent 的预写样式复制到应用程序中，以便应用程序能正确渲染。

    1. 打开 src/app/housing-location/housing-location.component.css，并将下面的样式粘贴到文件中：

      注意：在浏览器中，这些样式可以放在 src/app/housing-location/housing-location.component.ts 的 styles 数组中。

      在 src/app/housing-location/housing-location.component.css 中为 housing location 组件添加 CSS 样式。

            
      .listing {
        background: var(--accent-color);
        border-radius: 30px;
        padding-bottom: 30px;
      }
      .listing-heading {
        color: var(--primary-color);
        padding: 10px 20px 0 20px;
      }
      .listing-photo {
        height: 250px;
        width: 100%;
        object-fit: cover;
        border-radius: 30px 30px 0 0;
      }
      .listing-location {
        padding: 10px 20px 20px 20px;
      }
      .listing-location::before {
        content: url("/assets/location-pin.svg") / "";
      }
      section.listing a {
        padding-left: 20px;
        text-decoration: none;
        color: var(--primary-color);
      }
      section.listing a::after {
        content: "\203A";
        margin-left: 5px;
      }

    2. 保存你的代码，返回浏览器并确认应用程序构建没有错误。你应该看到屏幕上渲染出了消息 “housing-location works!”。在继续下一步之前更正所有错误。

摘要：在本课中，你为应用程序创建了一个新组件，并将其添加到应用程序的布局中。
