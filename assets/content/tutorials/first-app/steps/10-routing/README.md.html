向应用添加路由

edit

本课程示范了如何将路由添加到应用程序。

重要： 我们建议使用本地环境学习路由。

你将学到什么

在这节课结束时，应用程序将支持路由。

路由的概念性预览

本教程介绍了 Angular 中的路由。路由是从应用中的一个组件导航到另一个组件的能力。在 单页应用（SPA） 中，只有页面的部分内容会更新，以呈现用户请求的视图。

Angular 路由器允许用户声明路由，并指定当应用请求该路由时应在屏幕上显示的组件。

在这节课中，你将在应用程序中启用路由以导航到详情页。

  1. 创建默认的详情组件

    1. 从终端输入以下命令以创建 DetailsComponent：

            
      ng generate component details

      该组件将展现详情页面，该页面提供有关给定房屋位置的更多信息。

  2. 为应用添加路由

    1. 在 src/app 目录中，创建一个名为 routes.ts 的文件。该文件是我们将在应用程序中定义路由的地方。

    2. 在 main.ts 中，进行以下更新以在应用程序中启用路由：

      1. 导入路由文件和 provideRouter 函数：

        在 src/main.ts 中导入路由详细信息

              
        /*
         *  Protractor support is deprecated in Angular.
         *  Protractor is used in this example for compatibility with Angular documentation tools.
         */
        import {bootstrapApplication, provideProtractorTestingSupport} from '@angular/platform-browser';
        import {AppComponent} from './app/app.component';
        import {provideRouter} from '@angular/router';
        import routeConfig from './app/routes';
        bootstrapApplication(AppComponent, {
          providers: [provideProtractorTestingSupport(), provideRouter(routeConfig)],
        }).catch((err) => console.error(err));

      2. 更新对 bootstrapApplication 的调用以包含路由配置：

        在 src/main.ts 中添加路由器配置

              
        /*
         *  Protractor support is deprecated in Angular.
         *  Protractor is used in this example for compatibility with Angular documentation tools.
         */
        import {bootstrapApplication, provideProtractorTestingSupport} from '@angular/platform-browser';
        import {AppComponent} from './app/app.component';
        import {provideRouter} from '@angular/router';
        import routeConfig from './app/routes';
        bootstrapApplication(AppComponent, {
          providers: [provideProtractorTestingSupport(), provideRouter(routeConfig)],
        }).catch((err) => console.error(err));

    3. 在 src/app/app.component.ts 中，更新组件以使用路由：

      1. 为 RoutingModule 添加文件级导入：

        在 src/app/app.component.ts 中导入 RouterModule

              
        import {Component} from '@angular/core';
        import {HomeComponent} from './home/home.component';
        import {RouterModule} from '@angular/router';
        @Component({
          selector: 'app-root',
          standalone: true,
          imports: [HomeComponent, RouterModule],
          template: `
            <main>
              <a [routerLink]="['/']">
                <header class="brand-name">
                  <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />
                </header>
              </a>
              <section class="content">
                <router-outlet></router-outlet>
              </section>
            </main>
          `,
          styleUrls: ['./app.component.css'],
        })
        export class AppComponent {
          title = 'homes';
        }

      2. 在 @Component 元数据的 imports 中添加 RouterModule

        在 src/app/app.component.ts 中导入 RouterModule

              
        import {Component} from '@angular/core';
        import {HomeComponent} from './home/home.component';
        import {RouterModule} from '@angular/router';
        @Component({
          selector: 'app-root',
          standalone: true,
          imports: [HomeComponent, RouterModule],
          template: `
            <main>
              <a [routerLink]="['/']">
                <header class="brand-name">
                  <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />
                </header>
              </a>
              <section class="content">
                <router-outlet></router-outlet>
              </section>
            </main>
          `,
          styleUrls: ['./app.component.css'],
        })
        export class AppComponent {
          title = 'homes';
        }

      3. 在 template 属性中，用 <router-outlet> 指令替换 <app-home></app-home> 标签，并添加一个返回主页的链接。你的代码应该与下列代码匹配：

        在 src/app/app.component.ts 中添加 router-outlet

              
        import {Component} from '@angular/core';
        import {HomeComponent} from './home/home.component';
        import {RouterModule} from '@angular/router';
        @Component({
          selector: 'app-root',
          standalone: true,
          imports: [HomeComponent, RouterModule],
          template: `
            <main>
              <a [routerLink]="['/']">
                <header class="brand-name">
                  <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />
                </header>
              </a>
              <section class="content">
                <router-outlet></router-outlet>
              </section>
            </main>
          `,
          styleUrls: ['./app.component.css'],
        })
        export class AppComponent {
          title = 'homes';
        }

  3. 为新组件添加路由

    在上一步中，你删除了模板中对 <app-home> 组件的引用。在此步骤中，你将向该组件添加一条新路由。

    1. 在 routes.ts 中，执行以下更新以创建路由。

      1. 为 HomeComponent、 DetailsComponent 和你将在路由定义中使用的 Routes 类型添加文件级别的导入。

        导入组件和 Routes

              
        import {Routes} from '@angular/router';
        import {HomeComponent} from './home/home.component';
        import {DetailsComponent} from './details/details.component';
        const routeConfig: Routes = [
          {
            path: '',
            component: HomeComponent,
            title: 'Home page',
          },
          {
            path: 'details/:id',
            component: DetailsComponent,
            title: 'Home details',
          },
        ];
        export default routeConfig;

      2. 定义一个名为 routeConfig 的变量，其类型为 Routes，并为应用定义两个路由：

        为应用添加路由

              
        import {Routes} from '@angular/router';
        import {HomeComponent} from './home/home.component';
        import {DetailsComponent} from './details/details.component';
        const routeConfig: Routes = [
          {
            path: '',
            component: HomeComponent,
            title: 'Home page',
          },
          {
            path: 'details/:id',
            component: DetailsComponent,
            title: 'Home details',
          },
        ];
        export default routeConfig;

        routeConfig 数组中的条目表示应用程序中的路由。只要 url 与 '' 匹配，第一个条目就会导航到 HomeComponent。第二个条目使用了一些特殊的格式，将在以后的课程中重新讨论。

    2. 保存所有更改并确认应用程序正在浏览器中运行。该应用程序应该仍旧显示着房屋位置列表。

总结：在本课中，你在应用中启用了路由并定义了新的路由。现在你的应用可以支持视图之间的导航。在下一课中，你将学习如何导航到给定房源位置的「详情」页面。
