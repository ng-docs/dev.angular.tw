為應用新增路由

edit

本教程課程示範了如何為你的應用新增路由。

重要提示： 我們建議使用本地環境來學習路由。

你將學到什麼

在本課程結束時，你的應用將支援路由。

路由的概念預覽

本教程介紹了 Angular 中的路由。路由是指從應用中的一個元件導向到另一個元件的能力。在單頁應用（SPA）中，只會更新頁面的部分內容，以呈現使用者所請求的檢視。

Angular 路由器戶能夠宣告路由，並指定當應用請求該路由時，應在螢幕上顯示哪個元件。

在本課程中，你將在你的應用中啟用路由，以便導向到詳情頁面。

  1. 建立一個預設的詳情元件

    1. 從終端中，輸入以下指令來建立 DetailsComponent：

            
      ng generate component details

      此元件將代表詳情頁面，其中提供了有關給定房屋地點的更多訊息。

  2. 為應用新增路由

    1. 在 src/app 目錄下，建立一個名為 routes.ts 的檔案。我們將在該檔案中定義應用中的路由。

    2. 在 main.ts 中，進行以下更新以在應用中啟用路由：

      1. 匯入路由檔案和 provideRouter 函式：

        在 src/main.ts 中匯入路由詳情

              
        /* *  Protractor support is deprecated in Angular. *  Protractor is used in this example for compatibility with Angular documentation tools. */import {bootstrapApplication, provideProtractorTestingSupport} from '@angular/platform-browser';import {AppComponent} from './app/app.component';import {provideRouter} from '@angular/router';import routeConfig from './app/routes';bootstrapApplication(AppComponent, {  providers: [provideProtractorTestingSupport(), provideRouter(routeConfig)],}).catch((err) => console.error(err));

      2. 更新對 bootstrapApplication 的呼叫，以包含路由設定：

        在 src/main.ts 中新增路由器設定

              
        /* *  Protractor support is deprecated in Angular. *  Protractor is used in this example for compatibility with Angular documentation tools. */import {bootstrapApplication, provideProtractorTestingSupport} from '@angular/platform-browser';import {AppComponent} from './app/app.component';import {provideRouter} from '@angular/router';import routeConfig from './app/routes';bootstrapApplication(AppComponent, {  providers: [provideProtractorTestingSupport(), provideRouter(routeConfig)],}).catch((err) => console.error(err));

    3. 在 src/app/app.component.ts 中，更新元件以使用路由：

      1. 為 RoutingModule 新增檔案級匯入：

        在 src/app/app.component.ts 中匯入 RouterModule

              
        import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';import {RouterModule} from '@angular/router';@Component({  selector: 'app-root',  imports: [HomeComponent, RouterModule],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

      2. 將 RouterModule 新增到 @Component 元資料 imports 中

        在 src/app/app.component.ts 中匯入 RouterModule

              
        import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';import {RouterModule} from '@angular/router';@Component({  selector: 'app-root',  imports: [HomeComponent, RouterModule],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

      3. 在 template 屬性中，將 <app-home></app-home> 標籤替換為 <router-outlet> 指令，並新增一個回傳首頁的連結。你的程式碼應與此程式碼對應：

        在 src/app/app.component.ts 中新增 router-outlet

              
        import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';import {RouterModule} from '@angular/router';@Component({  selector: 'app-root',  imports: [HomeComponent, RouterModule],  template: `    <main>      <a [routerLink]="['/']">        <header class="brand-name">          <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />        </header>      </a>      <section class="content">        <router-outlet></router-outlet>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

  3. 為新元件新增路由

    在上一步中，你移除了樣板中對 <app-home> 元件的引用。在這一步中，你將為該組件新增一個新的路由。

    1. 在 routes.ts 中，執行以下更新以建立一個路由。

      1. 為 HomeComponent、DetailsComponent 和 Routes 型別新增檔案級匯入，你將在路由定義中使用它們。

        匯入元件和路由

              
        import {Routes} from '@angular/router';import {HomeComponent} from './home/home.component';import {DetailsComponent} from './details/details.component';const routeConfig: Routes = [  {    path: '',    component: HomeComponent,    title: 'Home page',  },  {    path: 'details/:id',    component: DetailsComponent,    title: 'Home details',  },];export default routeConfig;

      2. 定義一個名為 routeConfig 的變數，其型別為 Routes，併為應用定義兩個路由：

        嚮應用新增路由

              
        import {Routes} from '@angular/router';import {HomeComponent} from './home/home.component';import {DetailsComponent} from './details/details.component';const routeConfig: Routes = [  {    path: '',    component: HomeComponent,    title: 'Home page',  },  {    path: 'details/:id',    component: DetailsComponent,    title: 'Home details',  },];export default routeConfig;

        routeConfig 陣列中的條目表示應用中的路由。第一個條目在 URL 對應 '' 時導向到 HomeComponent。第二個條目使用了一些特殊的格式，這將在以後的課程中重新討論。

    2. 儲存所有更改並確認應用在瀏覽器中工作正常。應用應仍然顯示房屋位置清單。

摘要： 在本課程中，你啟用了應用中的路由，並定義了新的路由。現在你的應用可以支援在檢視之間導向。在下一課中，你將學習如何導向到給定房屋位置的『詳情』頁面。

你的應用進展順利，做得很好。
