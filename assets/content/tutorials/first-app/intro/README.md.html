建立你的第一個 Angular 應用

edit

本教程包含多個課程，它們將介紹開始使用 Angular 編碼所需的 Angular 概念。

你可以根據自己的意願選擇學習任意數量的課程，並且可以按任何順序學習。

提示： 喜歡影片？我們還為本教程準備了完整的 YouTube 課程！

開始之前

為了獲得最佳的教程體驗，請檢視以下要求，確保你已做好充分準備。

你的經驗

本教程中的課程假定你具有以下經驗：

  1. 透過直接編輯 HTML 建立了 HTML 網頁。
  2. 使用 JavaScript 編寫過網站內容。
  3. 閱讀過層疊樣式表 (CSS) 內容，並瞭解選擇器的使用方式。
  4. 使用過指令行指令在電腦上執行任務。

你的裝置

這些課程可以使用 Angular 工具的本地安裝或在我們的嵌入式編輯器中完成。本地 Angular 開發可以在 Windows、MacOS 或基於 Linux 的系統上完成。

注意： 請留意像這樣的提示，它們會指出可能僅適用於本地編輯器的步驟。

你的第一個 Angular 應用的概念預覽

本教程中的課程將建立一個 Angular 應用，用於列出租房並顯示各個房屋的詳細訊息。 此應用使用了許多 Angular 應用通用的屬性。

本地開發環境

注意： 此步驟僅適用於你的本地環境！

在你想要用於本教程的電腦上的指令行工具中執行這些步驟。

  1. 確定 Angular 所需的 node.js 版本

    Angular 需要 Node 的 active LTS 或 maintenance LTS 版本。讓我們確認你的 node.js 版本。有關特定版本要求的訊息，請參閱 package.json 檔案中的 engines 屬性。

    從 終端 視窗中：

    1. 執行以下指令：node --version
    2. 確認顯示的版本號符合要求。

  2. 安裝適用於 Angular 的正確版本的 node.js

    如果你沒有安裝 node.js 版本，請按照 nodejs.org 上的安裝說明進行安裝

  3. 安裝最新版本的 Angular

    在安裝了 node.js 和 npm 之後，下一步是安裝 Angular CLI，它為高效的 Angular 開發提供了工具。

    從 終端 視窗執行以下指令：npm install -g @angular/cli。

  4. 安裝整合開發環境 (IDE)

    你可以自由使用任何你喜歡的工具來建立 Angular 應用。我們推薦以下工具：

    1. Visual Studio Code
    2. 作為一個可選但推薦的步驟，你可以透過安裝 Angular 語言服務來進一步改善你的開發者體驗
    3. WebStorm
