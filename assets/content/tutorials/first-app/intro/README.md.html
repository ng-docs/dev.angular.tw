构建你的第一个 Angular 应用

edit

本教程由一些介绍 Angular 概念的课程组成，你需要了解这些概念以开始在 Angular 中编写代码。

你可以根据需要完成任意数量的课程，并且可以按任何顺序进行。

提示： 喜欢视频？我们还为本教程提供了完整的 YouTube 课程！

开始之前

为了获得本教程的最佳体验，请查看这些要求以确保你拥有成功所需的条件。

你的经验

本教程中的课程假设你具备以下经验：

  1. 通过直接编辑 HTML 创建了一个 HTML 网页。
  2. 使用 JavaScript 编写了网站内容。
  3. 阅读级联样式表（CSS）内容并理解选择器的使用方法。
  4. 使用命令行指令在计算机上执行任务。

你的设备

这些课程可以使用本地安装的 Angular 工具或我们嵌入的编辑器完成。本地 Angular 开发可以在 Windows、MacOS 或基于 Linux 的系统上完成。

注意：请注意类似这样的警告，它们指出的步骤可能仅适用于你的本地编辑器。

你的第一个 Angular 应用的概念性预览

本教程中的课程创建了一个列出出租房屋并显示各个房屋详细信息的 Angular 应用。这个应用使用了许多 Angular 应用中常见的特性。

本地开发环境

注意：此步骤仅适用于你的本地环境！

在你要用于本教程的计算机上的命令行工具中执行以下步骤。

  1. 识别 Angular 所需的 node.js 版本

    Angular 需要一个处于活跃 LTS 或维护 LTS 版本的 Node。让我们确认你的 node.js 版本。有关特定版本要求的信息，请参见 package.json 文件中的 engines 属性。

    从一个 终端窗口：

    1. 运行以下命令： node --version
    2. 确认显示的版本号符合要求。

  2. 安装适用于 Angular 的正确版本的 node.js

    如果你尚未安装 node.js 版本，请按照 nodejs.org 上的安装说明进行操作。

  3. 安装最新版本的 Angular

    安装了 node.js 和 npm 后，下一步是安装提供有效 Angular 开发工具的 Angular CLI。

    从一个 终端窗口运行以下命令： npm install -g @angular/cli。

  4. 安装集成开发环境 (IDE)

    你可以自由选择任何工具来构建 Angular 应用。我们推荐以下工具：

    1. Visual Studio Code
    2. 作为一个可选但建议的步骤，你可以通过安装 Angular 语言服务来进一步提升开发体验。
