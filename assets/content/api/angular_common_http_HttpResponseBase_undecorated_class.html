constructor

HttpResponseBase

所有回應的父類別建構式函式。

接受的唯一引數是一個初始化雜湊。在那裡傳遞的回應的任何屬性都將重寫預設值。

@paraminit{ headers?: HttpHeaders | undefined; status?: number | undefined; statusText?: string | undefined; url?: string | undefined; }
@paramdefaultStatusnumber
@paramdefaultStatusTextstring
@returnsHttpResponseBase

headers

HttpHeaders

所有回應標頭。

status

number

回應狀態碼。

statusText

string

回應狀態碼的文字描述，預設為 OK。

不要依賴它。

url

string | null

檢索到的資源的 URL；如果不可用，則為 null。

ok

boolean

狀態碼是否在 2xx 範圍內。

型別

HttpEventType.ResponseHeader | HttpEventType.Response

回應的型別，窄化為完整回應或標頭。
