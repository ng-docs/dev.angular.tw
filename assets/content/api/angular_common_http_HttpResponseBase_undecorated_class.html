HttpResponseBase

Class
code

是 HttpResponse 和 HttpHeaderResponse 的基类。

abstract class HttpResponseBase {
}

headers

HttpHeaders

所有响应头。

status

number

响应的状态码。

statusText

string

响应状态码的文本描述，默认为 OK。

请不要在代码中依赖它。

url

string

检索到资源的 URL，如果不可用则为 null。

ok

boolean

状态码是否位于 2xx 范围内。

type

HttpEventType.ResponseHeader | HttpEventType.Response

响应对象的类型，窄化为完整的响应对象或只有响应头。
