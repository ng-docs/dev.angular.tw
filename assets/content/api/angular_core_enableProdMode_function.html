禁用 Angular 的开发模式，该模式将关闭框架中的断言和其他检查。

一个重要的断言，它禁用了对变更检测不会导致对任何绑定的（也称为单向数据流）额外更改的验证。

不鼓励使用此方法，因为 Angular CLI 将在使用 optimization 选项时设置生产模式。
