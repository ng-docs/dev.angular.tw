HttpResourceRef<TResult>

建立一個 Resource，它使用 HTTP GET 請求從給定的 URL 獲取資料。

如果為 URL 傳入了回應式函式，則當 URL 透過 Signal 更改時，資源將更新。

使用 HttpClient 發出請求，並支援攔截器、測試以及 HttpClient API 的其他屬性。預設情況下，資料會解析為 JSON - 使用 httpResource 的子函式（例如 httpResource.text()）以不同的方式解析回應。
