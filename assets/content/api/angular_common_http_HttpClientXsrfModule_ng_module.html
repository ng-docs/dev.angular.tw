配置 XSRF 保护，以支持外发请求。

对于支持基于 Cookie 的 XSRF 保护体系的服务器，直接使用以配置 XSRF 保护，并使用正确的 Cookie 和标头名称。

如果没有提供名字，则默认的 Cookie 名是 XSRF-TOKEN，默认的请求头名字是 X-XSRF-TOKEN。

disable

ModuleWithProviders<HttpClientXsrfModule>

禁用默认的 XSRF 保护。
