@angular/animations

AnimationQueryOptions

interface
程式碼

封裝了動畫查詢選項。 傳遞給 query() 函式。

API

  
    interface AnimationQueryOptions extends AnimationOptions {}

optional

boolean | undefined

如果此查詢是可選的，則為 True；如果此查詢是必要的，則為 false。預設為 false。 如果執行查詢時未檢索到任何元素，則必要的查詢會丟擲錯誤。可選查詢則不會。

limit

number | undefined

要從查詢回傳的最大結果總數。 如果為負數，則結果將從查詢清單的末尾向開頭方向限制。 預設情況下，結果不受限制。

延遲

string | number | undefined

設定啟動動畫操作的延遲時間。 一個數字和一個可選的時間單位，例如『1 秒』為『1s』，10 毫秒為『10ms』。預設單位為毫秒。 預設值為 0，表示沒有延遲。

引數

{ [name: string]: any; } | undefined

一組由開發人員定義的引數，用於在動畫操作開始時修改樣式和時序。 一個鍵值對陣列，其中提供的值用作預設值。
