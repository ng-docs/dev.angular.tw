ClassProvider

interface
code

配置 Injector 以返回某个令牌的 useClass 实例。

interface ClassProvider {
}

请注意，以下两个提供者并不相等：

多值示例

provide

any

一个注入令牌。（通常是 Type 或 InjectionToken 的实例，但可以是 any）。

multi

boolean

如果为 true，则注入器返回实例数组。这对于允许多个提供者散布在多个文件中以向公共令牌提供配置信息很有用。

useClass

Type<any>

用于将此 token 实例化的类。
