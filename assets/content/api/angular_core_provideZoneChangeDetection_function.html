为使用 bootstrapApplication 启动的应用提供基于 NgZone 的变更检测。

NgZone 默认在应用中已被提供。此提供者允许你配置 NgZone 中的选项（如 eventCoalescing）。此提供者不适用于 platformBrowser().bootstrapModule，后者使用 BootstrapOptions。

          
            bootstrapApplication(MyApp, {providers: [  provideZoneChangeDetection({eventCoalescing: true}),]});
