type CanMatchFn = (route: Route, segments: UrlSegment[]) => MaybeAsync<GuardResult>

用作 canMatch守卫在 Route上的函数签名。

如果所有守卫都返回 true，导航继续，并且 Router将在激活过程中使用该 Route。如果任何一个守卫返回 false，该 Route将被跳过匹配，改为处理其他 Route配置。
