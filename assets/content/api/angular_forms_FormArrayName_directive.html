用法說明

例如，對於以下 FormGroup：

          
form = new FormGroup({  address: new FormGroup({ street: new FormControl() })});

從根表單到『street』控制元件的路徑將是『address』->『street』。

可以透過以下兩種格式之一將其提供給此方法：

  1. 字串控制元件名稱的陣列，例如 ['address', 'street']
  2. 一個以句點分隔的控制名稱清單，在一個字串中，例如 'address.street'

如果未提供路徑，則此方法檢查當前控制元件上的錯誤。

描述

將巢狀的 FormArray 同步到 DOM 元素。

此指令旨在與父 FormGroupDirective（選擇器： [formGroup]）一起使用。

它接受你要連結的巢狀 FormArray 的字串名稱，並且將查詢在父 FormGroup 實例中以該名稱註冊的 FormArray，該實例是你傳入 FormGroupDirective 的。
