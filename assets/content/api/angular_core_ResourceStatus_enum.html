ResourceStatus

列舉
實驗性的
程式碼

Resource 的狀態。

API

  
    enum ResourceStatus {}

空閒

資源沒有有效的請求，將不會執行任何載入。

value() 將為 undefined。

錯誤

載入失敗並出現錯誤。

value() 將為 undefined。

載入中

由於其 request 發生更改，資源當前正在載入新值。

value() 將為 undefined。

重新載入中

資源當前正在為相同的請求重新載入一個全新的值。

value() 將在重新載入操作期間繼續回傳先前獲取的值。

已求解

載入已完成，並且資源具有從載入器回傳的值。

本地

資源的值是透過 .set() 或 .update() 在本地設定的。
