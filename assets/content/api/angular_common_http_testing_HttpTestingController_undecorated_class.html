verify

void

驗證沒有未對應的待處理請求。

如果有任何待處理的請求，則會失敗，並顯示一條錯誤訊息，指示哪些請求未處理。

如果未設定 ignoreCancelled（預設值），則如果取消的請求未顯式對應，verify() 也會失敗。

@paramopts{ ignoreCancelled?: boolean | undefined; } | undefined
@returnsvoid
