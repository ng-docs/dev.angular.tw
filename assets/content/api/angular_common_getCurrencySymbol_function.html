检索给定货币代码的货币符号。

比如，对于默认 en-US 语言环境，代码 USD 可以由窄符号 $ 或宽符号 US$ 表示。

getCurrencySymbol

string
@deprecated

检索给定货币代码的货币符号。

比如，对于默认 en-US 语言环境，代码 USD 可以由窄符号 $ 或宽符号 US$ 表示。

@deprecated

Angular 建议依赖 Intl API 进行国际化。你可以使用 Intl.NumberFormat().formatToParts() 提取货币符号。例如： Intl.NumberFormat('en', {style:'currency', currency: 'USD'}).formatToParts().find(part => part.type === 'currency').value 会返回 $ 对于 en 语言环境的 USD 货币代码。注意： US$ 是 en-ca 语言环境的货币符号，但不是 en-us 语言环境的货币符号。

@paramcodestring

货币代码。

@paramformat"wide" | "narrow"

格式，如 wide 或 narrow 。

@paramlocalestring

用于要使用的语言环境格式规则的语言环境代码。

@returnsstring
