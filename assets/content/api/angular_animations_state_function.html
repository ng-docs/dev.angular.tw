state

AnimationStateMetadata

在一個附著到元素上的觸發器中宣告一個動畫狀態。

@paramnamestring

以逗號分隔的字串形式表示的已定義狀態的一個或多個名稱。 以下保留的狀態名稱可用於為特定用例定義樣式：

  • void 你可以將樣式與此名稱關聯，以便在該元素從應用中分離時使用。例如，當 ngIf 求值 結果為 false 時，關聯元素的狀態為 void。
  • * （星號）表示預設狀態。你可以將樣式與此名稱關聯，以便在動畫狀態未在觸發器中宣告時用作回退。
@paramstylesAnimationStyleMetadata

一組與此狀態關聯的 CSS 樣式，使用 style() 函式建立。 一旦達到該狀態，這組樣式就會在元素上持續存在。

@paramoptions{ params: { [name: string]: any; }; } | undefined

當狀態被呼叫時，可以傳遞給狀態的引數。 0 個或多個鍵值對。

@returnsAnimationStateMetadata

用法說明

使用 trigger() 函式向動畫觸發器註冊狀態。 使用 transition() 函式在狀態之間進行動畫處理。 當某個狀態在元件中處於活動狀態時，即使動畫結束，其關聯的樣式也會在元素上持續存在。
