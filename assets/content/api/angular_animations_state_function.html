使用 trigger() 函数为动画触发器注册状态。使用 transition() 函数在状态之间进行动画。当一个状态在组件中处于活动状态时，其关联的样式会在元素上保持，即使动画结束。

state

AnimationStateMetadata

在附加到元素的触发器上，声明一个动画状态。

@paramnamestring

所定义的状态的一个或多个名字（用逗号分隔）。 下面这些保留的状态名可用于为一些特殊用例定义样式：

  • void 你可以将样式与该名称关联，以便在元素从应用中分离时使用。例如，当 ngIf 求值为 false 时，关联元素的状态为 void。
  • *（星号）表示默认状态。当触发器中未声明要设置的动画状态时，就会把该名称所关联的样式用作回退（fallback）值。
@paramstylesAnimationStyleMetadata

使用 style() 函数创建的与此状态相关的一组 CSS 样式。这组样式在状态达到后会保持在元素上。

@paramoptions{ params: { [name: string]: any; }; }

可以在状态调用时传递的参数。0 个或多个键值对。

@returnsAnimationStateMetadata
Usage notes

使用 trigger() 函数为动画触发器注册状态。使用 transition() 函数在状态之间进行动画。当一个状态在组件中处于活动状态时，其关联的样式会在元素上保持，即使动画结束。
