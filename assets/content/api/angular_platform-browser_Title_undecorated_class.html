可以用来获取和设置当前 HTML 文档标题的服务。

由于 Angular 应用程序不能在整个 HTML 文档(<html> 标签)上引导，因此无法绑定到 HTMLTitleElement 的 text 属性（即 <title> 标签）。可以改用此服务来设置和获取当前标题值。
