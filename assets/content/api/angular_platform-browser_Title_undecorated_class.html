描述

該服務可以用來獲取和設定當前 HTML 文件的標題。

由於 Angular 應用無法在整個 HTML 文件（<html> 標籤）上啟動，因此無法繫結到代表 <title> 標籤的 HTMLTitleElement 元素的 text 屬性（Property）。但是，可以使用此服務來設定和獲取當前的標題值。
