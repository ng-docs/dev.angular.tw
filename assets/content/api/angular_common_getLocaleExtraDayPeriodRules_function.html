getLocaleExtraDayPeriodRules

(Time | [Time, Time])[]

檢索用於確定在為語言環境定義了多個週期時，要使用哪個日週期的特定於語言環境的規則。

每個定義的日週期都有一條規則。第一條規則應用於第一個日週期，依此類別推。 當沒有規則可用時，回退到 AM/PM。

規則可以將週期指定為時間範圍，或指定為單個時間值。

僅當載入完整的語言環境資料時，此功能才可用。 請參閱『I18n 指南』。
