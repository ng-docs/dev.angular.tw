检索特定于语言环境的规则，该规则用于确定在为一个语言环境中定义了多个时段时要使用的一天时段。

每个定义的日期段都有一个规则。第一个规则应用于第一个日期段，依此类推。当没有规则可用时，回退到 AM/PM。

本规则可以将时间段指定为时间范围或单个时间值。

此功能仅在你加载了完整的语言环境数据时可用。参见 "I18n 指南"。
