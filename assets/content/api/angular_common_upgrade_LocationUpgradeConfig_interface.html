LocationUpgradeConfig

interface
code

LocationUpgrade 的配置选项。

interface LocationUpgradeConfig {
}

useHash

boolean

配置位置升级模块应使用 HashLocationStrategy 还是 PathLocationStrategy

hashPrefix

string

配置使用 HashLocationStrategy 时在 URL 中使用的哈希前缀

urlCodec

typeof UrlCodec

配置 URL 编解码器以对 URL 进行编码和解码。默认为 AngularJSCodec

serverBaseHref

string

配置在服务器端渲染的应用程序中使用时的 base href

appBaseHref

string

配置在客户端渲染的应用程序中使用时的 base href
