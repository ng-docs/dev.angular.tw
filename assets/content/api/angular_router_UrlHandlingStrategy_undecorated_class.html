shouldProcessUrl

boolean

告诉路由器是否应处理此 URL。

当返回 true 时，路由器将执行常规导航。当返回 false 时，路由器会将路由器状态设置为空状态。结果，所有活动组件都将被破坏。

@paramurlUrlTree
@returnsboolean
