注册一个回调函数，该回调将在每次应用完成渲染后被调用。

afterRender(
  callback: VoidFunction,
  options?: AfterRenderOptions
): AfterRenderRef;

注册一个回调函数，该回调将在每次应用完成渲染后被调用。

你应始终显式指定一个非默认的 阶段，否则会有显著的性能下降风险。

注意回调将按注册顺序运行

  • 按注册顺序
  • 每次渲染一次
  • 仅在浏览器平台上

在回调运行之前，不保证组件已 水合。直接读写 DOM 和布局时必须小心。

使用 afterRender 在每次渲染后读写 DOM。

范例

          
            @Component({  selector: 'my-cmp',  template: `<span #content>{{ ... }}</span>`,})export class MyComponent {
