DestroyRef

Class
code

DestroyRef 让你可以设置回调，以便在进行任何清理或销毁行为时运行。销毁的范围取决于 DestroyRef 被注入的位置。如果 DestroyRef 被注入到某个组件或指令中，则在该组件或指令销毁时运行回调。否则，在相应的注入器销毁时运行回调。

abstract class DestroyRef {
}

onDestroy

() => void

在给定的生命周期范围内注册销毁回调。返回一个清理函数，可以调用该函数来注销回调。

@paramcallback() => void
@returns() => void
