HTTP 指南

API

  
    type HttpHandlerFn = (req: HttpRequest<unknown>) => Observable<HttpEvent<unknown>>

描述

表示攔截器鏈中的下一個攔截器；如果沒有更多攔截器，則表示真正的後端。

大多數攔截器將委託給此函式，並在發出的請求或回應到達時對其進行修改。但是，在當前請求的範圍內，可以為任意數量的下游請求多次呼叫此函式。這些下游請求不必與當前請求具有相同的 URL，甚至不必與當前請求具有相同的來源。完全不呼叫下游處理程式，而在攔截器中完全處理當前請求也是有效的。

此函式應僅在當前正在被攔截的請求範圍內呼叫。一旦該請求完成，就不應再呼叫此下游處理程式函式。
