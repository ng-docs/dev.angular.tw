@angular/router

InitialNavigation

型別別名
程式碼

ExtraOptions 物件中允許的值，用於設定路由器何時執行初始導向操作。

forRoot

API

  
    type InitialNavigation = 'disabled' | 'enabledBlocking' | 'enabledNonBlocking'

描述

ExtraOptions 物件中允許的值，用於設定路由器何時執行初始導向操作。

  • 'enabledNonBlocking' - （預設值）初始導向在根元件建立之後啟動。啟動過程不會因初始導向完成而被阻塞。
  • 'enabledBlocking' - 初始導向在根元件建立之前啟動。啟動過程會被阻塞，直到初始導向完成。如果你使用伺服器端渲染，但沒有為你的應用啟用水合，則應設定此值。
  • 'disabled' - 不執行初始導向。位置監聽器在根元件建立之前建立。如果由於某些複雜的初始化邏輯，需要更好地控制路由器何時啟動其初始導向，請使用此選項。
