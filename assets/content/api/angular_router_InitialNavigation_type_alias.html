()
type InitialNavigation = 'disabled' | 'enabledBlocking' | 'enabledNonBlocking'

在 ExtraOptions 对象中允许的值，这些值配置路由器执行初始导航操作的时间。

  • 'enabledNonBlocking' -（默认值）在创建根组件之后开始初始导航。初始导航完成后，引导程序不会被阻止。
  • 'enabledBlocking' —— 初始导航在根组件创建之前启动。启动被阻塞，直到初始导航完成。此值是 服务器端渲染工作所必需的。
  • 'disabled' - 初始导航不执行。在根组件创建之前设置位置监听器。如果由于某些复杂的初始化逻辑需要对路由器的初始导航进行更多控制，请使用此选项。
