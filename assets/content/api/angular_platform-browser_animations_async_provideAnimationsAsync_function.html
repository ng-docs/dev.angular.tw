provideAnimationsAsync

EnvironmentProviders

回傳一組依賴注入提供者，以在應用中啟用動畫。請參閱動畫指南，以瞭解有關 Angular 動畫的更多訊息。

當你使用此函式而不是急性的 provideAnimations() 時，動畫將不會被渲染，直到渲染器載入完畢。

@paramtype"animations" | "noop"

傳入 'noop' 作為引數以停用動畫。

@returnsEnvironmentProviders

描述

回傳一組依賴注入提供者，以在應用中啟用動畫。請參閱動畫指南，以瞭解有關 Angular 動畫的更多訊息。

當你使用此函式而不是急性的 provideAnimations() 時，動畫將不會被渲染，直到渲染器載入完畢。

用法說明

當你想在使用 bootstrapApplication 函式引導的應用中啟用動畫時，此函式非常有用。在這種情況下，完全不需要匯入 BrowserAnimationsModule NgModule，只需將此函式回傳的提供者新增到 providers 清單中，如下所示。

          
bootstrapApplication(RootComponent, {  providers: [    provideAnimationsAsync()  ]});
