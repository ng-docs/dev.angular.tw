FormControlStatus

型別別名
程式碼

表單可以有幾種不同的狀態。每種可能的狀態都以字串字面量的形式回傳。

API

  
    type FormControlStatus = 'VALID' | 'INVALID' | 'PENDING' | 'DISABLED'

描述

表單可以有幾種不同的狀態。每種可能的狀態都以字串字面量的形式回傳。

  • VALID：報告控制元件是有效的，意味著輸入值中不存在錯誤。
  • INVALID：報告控制元件是無效的，意味著輸入值中存在錯誤。
  • PENDING：報告控制元件處於待定狀態，意味著正在進行非同步驗證，並且輸入值的錯誤尚不可用。
  • DISABLED：報告控制元件已停用，意味著該控制元件免於祖先對有效性或值的計算。
