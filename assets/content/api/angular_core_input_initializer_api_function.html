@angular/core

input

初始化器 API
程式碼

input 函式允許在指令和元件中宣告 Angular 的輸入屬性（Property）。

可以宣告以下兩種變體的輸入屬性（Property）：

  1. 帶有初始值的可選輸入屬性（Property）。
  2. 使用者需要設定的必要輸入屬性（Property）。

預設情況下，input 函式將宣告始終具有初始值的可選輸入屬性（Property）。可以使用 input.required() 函式宣告必要輸入屬性（Property）。

輸入屬性（Property）是訊號。輸入屬性（Property）的值會作為 Signal 公開。此訊號始終持有從父元件繫結的輸入屬性（Property）的最新值。

API

  
    
  
  
function input<T>(): InputSignal<T | undefined>;

初始化型別為 T 的輸入屬性（Property），其初始值為 undefined。 Angular 將隱式地使用 undefined 作為初始值。

@returnsInputSignal<T | undefined>
function input<T>(initialValue: T, opts?: InputOptionsWithoutTransform<T> | undefined): InputSignal<T>;

宣告型別為 T 的輸入屬性（Property），並帶有顯式的初始值。

@paraminitialValueT
@paramoptsInputOptionsWithoutTransform<T> | undefined
@returnsInputSignal<T>
function input<T>(initialValue: undefined, opts: InputOptionsWithoutTransform<T>): InputSignal<T | undefined>;

宣告型別為 T|undefined 的輸入屬性（Property），不帶初始值，但帶有輸入選項

@paraminitialValueundefined
@paramoptsInputOptionsWithoutTransform<T>
@returnsInputSignal<T | undefined>
function input<T, TransformT>(initialValue: T, opts: InputOptionsWithTransform<T, TransformT>): InputSignalWithTransform<T, TransformT>;

宣告型別為 T 的輸入屬性（Property），帶有初始值和一個轉換函式。

此輸入屬性（Property）接受 TransformT 型別的值，並且給定的轉換函式會將該值轉換為 T 型別。

@paraminitialValueT
@paramoptsInputOptionsWithTransform<T, TransformT>
@returnsInputSignalWithTransform<T, TransformT>
function input<T, TransformT>(initialValue: undefined, opts: InputOptionsWithTransform<T | undefined, TransformT>): InputSignalWithTransform<T | undefined, TransformT>;

宣告型別為 T|undefined 的輸入屬性（Property），不帶初始值，但帶有一個轉換函式。

此輸入屬性（Property）接受 TransformT 型別的值，並且給定的轉換函式會將該值轉換為 T|undefined 型別。

@paraminitialValueundefined
@paramoptsInputOptionsWithTransform<T | undefined, TransformT>
@returnsInputSignalWithTransform<T | undefined, TransformT>
function input.required<T>(opts?: InputOptionsWithoutTransform<T> | undefined): InputSignal<T>;

宣告型別為 T 的必要輸入屬性（Property）。

@paramoptsInputOptionsWithoutTransform<T> | undefined
@returnsInputSignal<T>
function input.required<T, TransformT>(opts: InputOptionsWithTransform<T, TransformT>): InputSignalWithTransform<T, TransformT>;

宣告型別為 T 的必要輸入屬性（Property），並帶有轉換函式。

此輸入屬性（Property）接受 TransformT 型別的值，並且給定的轉換函式會將該值轉換為 T 型別。

@paramoptsInputOptionsWithTransform<T, TransformT>
@returnsInputSignalWithTransform<T, TransformT>

使用須知

要使用基於訊號的輸入屬性（Property），請從 @angular/core 匯入 input。

          
import {input} from '@angular/core`;

在你的元件內部，引入一個新的類別成員，並透過呼叫 input 或 input.required 來初始化它。

          
@Component({  ...})export class UserProfileComponent {  firstName = input<string>();             // Signal<string|undefined>  lastName  = input.required<string>();    // Signal<string>  age       = input(0)                     // Signal<number>}

在你的元件樣板內部，你可以透過呼叫訊號來顯示輸入屬性（Property）的值。

          
<span>{{firstName()}}</span>
