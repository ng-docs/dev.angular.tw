input 函数允许在指令和组件中声明 Angular 输入。

可以声明两种变体的输入：

  1. 可选输入，具有初始值。
  2. 必要输入，需要使用者进行设置。

默认情况下， input 函数会声明始终具有初始值的可选输入。可以使用 input.required() 函数声明必要输入。

输入是信号。输入的值作为 Signal 暴露。该信号始终持有从父组件绑定的输入的最新值。

要使用基于信号的输入，从 @angular/core 导入 input。

          
            import {input} from '@angular/core`;

在你的组件中，引入一个新的类成员，并使用调用 input 或 input.required 进行初始化。

          
            @Component({  ...})export class UserProfileComponent {  firstName = input<string>();             // Signal<string|undefined>  lastName  = input.required<string>();    // Signal<string>  age       = input(0)                     // Signal<number>}

在组件模板中，你可以通过调用信号来显示输入的值。

          
            <span>{{firstName()}}</span>
input<T>(): InputSignal<T>;

初始化类型为 T 的输入，并初始值为 undefined。Angular 将隐式地使用 undefined 作为初始值。

@returnsInputSignal<T>
input<T>(initialValue: T, opts?: InputOptionsWithoutTransform<T>): InputSignal<T>;

声明一个具有显式初始值的类型为 T 的输入。

@paraminitialValueT
@paramoptsInputOptionsWithoutTransform<T>
@returnsInputSignal<T>
input<T, TransformT>(initialValue: T, opts: InputOptionsWithTransform<T, TransformT>): InputSignalWithTransform<T, TransformT>;

声明一个具有初始值和转换函数的类型为 T 的输入。

该输入接受类型为 TransformT 的值，并且给定的转换函数会将值转换为类型 T。

@paraminitialValueT
@paramoptsInputOptionsWithTransform<T, TransformT>
@returnsInputSignalWithTransform<T, TransformT>
input.required<T>(opts?: InputOptionsWithoutTransform<T>): InputSignal<T>;
