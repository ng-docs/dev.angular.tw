定义一个动画步骤列表，逐个依次运行它们。

sequence(
  steps: AnimationMetadata[],
  options?: AnimationOptions
): AnimationSequenceMetadata;

当你将一个步骤数组传递给 transition() 调用时，步骤默认按顺序运行。与 {@link animations/group group()} 调用相比，它会并行运行动画步骤。

当在 {@link animations/group group()} 或 transition() 调用中使用序列时，只有在每个内部动画步骤完成后，执行才会继续到下一个指令。

sequence

AnimationSequenceMetadata

定义一个动画步骤列表，逐个依次运行它们。

@paramstepsAnimationMetadata[]

一个由动画步骤对象构成的数组。

  • 由 style() 调用定义的步骤会立即应用样式数据。
  • 由 animate() 调用定义的步骤会在指定的时间内应用样式数据。
          
            sequence([style({ opacity: 0 }),animate("1s", style({ opacity: 1 }))])
@paramoptionsAnimationOptions

一个选项对象，包含延迟和开发人员定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。

@returnsAnimationSequenceMetadata
Usage notes

当你将一个步骤数组传递给 transition() 调用时，步骤默认按顺序运行。与 {@link animations/group group()} 调用相比，它会并行运行动画步骤。

当在 {@link animations/group group()} 或 transition() 调用中使用序列时，只有在每个内部动画步骤完成后，执行才会继续到下一个指令。
