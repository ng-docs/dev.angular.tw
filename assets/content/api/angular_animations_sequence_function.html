sequence

AnimationSequenceMetadata

定義要按順序逐個執行的動畫步驟清單。

@paramstepsAnimationMetadata[]

動畫步驟物件陣列。

  • 由 style() 呼叫定義的步驟會立即應用樣式資料。
  • 由 animate() 呼叫定義的步驟會在一段時間內應用樣式資料，具體時間由時序資料指定。
          
sequence([style({ opacity: 0 }),animate("1s", style({ opacity: 1 }))])
@paramoptionsAnimationOptions | null

一個 options 物件，其中包含延遲和開發者定義的引數，這些引數提供樣式預設值，並且可以在呼叫時被重寫。

@returnsAnimationSequenceMetadata

用法說明

當您將步驟陣列傳遞給 transition() 呼叫時，預設情況下這些步驟會按順序執行。將其與 group() 呼叫進行比較，後者並行執行動畫步驟。

當序列在 group() 或 transition() 呼叫中使用時，僅當每個內部動畫步驟都完成後，執行才會繼續執行下一個指令。
