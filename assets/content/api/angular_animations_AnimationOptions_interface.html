@angular/animations

AnimationOptions

interface
程式碼

用於控制動畫樣式和時序的選項。

API

  
    interface AnimationOptions {}

延遲

string | number | undefined

為啟動動畫動作設定時間延遲。 一個數字以及可選的時間單位，例如 "1s" 或 "10ms" 分別表示一秒和 10 毫秒。預設單位為毫秒。 預設值為 0，表示沒有延遲。

引數

{ [name: string]: any; } | undefined

一組由開發人員定義的引數，用於在動畫動作開始時修改樣式和時序。一個鍵值對陣列，其中提供的值用作預設值。

描述

用於控制動畫樣式和時序的選項。

以下動畫函式接受 AnimationOptions 資料：

  • transition()
  • sequence()
  • <a href="/api/animations/group"><code>group()</code></a>
  • query()
  • animation()
  • useAnimation()
  • animateChild()

使用 AnimationBuilder 服務建立的程式設計式動畫也使用 AnimationOptions。
