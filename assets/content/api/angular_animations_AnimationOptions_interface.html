interface AnimationOptions {
}

用来控制动画样式和时序的选项。

以下动画函数接受 AnimationOptions 数据：

  • transition()
  • sequence()
  • <a href="api/animations/animations/group group()"><code>animations/group group()</code></a>
  • query()
  • animation()
  • useAnimation()
  • animateChild()

使用 AnimationBuilder 服务构建的编程动画也使用 AnimationOptions。

delay

string | number

设置第一个动画动作开始前的延迟时间。 包括一个数字和一个可选的时间单位，比如 "1s" 表示一秒，"10ms" 表示十毫秒。 默认单位是毫秒。 默认值为 0，表示不延迟。

params

{ [name: string]: any; }

一组可由开发人员来定义的参数，用于在动画开始时修改样式和时序。 这是一组键值对，它所提供的值会用做默认值。
