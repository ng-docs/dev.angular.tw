query

AnimationQueryMetadata

在當前元素（正在序列中執行動畫）內查詢一個或多個內部元素。與 animate() 一起使用。

@paramselectorstring

要查詢的元素，或包含 Angular 特有特徵的一組元素，可以使用以下一個或多個標記來指定。

  • query(":enter") 或 query(":leave")：查詢新插入/移除的元素（並非所有元素都可以透過這些標記查詢，請參閱進入和離開元素）
  • query(":animating")：查詢當前正在執行動畫的所有元素。
  • query("@triggerName")：查詢包含動畫觸發器的元素。
  • query("@*")：查詢包含動畫觸發器的所有元素。
  • query(":self")：將當前元素包含到動畫序列中。
@paramanimationAnimationMetadata | AnimationMetadata[]

要應用於查詢到的一個或多個元素的動畫步驟。陣列被視為動畫序列。

@paramoptionsAnimationQueryOptions | null

一個選項物件。使用 'limit' 欄位來限制要收集的條目總數。

@returnsAnimationQueryMetadata

用法說明

多個標記

標記可以合併到組合的查詢選擇器字串中。例如：

          
 query(':self, .record:enter, .record:leave,
