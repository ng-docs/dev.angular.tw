声明了一个 Angular 输出，该输出使用 RxJS 可观察者作为事件源，传递给父订阅者。

可观察者作为源的行为定义如下：

  1. 新值会被转发到 Angular 输出（下一次通知）。
  2. 错误通知不会被 Angular 处理。你需要手动处理这些错误。例如使用 catchError。
  3. 完成通知会停止输出新值。

通过声明一个类字段并使用 outputFromObservable() 函数初始化它，在指令中初始化一个输出。

          
            @Directive({..})export class MyDir {  nameChange$ = <some-observable>;  nameChange = outputFromObservable(this.nameChange$);}
