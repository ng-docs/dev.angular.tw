outputFromObservable

OutputRef<T>

宣告一個 Angular 輸出，它使用 RxJS 可觀察者作為事件源，這些事件將被分發給父訂閱者。

作為事件源的可觀察者的行為定義如下：

  1. 新值會被轉發到 Angular 輸出（下一個通知）。
  2. 錯誤通知不會被 Angular 處理。你需要手動處理這些錯誤。例如，透過使用 catchError。
  3. 完成通知會阻止輸出發出新值。

描述

宣告一個 Angular 輸出，它使用 RxJS 可觀察者作為事件源，這些事件將被分發給父訂閱者。

作為事件源的可觀察者的行為定義如下：

  1. 新值會被轉發到 Angular 輸出（下一個通知）。
  2. 錯誤通知不會被 Angular 處理。你需要手動處理這些錯誤。例如，透過使用 catchError。
  3. 完成通知會阻止輸出發出新值。

用法說明

在你的指令中初始化一個輸出，方法是宣告一個類別欄位，並使用 outputFromObservable() 函式來初始化它。

          
@Directive({..})export class MyDir {  nameChange$ = <some-observable>;  nameChange = outputFromObservable(this.nameChange$);}
