描述

一個用於管理 HTML <meta> 標籤的服務。

MetaDefinition 物件的屬性與 HTML <meta> 標籤的屬性（Attribute）相對應。這些標籤定義了文件元資料，對於諸如設定 Content Security Policy、定義瀏覽器相容性和安全設定、設定 HTTP 標頭、定義用於社交分享的富內容以及搜尋引擎最佳化 (SEO) 之類別的事情非常重要。

要在文件中識別特定的 <meta> 標籤，請使用格式為 "tag_attribute='value string'" 的屬性選擇字串。例如，attrSelector 值為 "name='description'" 的選擇器會對應 name 屬性（Attribute）的值為 "description" 的標籤。選擇器與 querySelector() Document 方法一起使用，格式為 meta[{attrSelector}]。
