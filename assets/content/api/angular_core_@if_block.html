@angular/core

@if

控制區塊
code

當 @if 控制區塊的條件表示式為真值時，會有條件地顯示其內容。

語法 

          
@if (a > b) {  {{a}} is greater than {{b}}} @else if (b > a) {  {{a}} is less than {{b}}} @else {  {{a}} is equal to {{b}}}

描述

根據 @if 和 @else 控制塊中條件表示式的求值結果，在 DOM 中新增和移除內容。

內建的 @if 支援引用表示式的結果，以便為常見的編碼模式保留解決方案：

          
@if (users$ | async; as users) {  {{ users.length }}}
