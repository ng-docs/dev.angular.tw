說明

根據語言環境規則格式化日期值。

DatePipe 僅當檢測到輸入值的純變更時才執行。 純變更要麼是原始型別輸入值（例如 String、Number、Boolean 或 Symbol）的變更，要麼是已更改的物件引用（例如 Date、Array、Function 或 Object）。

請注意，修改 Date 物件不會導致通道再次渲染。 要確保通道被執行，你必須建立一個新的 Date 物件。

只有 en-US 語言環境資料隨 Angular 一起提供。要在另一種語言中本地化日期，你必須匯入相應的語言環境資料。 有關更多訊息，請參閱 I18n 指南。

格式化值的時區可以透過將其作為通道的第二個引數傳入來指定，也可以透過 DATE_PIPE_DEFAULT_OPTIONS 注入令牌設定預設值。作為第二個引數傳入的值優先於使用注入令牌定義的值。

用法說明

當輸入發生變化時，此通道的結果不會重新求值。為了避免在每次變更檢測週期都重新格式化日期，請將日期視為不可變物件，並在通道需要再次執行時更改引用。

格式範例

這些範例將日期轉換為各種格式，假設 dateObj 是一個 JavaScript Date 物件，其值為：年份：2015，月份：6，日期：15，小時：21，分鐘：43，秒：11，並且給定的時間為 en-US 語言環境的本地時間。

          
{{ dateObj | date }}               // output is 'Jun 15, 2015'{{ dateObj | date:'medium' }}      // output is 'Jun 15, 2015, 9:43:11 PM'{{ dateObj | date:'shortTime' }}   // output is '9:43 PM'{{ dateObj | date:'mm:ss' }}       // output is '43:11'{{ dateObj | date:"MMM dd, yyyy 'at' hh:mm a" }}  // output is 'Jun 15, 2015 at 09:43 PM'

用法範例

以下元件使用日期通道以不同的格式顯示當前日期。

          
@Component({ selector: 'date-pipe', template: `<div>   <p>Today is {{today | date}}</p>   <p>Or if you prefer, {{today | date:'fullDate'}}</p>   <p>The time is {{today | date:'h:mm a z'}}</p> </div>`})// Get the current date and time as a date-time value.export class DatePipeComponent {  today: number = Date.now();}
