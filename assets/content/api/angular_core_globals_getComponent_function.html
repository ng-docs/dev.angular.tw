getComponent

T | null

檢索與給定的 DOM 元素關聯的元件實例。

@paramelementElement

要從中檢索元件的 DOM 元素。

@returnsT | null

使用須知

給定以下 DOM 結構：

          
<app-root>  <div>    <child-comp></child-comp>  </div></app-root>

在 <child-comp> 上呼叫 getComponent 將回傳與此 DOM 元素關聯的 ChildComponent 實例。

在 <app-root> 上呼叫此函式將回傳 MyApp 實例。
