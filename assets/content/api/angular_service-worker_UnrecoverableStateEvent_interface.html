說明

當 Service Worker 用來為該用戶端提供服務的應用版本處於無法恢復的損壞狀態，並且需要完全重新載入頁面時，會發出此事件。

例如，Service Worker 可能無法從快取或伺服器檢索到必需的資源。如果新版本已部署到伺服器，並且瀏覽器已部分清理了 Service Worker 快取，從而刪除了先前應用版本的一些檔案但並非全部檔案，則可能會發生這種情況。
