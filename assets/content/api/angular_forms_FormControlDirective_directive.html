@returnsboolean
用法說明

例如，對於以下 FormGroup：

          
form = new FormGroup({  address: new FormGroup({ street: new FormControl() })});

從根表單到『street』控制元件的路徑將是『address』->『street』。

可以透過以下兩種格式之一將其提供給此方法：

  1. 字串控制元件名稱的陣列，例如 ['address', 'street']
  2. 一個字串中以句點分隔的控制元件名稱清單，例如 'address.street'

如果未提供路徑，則此方法檢查當前控制元件上是否存在錯誤。

描述

將獨立的 FormControl 實例同步到表單控制元件元素。

請注意，在 Angular v6 中，已棄用對將 ngModel 輸入屬性和 ngModelChange 事件與回應式表單指令一起使用的支援，並計劃在 Angular 的未來版本中將其移除。
