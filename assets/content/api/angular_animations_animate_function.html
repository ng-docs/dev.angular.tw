定義動畫步驟，將樣式訊息與時序訊息相結合。

API

  
    function animate(  timings: string | number,  styles?: AnimationStyleMetadata | AnimationKeyframesSequenceMetadata | null): AnimationAnimateMetadata;

animate

AnimationAnimateMetadata

定義動畫步驟，將樣式訊息與時序訊息相結合。

@paramtimingsstring | number

為父動畫設定 AnimateTimings。 格式為 "duration [delay] [easing]" 的字串。

  • 持續時間和延遲表示為一個數字和可選的時間單位，例如 "1s" 或 "10ms" 分別表示一秒和 10 毫秒。 預設單位是毫秒。
  • 緩動值控制動畫在其執行時如何加速和減速。值是 ease、ease-in、ease-out、ease-in-out 或 cubic-bezier() 函式呼叫之一。 如果未提供，則不應用緩動。

例如，字串 "1s 100ms ease-out" 指定持續時間為 1000 毫秒，延遲為 100 毫秒，以及 "ease-out" 緩動樣式， 該樣式在持續時間接近結束時減速。

@paramstylesAnimationStyleMetadata | AnimationKeyframesSequenceMetadata | null

為父動畫設定 AnimationStyles。 呼叫 style() 或 keyframes() 的函式呼叫， 它回傳要應用於父動畫的 CSS 樣式條目的集合。 當為 null 時，使用來自目標狀態的樣式。 當描述將完成動畫的動畫步驟時，這很有用； 請參閱 transitions() 中的『為最終狀態製作動畫』。

@returnsAnimationAnimateMetadata

用法說明

在動畫 sequence()、group() 或 transition() 呼叫中呼叫，以指定一個動畫步驟，該步驟在給定的時間內將給定的樣式資料應用於父動畫。

語法範例

時序範例

以下範例展示了各種 timings 規範。

  • animate(500)：持續時間為 500 毫秒。
  • animate("1s")：持續時間為 1000 毫秒。
  • animate("100ms 0.5s")：持續時間為 100 毫秒，延遲為 500 毫秒。
  • animate("5s ease-in")：持續時間為 5000 毫秒，緩入。
  • animate("5s 10ms cubic-bezier(.17,.67,.88,.1)")：持續時間為 5000 毫秒，延遲為 10 毫秒，根據貝塞爾曲線緩動。

樣式範例

以下範例呼叫 style() 來設定單個 CSS 樣式。

          
animate(500, style({ background: "red" }))

以下範例呼叫 keyframes() 來為連續的關鍵幀將 CSS 樣式設定為不同的值。

          
animate(500, keyframes( [  style({ background: "blue" }),  style({ background: "red" }) ])
