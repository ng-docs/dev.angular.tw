@angular/animations

animate

function
code

定义一个动画步骤，它把一些样式信息和时序信息组合在一起。

animate(
  timings: string | number,
  styles?: AnimationStyleMetadata | AnimationKeyframesSequenceMetadata
): AnimationAnimateMetadata;

在动画 sequence()、 {@link animations/group group()} 或 transition() 调用中调用，以指定一个动画步骤（将给定的样式数据应用于父动画的指定时间内）。

语法范例

时序范例

下面的例子展示了各种 timings（时序）规范。

  • animate(500) ：时长为 500 毫秒。
  • animate("1s") ：时长为 1000 毫秒。
  • animate("100ms 0.5s") ：时长为 100 毫秒，延迟为 500 毫秒。
  • animate("5s ease-in") ：时长为 5000 毫秒，缓入。
  • animate("5s 10ms cubic-bezier(.17,.67,.88,.1)") ：时长为 5000 毫秒，延迟为 10 毫秒，按照贝塞尔曲线缓动。

样式范例

以下示例调用 style() 设置单个 CSS 样式。

          
            animate(500, style({ background: "red" }))

以下示例调用 keyframes() 设置 CSS 样式（为连续的关键帧设置不同的值）。

          
            animate(500, keyframes( [  style({ background: "blue" }),  style({ background: "red" }) ])

animate

AnimationAnimateMetadata

定义一个动画步骤，它把一些样式信息和时序信息组合在一起。

@paramtimingsstring | number

为父动画设置 AnimateTimings。格式为 "时长 [延迟] [缓动]" 的字符串。

  • 持续时间和延迟都用一个动画和一个可选的时间单位来表示，比如 "1s" 代表一秒，"10ms" 代表十毫秒。默认单位是毫秒。
  • 缓动效果的值控制该动画在运行期间如何加速和减速。它的取值是 ease、ease-in、ease-out、 ease-in-out 之一或一个 cubic-bezier() 函数调用。 如果未提供，则没有缓动效果。

比如，字符串 "1s 100ms ease-out" 指定了一个 1000 毫秒的持续时间，一个 100 毫秒的延迟和一个 "ease-out" 缓动效果，它会快结束时减速。

@paramstylesAnimationStyleMetadata | AnimationKeyframesSequenceMetadata

为父动画设置 AnimationStyles。调用 style() 或 keyframes() 函数，返回要应用于父动画的 CSS 样式集合。当为 null 时，使用目标状态的样式。这在描述将完成动画的步骤时很有用；参见 transitions() 中的“动画到最终状态”。

@returnsAnimationAnimateMetadata
Usage notes

在动画 sequence()、 {@link animations/group group()} 或 transition() 调用中调用，以指定一个动画步骤（将给定的样式数据应用于父动画的指定时间内）。
