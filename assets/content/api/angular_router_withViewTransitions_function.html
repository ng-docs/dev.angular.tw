withViewTransitions(
  options?: ViewTransitionsFeatureOptions
): ViewTransitionsFeature;

通过在 document.startViewTransition 内运行路由激活和停用来启用路由器中的视图过渡。

注意：视图过渡 API 并非在所有浏览器中都可用。如果浏览器不支持视图过渡，路由器将不会尝试启动视图过渡，并继续正常处理导航。

如何启用该功能的基本示例：

          
            const appRoutes: Routes = [];bootstrapApplication(AppComponent,  {    providers: [      provideRouter(appRoutes, withViewTransitions())    ]  });
