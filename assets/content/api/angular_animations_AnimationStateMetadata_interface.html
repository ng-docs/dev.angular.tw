@angular/animations

AnimationStateMetadata

interface
程式碼

透過將狀態名稱與一組 CSS 樣式相關聯來封裝動畫狀態。 由 state() 函式實例化並回傳。

API

  
    interface AnimationStateMetadata extends AnimationMetadata {}

name

string

狀態名稱，在元件內是唯一的。

styles

AnimationStyleMetadata

與此狀態關聯的 CSS 樣式。

options

{ params: { [name: string]: any; }; } | undefined

一個選項物件，其中包含開發者定義的引數，這些引數提供樣式預設值，並且可以在呼叫時被重寫。

型別

AnimationMetadataType
