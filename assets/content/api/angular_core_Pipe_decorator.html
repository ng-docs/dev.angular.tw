class Pipe {

本装饰器用于将类标记为管道并提供配置元数据。

一个管道类必须实现 PipeTransform 接口。例如，如果名称为 "myPipe"，请使用如下模板绑定表达式：

          
            {{ exp | myPipe }}

表达式的结果被传递给管道的 transform() 方法。

一个管道必须属于某个 NgModule，才能在模板中使用。要使其成为 NgModule 的成员，请将其列在 NgModule 元数据的 declarations 字段中。

name

string

在模板绑定中使用的管道名称。通常使用 lowerCamelCase，因为名称不能包含连字符。

pure

boolean

当为 true 时，管道是纯的，这意味着只有在其输入参数更改时才会调用 transform() 方法。管道默认是纯的。

如果管道具有内部状态（即结果依赖于其参数之外的状态），将 pure 设置为 false。在这种情况下，即使参数没有更改，管道也会在每个变更检测周期中调用。

standalone

boolean

标记为 standalone 的 Angular 管道不需要在 NgModule 中声明。此类管道不依赖于 NgModule 的任何“中间上下文”（例如配置的提供程序）。
