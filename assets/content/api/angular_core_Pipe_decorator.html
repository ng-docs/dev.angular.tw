風格指南：通道名稱

API

  
    @Pipe ({})

name

string

在範本繫結中使用的通道名稱。 通常使用 lowerCamelCase 格式， 因為名稱不能包含連字元。

pure

boolean | undefined

當為 true 時，通道是純通道，這意味著僅當其輸入引數發生變化時，才會呼叫 transform() 方法。通道預設是純通道。

如果通道具有內部狀態（即，結果取決於除其引數之外的狀態），請將 pure 設定為 false。 在這種情況下，即使引數沒有更改，也會在每個變更檢測週期呼叫該通道。

standalone

boolean | undefined

標記為 standalone 的 Angular 通道不需要在 NgModule 中宣告。此類別通道不依賴 NgModule 的任何『中間上下文』（例如，設定的提供者）。

有關獨立元件、指令和通道的更多訊息，請參閱本指南。

描述

將類別標記為通道並提供設定元資料的裝飾器。

通道類別必須實現 PipeTransform 介面。 例如，如果名稱為『myPipe』，請使用如下範本繫結表示式：

          
{{ exp | myPipe }}

表示式的結果會傳遞給通道的 transform() 方法。

通道必須屬於 NgModule 才能在範本中使用。要使其成為 NgModule 的成員，請在 NgModule 元資料的 declarations 欄位中列出它。
