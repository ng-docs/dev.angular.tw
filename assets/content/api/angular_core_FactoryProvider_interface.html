FactoryProvider

interface
code

配置 Injector 通过调用 useFactory 函数返回一个值。

interface FactoryProvider {
}

依赖项也可以标记为可选：

多值示例

provide

any

一个注入令牌。（通常是 Type 或 InjectionToken 的实例，但可以是 any）。

multi

boolean

如果为 true，则注入器返回实例数组。这对于允许多个提供者散布在多个文件中以向公共令牌提供配置信息很有用。

useFactory

Function

供调用，来为此 token 创建值的函数。调用该函数时，会在 deps 字段中的传入 token 的解析结果。。

deps

any[]

供注入器解析的 token 列表。然后，将值列表将用作 useFactory 函数的参数。
