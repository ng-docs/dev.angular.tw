一個執行變更檢測的回呼方法，在預設變更檢測器執行後呼叫。 有關為集合實現自定義變更檢查，請參閱 KeyValueDiffers 和 IterableDiffers。

@returnsvoid

描述

一個生命週期鉤子，除了預設變更檢測器執行的檢查之外，它還為指令呼叫自定義變更檢測函式。

預設的變更檢測演算法透過跨變更檢測執行按引用比較繫結屬性值來查詢差異。你可以使用此鉤子透過其他方式檢查和回應更改。

當預設變更檢測器檢測到更改時，如果提供了 ngOnChanges()，它將呼叫它，無論你是否執行額外的變更檢測。 通常，你不應該同時使用 DoCheck 和 OnChanges 來回應同一輸入上的更改。

使用須知

以下程式碼片段展示了一個元件如何實現此介面以呼叫其自身的變更檢測週期。

          
@Component({        selector: 'my-cmp',        template: `...`,        standalone: false,      })      class MyComponent implements DoCheck {        ngDoCheck() {          // ...        }      }
