group

AnimationGroupMetadata

定义一个可以并行运行的动画步骤列表。

@paramstepsAnimationMetadata[]

一个由动画步骤对象构成的数组。

  • 当由 style() 或 animate() 函数调用定义步骤时，组内的每个调用都会立即执行。
  • 要指定稍后应用的偏移样式，请使用 keyframes() 定义步骤，或使用带有延迟值的 animate() 调用。例如：
          
            group([animate("1s", style({ background: "black" })),animate("2s", style({ color: "white" }))])
@paramoptionsAnimationOptions

一个选项对象，包含延迟和开发人员定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。

@returnsAnimationGroupMetadata
Usage notes

当一系列样式分别需要在不同的起始时间开始动画并在不同的结束时间停止时，分组动画非常有用。

当在 sequence() 或 transition() 调用中调用时，不会继续到下一条指令，直到所有内部动画步骤完成。
