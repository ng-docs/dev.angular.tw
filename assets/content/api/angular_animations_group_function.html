定義要並行執行的動畫步驟清單。

API

  
    function group(  steps: AnimationMetadata[],  options?: AnimationOptions | null): AnimationGroupMetadata;

組

AnimationGroupMetadata

定義要並行執行的動畫步驟清單。

@paramstepsAnimationMetadata[]

動畫步驟物件陣列。

  • 當步驟由 style() 或 animate() 函式呼叫定義時，組內的每個呼叫都會立即執行。
  • 要指定稍後應用的偏移樣式，請使用 keyframes() 定義步驟，或使用帶有延遲值的 animate() 呼叫。 例如：
          
group([animate("1s", style({ background: "black" })),animate("2s", style({ color: "white" }))])
@paramoptionsAnimationOptions | null

一個 options 物件，其中包含延遲和開發者定義的引數，這些引數提供樣式預設值，並且可以在呼叫時被重寫。

@returnsAnimationGroupMetadata

用法說明

當一系列樣式必須在不同的開始時間啟動動畫並在不同的結束時間結束時，分組動畫非常有用。

當在 sequence() 或 transition() 呼叫中呼叫時，在所有內部動畫步驟完成之前，不會繼續執行下一個指令。
