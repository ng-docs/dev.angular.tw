enableDebugTools(
  ref: ComponentRef<T>
): ComponentRef<T>;

可通过浏览器的开发者控制台访问的已启用 Angular 调试工具。

用法：

  1. 打开开发人员控制台（比如，Chrome 中是 Ctrl + Shift + j）
  2. 输入 ng.（通常，控制台会显示自动完成建议）
  3. 试用变更检测剖析器 ng.profiler.timeChangeDetection() 然后按 Enter。。
