初始化一个视图子查询。

查询结果表示为包含所有匹配元素的只读集合信号。

在你的组件中创建一个子查询，通过声明一个类字段并用 viewChildren() 函数初始化它。

          
            @Component({...})export class TestComponent {  divEls = viewChildren<ElementRef>('el');   // Signal<ReadonlyArray<ElementRef>>}
