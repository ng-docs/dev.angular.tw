provideAppInitializer

函式
程式碼

所提供的函式會在應用啟動時被注入，並在應用初始化期間執行。如果該函式回傳一個 Promise 或一個 Observable，則初始化不會完成，直到 Promise 被求解或 Observable 完成。

APP_INITIALIZER

API

  
    function provideAppInitializer(  initializerFn: () => void | Observable<unknown> | Promise<unknown>): EnvironmentProviders;

provideAppInitializer

EnvironmentProviders

所提供的函式會在應用啟動時被注入，並在應用初始化期間執行。如果該函式回傳一個 Promise 或一個 Observable，則初始化不會完成，直到 Promise 被求解或 Observable 完成。

例如，你可以建立一個載入語言資料或外部設定的函式，並使用 provideAppInitializer() 來提供該函式。該函式在應用引導啟動過程中執行，所需的資料在啟動時可用。

請注意，所提供的初始化器是在注入上下文中執行的。

以前，這是透過 APP_INITIALIZER 令牌實現的，但現在已棄用。
