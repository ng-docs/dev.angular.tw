包装一个函数以在 fakeAsync Zone 中执行：

  • 通过调用 flushMicrotasks() 手动执行微任务。
  • 计时器是同步的； tick() 模拟异步时间的流逝。

如果函数末尾有任何待处理的计时器，则将引发异常。

可用于包装 inject() 调用。
