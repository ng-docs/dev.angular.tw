@angular/core

BaseResourceOptions

interface
實驗性
程式碼

用於建立資源的的 resource 函式的選項。

API

  
    interface BaseResourceOptions<T, R> {}

request

(() => R) | undefined

一個回應式函式，用於確定要發出的請求。每當請求發生變化時，將觸發載入器以獲取資源的新值。

如果沒有提供請求函式，則載入器將不會重新執行，除非重新載入資源。

defaultValue

NoInfer<T> | undefined

當服務器值不可用時（例如，當資源仍在載入或處於錯誤狀態時），將從資源回傳的值。

equal

ValueEqualityFn<T> | undefined

用於比較載入器回傳值的相等性函式。

injector

Injector | undefined

覆蓋 resource 使用的 Injector。
