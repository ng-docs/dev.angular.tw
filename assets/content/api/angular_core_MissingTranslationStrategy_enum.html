在系统启动时使用此枚举作为 bootstrapModule 的一个选项来定义策略，编译器应该在缺少翻译的情况下使用：

  • Error：如果缺少翻译，则抛出该错误。
  • Warning（默认）：在控制台和/或应用外壳中显示警告。
  • Ignore：什么都不做。
