在运行时运行 Angular 编译器的低级服务，用于创建 ComponentFactory ，之后可用于创建和渲染组件实例。

每个 @NgModule 都为其注入器提供一个自身的 Compiler，该编译器将使用此 ng 模块的指令/管道来编译组件。
