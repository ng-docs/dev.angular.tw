描述

用於在執行時執行 angular 編譯器的底層服務，以建立 ComponentFactory，稍後可用於建立和渲染 Component 實例。

每個 @NgModule 都為其注入器提供了一個自己的 Compiler，它將使用 ng 模組的指令/通道來編譯元件。
