允许从 Angular 使用 AngularJS 组件的帮助器类。

支持 AOT 编译的混合升级应用的 upgrade/static 库的一部分。

此帮助器类应该用作创建包装需要“升级”的 AngularJS 组件的 Angular 指令的基类。

例子

假设你有一个名为 ng1Hero 的 AngularJS 组件，需要在 Angular 模板中使用。

我们必须创建一个 Directive，以便在 Angular 模板中使用此 AngularJS 组件。

在这个示例中，你可以看到我们必须继承 UpgradeComponent 基类，同时还要提供一个 {@link Directive @Directive} 装饰器。这是因为 AOT 编译器要求这些信息在编译时静态可用。

请注意，我们必须执行以下操作：

  • 指定指令的选择器 ( ng1-hero )
  • 指定 AngularJS 组件期望的所有输入和输出
  • 继承 UpgradeComponent
  • call the base class from the constructor, passing
    • 组件的 AngularJS 名称 ( ng1Hero )
    • 组件包装器的 ElementRef 和 Injector
