一组可以为动画定义参数类别的常量。

一个对应的函数为每个类别定义一组参数，并将它们收集到一个相应的 AnimationMetadata 对象中。

状态

把一个命名的动画状态和一组 CSS 样式关联起来。 参见 state()。

过渡

从一个动画状态到另一个状态的过渡数据。参见 transition()

序列

包含一组动画步骤。参见 sequence()

组

包含一组动画步骤。参见 animations/group group()

动画

包含一个动画步骤。参见 animate()

关键帧动画

包含一组动画步骤。参见 keyframes()

样式

包含一组 CSS 属性值对，成为一个命名样式。参见 style()

触发器

将动画与一个可以附着到元素的进入触发器关联。参见 trigger()

参考手册

包含一个可复用的动画。参见 animation()

AnimateChild

包含用于执行由查询返回的子动画的数据。参见 animateChild()

AnimateRef

包含可复用动画的动画参数。参见 useAnimation()

查询

包含子动画查询数据。参见 query()

交错

包含交错动画序列的数据。参见 stagger()
