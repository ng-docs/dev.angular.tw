AnimationMetadataType

列舉
程式碼

可以為動畫定義的引數類別常數。

API

  
    enum AnimationMetadataType {}

狀態

將命名的動畫狀態與一組 CSS 樣式關聯起來。 請參閱 state()

轉場

用於表示從一個動畫狀態到另一個動畫狀態的轉場的資料。 請參閱 transition()

序列

包含一組動畫步驟。 請參閱 sequence()

組

包含一組動畫步驟。 請參閱 <a href="/api/animations/group"><code>group()</code></a>

動畫

包含一個動畫步驟。 請參閱 animate()

關鍵幀

包含一組動畫步驟。 請參閱 keyframes()

樣式

將一組 CSS 屬性-值對包含到命名的樣式中。 請參閱 style()

觸發器

將動畫與可以附著到元素上的進入觸發器相關聯。 請參閱 trigger()

引用

包含一個可複用的動畫。 請參閱 animation()

AnimateChild

包含用於執行查詢回傳的子動畫的資料。 請參閱 animateChild()

AnimateRef

包含可複用動畫的動畫引數。 請參閱 useAnimation()

查詢

包含子動畫查詢資料。 請參閱 query()

交錯

包含用於交錯動畫序列的資料。 請參閱 stagger()

描述

可以為動畫定義的引數類別常數。

相應的函式為每個類別定義一組引數，並將它們收集到相應的 AnimationMetadata 物件中。
