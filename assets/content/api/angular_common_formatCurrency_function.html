formatCurrency

string

使用区域设置规则将数字格式化为货币。

@paramvaluenumber

要格式化的数字。

@paramlocalestring

用于要使用的语言环境格式规则的语言环境代码。

@paramcurrencystring

包含货币符号或其名称的字符串，比如 “$” 或 “加元”。在输出字符串中使用，但不影响该函数的操作。

@paramcurrencyCodestring

ISO 4217 货币代码，比如 USD 表示美元，EUR 表示欧元。用于确定小数部分的位数。

@paramdigitsInfostring

十进制表示选项，由以下格式的字符串指定： {minIntegerDigits}.{minFractionDigits}-{maxFractionDigits}。参见 DecimalPipe 了解更多详情。

@returnsstring
