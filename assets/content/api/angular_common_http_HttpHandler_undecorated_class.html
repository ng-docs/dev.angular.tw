将一个 HttpRequest转换为 HttpEvent流，其中一个很可能是 HttpResponse。

HttpHandler是可注入的。当被注入时，处理器实例将请求派发给链中的第一个拦截器，第一个拦截器再派发给第二个拦截器，如此类推，最终到达 HttpBackend。

在 HttpInterceptor中， HttpHandler参数是链中的下一个拦截器。
