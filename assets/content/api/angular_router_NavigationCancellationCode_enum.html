@angular/router

NavigationCancellationCode

列舉
程式碼

用於 Router 的 NavigationCancel 事件的程式碼，以指示導向失敗的原因。

API

  
    enum NavigationCancellationCode {}

重導向

導向失敗，因為守衛回傳了一個 UrlTree 以進行重導向。

SupersededByNewNavigation

導向失敗，因為啟動了一個更新的導向。

NoDataFromResolver

導向失敗，因為其中一個求解器在完成時沒有發出值。

GuardRejected

導向失敗，因為守衛回傳了 false。
