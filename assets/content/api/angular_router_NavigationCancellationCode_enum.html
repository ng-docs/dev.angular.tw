@angular/router

NavigationCancellationCode

enum
code

Router 的 NavigationCancel 事件的代码，用于指示导航失败的原因。

enum NavigationCancellationCode {
}

重定向

导航失败，因为某个守卫返回了一个 UrlTree 进行重定向。

SupersededByNewNavigation

导航失败，因为启动了更新的导航。

NoDataFromResolver

导航失败，因为其中一个解析器在未发出值的情况下完成。

GuardRejected

导航失败，因为守卫返回 false。
