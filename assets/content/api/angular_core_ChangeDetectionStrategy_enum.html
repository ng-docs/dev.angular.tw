ChangeDetectionStrategy

列舉
程式碼

預設變更檢測器用來檢測變更的策略。 當設定後，將在下次變更檢測觸發時生效。

變更檢測用法跳過元件子樹

API

  
    enum ChangeDetectionStrategy {}

OnPush

使用 CheckOnce 策略，意味著自動變更檢測將被停用，直到透過將策略設定為 Default (CheckAlways) 重新啟用。 變更檢測仍然可以被顯式呼叫。 此策略適用於所有子指令，並且不能被重寫。

預設

使用預設的 CheckAlways 策略，在該策略中，變更檢測是自動的，直到被顯式停用。
