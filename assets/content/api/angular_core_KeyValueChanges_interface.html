@angular/core

KeyValueChanges

interface
程式碼

一個物件，用於描述自上次呼叫 KeyValueDiffer#diff() 以來 Map 或 {[k:string]: string} 中的更改。

API

  
    interface KeyValueChanges<K, V> {}

forEachItem

void

迭代所有更改。KeyValueChangeRecord 將包含關於每個條目更改的訊息。

@paramfn(r: KeyValueChangeRecord<K, V>) => void
@returnsvoid

forEachPreviousItem

void

按照原始 Map 的順序迭代更改，顯示原始條目已移動到哪裡。

@paramfn(r: KeyValueChangeRecord<K, V>) => void
@returnsvoid

forEachChangedItem

void

迭代值已更改的所有鍵。

@paramfn(r: KeyValueChangeRecord<K, V>) => void
@returnsvoid

forEachAddedItem

void

迭代所有已新增的條目。

@paramfn(r: KeyValueChangeRecord<K, V>) => void
@returnsvoid

forEachRemovedItem

void

迭代所有已移除的條目。

@paramfn(r: KeyValueChangeRecord<K, V>) => void
@returnsvoid
