@angular/common/http

HttpResourceOptions

interface
實驗性的
程式碼

用於建立 httpResource 的選項。

API

  
    interface HttpResourceOptions<TResult, TRaw> {}

解析

((value: TRaw) => TResult) | undefined

在 HTTP 請求的結果傳遞給資源之前對其進行轉換。

parse 以其原始型別（例如，對於 JSON 資料，為 unknown）接收來自 HTTP 層的值。 它可用於驗證或轉換資源的型別，並回傳更具體的型別。這對於使用執行時模式驗證 函式庫（如 Zod）驗證後端回應也很有用。

defaultValue

NoInfer<TResult> | undefined

資源在空閒、載入或錯誤狀態時將採用的值。

如果未設定，資源將使用 undefined 作為其預設值。

injector

Injector | undefined

要在其中建立 httpResource 的 Injector。

如果未提供，則將使用當前的注入上下文（透過 inject）。

equal

ValueEqualityFn<NoInfer<TResult>> | undefined

一個比較函式，用於定義回應值的相等性。
