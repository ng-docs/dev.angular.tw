NgControlStatus

directive
code

指令自动应用于 Angular 表单控件，该控件会根据控件状态设置 CSS 类。

class NgControlStatus {
}

应用的 CSS 类

当这些属性变为 true 时，将应用以下类：

  • ng-valid
  • ng-invalid
  • ng-pending
  • ng-pristine
  • ng-dirty
  • ng-untouched
  • ng-touched
