@angular/core

OnInit

介面
程式碼

這是一個生命週期鉤子，當 Angular 初始化完指令的所有資料繫結屬性之後呼叫。 定義 ngOnInit() 方法以處理任何額外的初始化任務。

AfterContentInit生命週期鉤子指南

API

  
    interface OnInit {}

ngOnInit

void

這是一個回呼方法，在預設變更檢測器首次檢查完指令的資料繫結屬性之後，並且在檢查任何檢視或內容子項之前立即呼叫。 它僅在指令實例化時調用一次。

@returnsvoid

使用須知
