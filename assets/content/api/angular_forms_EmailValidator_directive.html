将 email验证器添加到带有 email属性标记的控件的指令。该指令提供了 NG_VALIDATORS多提供者列表。

电子邮件验证基于 WHATWG HTML 规范，并进行了一些增强以包含更多 RFC 规则。更多信息，请参阅Validator.email 页面。

添加 email 验证器

以下示例展示了如何向附加到ngModel绑定的输入添加电子邮件验证器。

          
            <input type="email" name="email" ngModel email><input type="email" name="email" ngModel email="true"><input type="email" name="email" ngModel [email]="true">
