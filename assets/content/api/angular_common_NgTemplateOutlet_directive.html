从准备好的 TemplateRef 插入一个嵌入视图。

你可以通过设置 [ngTemplateOutletContext] 将一个上下文对象附着到 EmbeddedViewRef。 [ngTemplateOutletContext] 应该是一个对象，该对象的键将可供本地模板的 let 声明进行绑定。

          
            <ng-container *ngTemplateOutlet="templateRefExp; context: contextExp"></ng-container>

在上下文对象中使用 $implicit 这个 key 会把对应的值设置为默认值。

范例

ngTemplateOutletContext
