style

AnimationStyleMetadata

宣告一個鍵/值物件，其中包含 CSS 屬性/樣式，然後可以將其用於動畫 state、動畫 sequence 中，或用作呼叫 animate() 和 keyframes() 的樣式資料。

@paramtokens"*" | { [key: string]: string | number; } | ("*" | { [key: string]: string | number; })[]

與動畫狀態關聯的一組 CSS 樣式或 HTML 樣式。 該值可以是以下任何一種：

  • 將 CSS 屬性與值關聯的鍵值樣式對。
  • 鍵值樣式對的陣列。
  • 星號 (*)，用於使用自動樣式，其中樣式源自正在動畫化的元素，並在動畫開始時應用於動畫。

自動樣式可用於定義依賴於佈局或其他環境因素的狀態。

@returnsAnimationStyleMetadata

用法說明

以下範例建立動畫樣式，這些樣式收集一組 CSS 屬性值：

          
// string values for CSS propertiesstyle({ background: "red", color: "blue" })// numerical pixel valuesstyle({ width: 100, height: 0 })

以下範例使用自動樣式以允許元素從高度 0 動畫化到其完整高度：

          
style({ height: 0 }),animate("1s", style({ height: "*" }))
