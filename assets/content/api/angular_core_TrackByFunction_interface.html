描述

一個可選的函式，傳入 NgForOf 指令，用於自定義 NgForOf 如何唯一標識可迭代物件中的條目。

NgForOf 需要唯一標識可迭代物件中的條目，以便在可迭代物件中的條目重新排序、新增新條目或刪除現有條目時，正確執行 DOM 更新。

在所有這些場景中，通常都希望僅更新與受更改影響的條目關聯的 DOM 元素。此行為對於以下方面非常重要：

  • 當可迭代物件被修改時，保留任何特定於 DOM 的 UI 狀態（如游標位置、焦點、文字選擇）
  • 啟用條目新增、刪除和可迭代物件重新排序的動畫
  • 當使用 NgForOf 動態填充巢狀的 <option> 元素並且繫結的可迭代物件被更新時，保留 <select> 元素的值

自定義 trackBy 函式的一個常見用途是當 NgForOf 迭代的模型包含具有唯一識別符號的屬性（Property）時。例如，給定一個模型：

          
class User {  id: number;  name: string;  ...}

自定義 trackBy 函式可能如下所示：

          
function userTrackBy(index, user) {  return user.id;}

自定義 trackBy 函式必須具有以下幾個屬性：

  • 是冪等的（沒有副作用，並且對於給定的輸入始終回傳相同的值）
  • 為所有唯一輸入回傳唯一值
  • 速度快
