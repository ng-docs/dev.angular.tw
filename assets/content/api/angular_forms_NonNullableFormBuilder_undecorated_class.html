@angular/forms

NonNullableFormBuilder

類
程式碼

NonNullableFormBuilder 類似於 FormBuilder，但自動建立的 FormControl 元素具有 {nonNullable: true} 並且是不可為空的。

API

  
    abstract class NonNullableFormBuilder {}
