檢索給定語言環境的每週的第一天。

@deprecated

Angular 建議依賴 Intl API 來實現 i18n。 Intl 的 getWeekInfo 具有部分支援（Chromium M99 和 Safari 17）。 你可能需要依賴以下替代方案：

  • 諸如 Luxon 之類的函式庫依賴於 Intl，但如果不支援 getWeekInfo，則回退到 ISO 8601 定義（星期一）。
  • 其他函式庫（如 date-fns、day.js 或 weekstart 函式庫）為每週的第一天提供他們自己基於語言環境的資料。
@paramlocalestring

要使用的語言環境格式規則的語言環境程式碼。
