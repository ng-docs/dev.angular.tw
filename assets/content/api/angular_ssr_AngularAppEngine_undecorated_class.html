@angular/ssr

AngularAppEngine

類
開發者預覽

Angular 伺服器端應用引擎。 管理 Angular 伺服器端應用（包括本地化的應用），處理渲染請求， 並可選擇在渲染前轉換索引 HTML。

API

  
    class AngularAppEngine {}

handle

Promise<Response | null>

透過提供預渲染內容、執行伺服器端渲染或根據 RenderMode 設定為用戶端渲染路由交付靜態檔案來處理傳入的 HTTP 請求。

@paramrequestRequest
  • 要處理的 HTTP 請求。
@paramrequestContextunknown
  • 用於渲染的可選上下文，例如與請求關聯的元資料。
@returnsPromise<Response | null>
用法說明

對 https://www.example.com/page/index.html 的請求將為 https://www.example.com/page 對應的 Angular 路由啟動伺服器或進行渲染。

用法說明

此類應實例化一次，並在伺服器端應用中用作單例，以確保渲染請求和資源管理的一致處理。
