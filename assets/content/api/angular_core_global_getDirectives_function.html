getDirectives

function
code

檢索與給定 DOM 元素關聯的指令實例。不包括元件實例。

getDirectives(
  node: Node
): {}[];

給定以下 DOM 結構：

          
            <app-root>  <button my-button></button>  <my-comp></my-comp></app-root>

呼叫 getDirectives 方法獲取 <button> 上的指令將回傳一個陣列，其中包含與該 DOM 節點關聯的 MyButton 指令實例。

呼叫 getDirectives 方法獲取 <my-comp> 上的指令將回傳一個空陣列。
