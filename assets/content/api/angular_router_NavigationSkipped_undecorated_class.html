@angular/router

NavigationSkipped

類別
程式碼

當導向被跳過時觸發的事件。 發生這種情況可能有多種原因，包括 onSameUrlHandling 設定為 ignore，並且導向 URL 與當前狀態沒有區別。

API

  
    class NavigationSkipped extends RouterEvent {}

@paramreasonstring

@paramcodeNavigationSkippedCode | undefined

一個程式碼，用於指示導向被跳過的原因。此程式碼對於該原因來說是穩定的，可以依賴它，而 reason 字串可能會更改，不應在生產環境中使用。
