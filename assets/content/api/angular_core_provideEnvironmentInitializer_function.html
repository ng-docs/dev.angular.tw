provideEnvironmentInitializer

EnvironmentProviders

此函式用於提供初始化函式，這些函式將在構造環境注入器時執行。

請注意，所提供的初始化器是在注入上下文中執行的。

以前，這是透過 ENVIRONMENT_INITIALIZER 令牌實現的，但現在已棄用。
