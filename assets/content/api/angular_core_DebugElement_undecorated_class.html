properties

{ [key: string]: any; }

獲取元素 Property 名到 Property 值的對映表。

此對映表包括：

  • 常規 Property 繫結（例如 [id]="id"）
  • 宿主 Property 繫結（例如 host: { '[id]': "id" }）
  • 內嵌 Property 繫結（例如 `id="{{ value }}"`）

它不包括：

  • 輸入 Property 繫結（例如 [myCustomInput]="value"）
  • Attribute 繫結（例如 [attr.role]="menu"）

注意：此物件的值將始終為 true。如果類鍵在元素上不存在，則不會出現在 KV 物件中。
