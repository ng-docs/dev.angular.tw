properties

{ [key: string]: any; }

获取元素的属性名称到属性值的映射。

这张地图包括：

  • 常规属性绑定（例如 [id]="id"）
  • 宿主属性绑定（例如 host: { '[id]': "id" }）
  • 插值属性绑定（例如 `id="{{ value }}"）

它不包括：

  • 输入属性绑定（例如 [myCustomInput]="value"）
  • 属性绑定（例如 [attr.role]="menu"）
