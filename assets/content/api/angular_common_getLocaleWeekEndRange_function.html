在给定语言环境中被视为周末的工作日范围。

@deprecated

Angular 推荐依赖 Intl API 进行国际化（i18n）。Intl 的 getWeekInfo 有部分支持（Chromium M99 和 Safari 17）。像 Luxon 这样的库依赖 Intl，但如果 getWeekInfo 不支持，则回退到 ISO 8601 定义（周六+周日）。

@paramlocalestring

用于要使用的语言环境格式规则的语言环境代码。

@returns[WeekDay, WeekDay]
