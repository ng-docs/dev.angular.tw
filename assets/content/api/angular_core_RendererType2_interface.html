@angular/core

RendererType2

interface
code

由 RendererFactory2 使用，以将自定义渲染数据和样式与渲染实现相关联。

interface RendererType2 {
}

id

string

创建新的封装样式时使用的新渲染器的唯一标识字符串。

encapsulation

ViewEncapsulation

视图封装类型，它确定如何将样式应用于 DOM 元素。为下列值之一

  • Emulated（默认）：模拟样式的原生作用域。
  • Native：使用渲染器的原生封装机制。
  • ShadowDom：使用现代Shadow DOM并为组件的宿主元素创建一个 ShadowRoot。
  • None：不提供任何模板或样式封装。

styles

string[]

定义要存储在渲染器实例上的 CSS 样式。

data

{ [kind: string]: any; }

定义要存储在渲染器实例上的任意由开发人员定义的数据。这对于要委托其他渲染器实现的渲染器很有用。
