将 URL 字符串序列化和反序列化为 URL 树。

URL 序列化策略是可定制的。可以通过提供自定义的 UrlSerializer 来使所有 URL 不区分大小写。

有关 URL 序列化器的示例，请参阅 DefaultUrlSerializer。
