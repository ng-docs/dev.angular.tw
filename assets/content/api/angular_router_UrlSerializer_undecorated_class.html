說明

將 URL 字串序列化和反序列化為 URL 樹。

URL 序列化策略是可自定義的。你可以透過提供自定義的 UrlSerializer 來使所有 URL 忽略大小寫。

有關 URL 序列化器的範例，請參閱 DefaultUrlSerializer。
