检索给定语言环境下的数字格式。

数字使用模式进行格式化，例如 #,###.00。例如，模式 #,###.00 用于格式化数字 12345.678 时，可能的结果是 "12'345,678"。如果你的语言的分组分隔符是撇号，十进制分隔符是逗号，会出现这种情况。

重要：字符 .、 ,、 0、 #（以及下方其他字符）是特殊占位符，代表小数点分隔符等，并不是实际字符。你不能「翻译」这些占位符。例如，即使你的语言中小数点用逗号表示，也不要将 . 改成 ,。这些符号应该使用适当的 NumberSymbol 替换为本地等价符号。

以下是数字模式中使用的特殊字符：

符号 含义
. 自动替换为用作小数点的字符。
, 替换为（千）“分组”分隔符。
0 替换为一个数字（如果没有足够的数字，则为零）。
# 用数字代替（如果数字不足，则不进行任何替换）。
¤ 替换为货币符号，比如 $ 或 USD。
% 标记百分比格式。％符号可能会更改位置，但必须保留。
E 标记科学计数法格式。 E 符号可能会改变位置，但必须保留。
' 表示文本字面量的特殊字符，用 ASCII 单引号引起来。
