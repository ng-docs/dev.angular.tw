@angular/common

DATE_PIPE_DEFAULT_OPTIONS

常數
程式碼

DI 令牌，用於為應用中的 DatePipe 實例提供預設設定。 該值是一個物件，可以包含以下欄位：

  • dateFormat：設定預設日期格式。 如果未提供，DatePipe 將使用 'mediumDate' 作為值。
  • timezone：設定預設時區。 如果未提供，DatePipe 將使用最終使用者的本地系統時區。
DatePipeConfig

API

  
    const DATE_PIPE_DEFAULT_OPTIONS: InjectionToken<DatePipeConfig>;

用法說明

可以透過提供帶有此介面值的令牌來覆蓋各種日期通道的預設值。

例如：

透過使用此令牌提供一個值來覆蓋預設日期格式：

          
providers: [  {provide: DATE_PIPE_DEFAULT_OPTIONS, useValue: {dateFormat: 'shortDate'}}]

透過使用此令牌提供一個值來覆蓋預設時區：

          
providers: [  {provide: DATE_PIPE_DEFAULT_OPTIONS, useValue: {timezone: '-1200'}}]
