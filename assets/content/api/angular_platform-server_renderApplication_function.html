@parambootstrap() => Promise<ApplicationRef>

一个方法，调用时返回一个 Promise，一旦求解完成返回一个 ApplicationRef 实例。

@paramoptions{ document?: string | Document; url?: string; platformProviders?: Provider[]; }

渲染操作的附加配置：

  • document - 要渲染的页面文档，可以是 HTML 字符串或对 document 实例的引用。
  • url - 当前呈现请求的 URL。
  • platformProviders - 当前渲染请求的平台级提供程序。
@returnsPromise<string>
