}

The input function allows declaration of inputs in directives and components.

The function exposes an API for also declaring required inputs via the input.required function.

Initialize an input in your directive or component by declaring a class field and initializing it with the input() or input.required() function.

          
            @Directive({..})export class MyDir {  firstName = input<string>();            // string|undefined  lastName = input.required<string>();    // string  age = input(0);                         // number}

required

{ <ReadT>(opts?: InputOptionsWithoutTransform<ReadT>): InputSignal<ReadT>; <ReadT, WriteT>(opts: InputOptionsWithTransform<ReadT, WriteT>): InputSignalWithTransform<...>; }

Initializes a required input.

Users of your directive/component need to bind to this input. If unset, a compile time error will be reported.
