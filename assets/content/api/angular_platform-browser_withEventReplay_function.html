启用对在水合逻辑完成之前发生在页面上的用户事件（例如 click）的重放支持。一旦应用被水合，所有捕获的事件将被重放，相关的事件监听器将被执行。

当使用 bootstrapApplication 函数时，启用事件重放的基本示例：

          
            bootstrapApplication(AppComponent, {  providers: [provideClientHydration(withEventReplay())]});
