getListeners

function
code

检索与 DOM 元素关联的事件监听器的列表。该列表包含宿主监听器，但不包含在 Angular 上下文之外定义的事件监听器（比如，通过 addEventListener）。

getListeners(
  element: Element
): Listener[];

给定以下 DOM 结构：

          
            <app-root>  <div (click)="doSomething()"></div></app-root>

调用 getListeners 方法获取 <div> 上的事件监听器将返回如下对象：

          
            {  name: 'click',  element: <div>,  callback: () => doSomething(),  useCapture: false}
