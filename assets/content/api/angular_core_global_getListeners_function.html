getListeners

function
code

檢索與 DOM 元素關聯的事件監聽器的清單。該清單包含宿主監聽器，但不包含在 Angular 上下文之外定義的事件監聽器（比如，透過 addEventListener）。

getListeners(
  element: Element
): Listener[];

給定以下 DOM 結構：

          
            <app-root>  <div (click)="doSomething()"></div></app-root>

呼叫 getListeners 方法獲取 <div> 上的事件監聽器將回傳如下物件：

          
            {  name: 'click',  element: <div>,  callback: () => doSomething(),  useCapture: false}
