}

void

将指定的输入名称更新为新值。使用此方法将正确标记使用 OnPush 变更检测策略的组件进行检查。还将确保在动态创建的组件进行变更检测时运行 OnChanges 生命周期钩子。

@paramnamestring

输入的名称。

@paramvalueunknown
