@angular/core

ComponentRef

類
程式碼

表示由 ComponentFactory 建立的元件。 提供對元件實例和相關物件的訪問，並提供銷燬該實例的方法。

API

  
    abstract class ComponentRef<C> {}

setInput

void

將指定的輸入名更新為新值。使用此方法將正確地標記要檢查的元件，該組件使用了 OnPush 變更檢測策略。它還將確保在動態建立的元件進行變更檢測時，OnChanges 生命週期鉤子會執行。

@paramnamestring

輸入的名稱。

@paramvalueunknown

輸入的新值。

@returnsvoid

location

ElementRef<any>

此元件實例的宿主或錨點元素。

injector

Injector

此元件實例的依賴注入器。

instance

C

此元件實例。

hostView

ViewRef

由此元件實例的樣板定義的宿主檢視。

changeDetectorRef

ChangeDetectorRef

此元件實例的變更檢測器。

componentType

Type<any>

此元件的型別（由 ComponentFactory 類建立）。

destroy

void

銷燬元件實例以及與之關聯的所有資料結構。

@returnsvoid

onDestroy

void

一個生命週期鉤子，為元件提供額外的開發者自定義的清理功能。

@paramcallbackFunction

一個處理函式，用於清理與此元件關聯的開發者定義的資料。當呼叫 destroy() 方法時調用。

@returnsvoid
