实例化一个 Component 类型并将其宿主视图插入当前视图。 NgComponentOutlet 提供了一种动态创建组件的声明式方法。

NgComponentOutlet 需要一个组件类型，如果设置了假值，视图将清空，任何现有组件将被销毁。

微调控制

你可以使用以下可选属性来控制组件的创建过程：

  • ngComponentOutletInputs：可选的组件输入对象，将绑定到组件。

  • ngComponentOutletInjector：可选的自定义 {@link Injector}，将作为组件的父级使用。默认为当前视图容器的注入器。

  • ngComponentOutletContent：要插入组件的 content 部分的可项目节点的可选列表（如果存在）。

  • ngComponentOutletNgModule：可选的 NgModule 类引用，以允许动态加载另一个模块，然后从该模块加载组件。

  • ngComponentOutletNgModuleFactory：已弃用的配置选项，允许提供可选的 NgModule 工厂，以便动态加载另一个模块，然后从该模块加载一个组件。请改用 ngComponentOutletNgModule。

语法

简单

          
            <ng-container *ngComponentOutlet="componentTypeExpression"></ng-container>

带有输入

          
            <ng-container *ngComponentOutlet="componentTypeExpression;                                  inputs: inputsExpression;"></ng-container>

定制的注入器/内容

          
            <ng-container *ngComponentOutlet="componentTypeExpression;                                  injector: injectorExpression;                                  content: contentNodesExpression;"></ng-container>

定制的 ngModule 引用

          
            <ng-container *ngComponentOutlet="componentTypeExpression;                                  ngModule: ngModuleClass;"></ng-container>
