描述

實例化一個 Component 型別，並將其宿主檢視插入到當前檢視中。 NgComponentOutlet 為動態元件建立提供了一種宣告式方法。

NgComponentOutlet 需要一個元件類別型；如果設定了假值，則檢視將被清除，並且任何現有元件都將被銷燬。

用法說明

微調控制

你可以透過使用以下可選屬性來控制組件建立過程：

  • ngComponentOutletInputs：可選的元件輸入物件，它將被繫結到元件。

  • ngComponentOutletInjector：可選的自定義 Injector，它將用作元件的父注入器。預設為當前檢視容器的注入器。

  • ngComponentOutletContent：可投影節點的可選清單，以插入到元件的內容部分（如果存在）。

  • ngComponentOutletNgModule：可選的 NgModule 類別引用，以允許動態載入另一個模組，然後從該模組載入元件。

  • ngComponentOutletNgModuleFactory：已棄用的設定選項，該選項允許提供可選的 NgModule 工廠，以允許動態載入另一個模組，然後從該模組載入元件。請改用 ngComponentOutletNgModule。

語法

簡單

          
<ng-container *ngComponentOutlet="componentTypeExpression"></ng-container>

帶輸入

          
<ng-container *ngComponentOutlet="componentTypeExpression;                                  inputs: inputsExpression;"></ng-container>

自定義注入器/內容

          
<ng-container *ngComponentOutlet="componentTypeExpression;                                  injector: injectorExpression;                                  content: contentNodesExpression;"></ng-container>

自定義 NgModule 引用

          
<ng-container *ngComponentOutlet="componentTypeExpression;                                  ngModule: ngModuleClass;"></ng-container>
