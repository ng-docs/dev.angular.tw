@angular/core

model

初始化器 API
程式碼

model 宣告一個可寫的訊號，該訊號在包含指令上作為輸入/輸出對公開。

輸入名取自類別成員或 alias 選項。 輸出名透過獲取輸入名並附加 Change 生成。

API

  
    
  
  
function model<T>(): ModelSignal<T | undefined>;

使用 undefined 的初始值初始化 T 型別的模型。 Angular 將隱式地使用 undefined 作為初始值。

@returnsModelSignal<T | undefined>
function model<T>(initialValue: T, opts?: ModelOptions | undefined): ModelSignal<T>;

使用給定的初始值初始化 T 型別的模型。

@paraminitialValueT
@paramoptsModelOptions | undefined
@returnsModelSignal<T>
function model.required<T>(opts?: ModelOptions | undefined): ModelSignal<T>;

初始化一個必要的模型。

你的指令/元件的使用者需要繫結到模型的輸入端。 如果未設定，將報告編譯時錯誤。

@paramoptsModelOptions | undefined
@returnsModelSignal<T>

使用須知

要使用 model()，請從 @angular/core 匯入此函式。

          
import {model} from '@angular/core`;

在你的元件內部，引入一個新的類別成員，並透過呼叫 model 或 model.required 來初始化它。

          
@Directive({  ...})export class MyDir {  firstName = model<string>();            // ModelSignal<string|undefined>  lastName  = model.required<string>();   // ModelSignal<string>  age       = model(0);                   // ModelSignal<number>}

在你的元件樣板內部，你可以透過呼叫訊號來顯示 model 的值。

          
<span>{{firstName()}}</span>

更新 model 相當於更新一個可寫訊號。

          
updateName(newFirstName: string): void {  this.firstName.set(newFirstName);}
