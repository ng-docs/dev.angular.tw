从一个异步原语中解出一个值。

async 管道订阅一个 Observable 或 Promise 并返回其发出的最新值。当发出新值时， async 管道标记组件以检查变化。当组件被销毁时， async 管道自动退订以避免潜在的内存泄漏。当表达式的引用发生变化时， async 管道自动从旧的 Observable 或 Promise 退订并订阅新的。
