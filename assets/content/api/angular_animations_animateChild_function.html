animateChild

AnimationAnimateChildMetadata

在動畫序列中執行查詢到的內部動畫元素。

@paramoptionsAnimateChildOptions | null

一個 options 物件，可以包含動畫啟動的延遲值，以及開發者定義的引數的額外重寫值。

@returnsAnimationAnimateChildMetadata

用法說明

每次在 Angular 中觸發動畫時，父動畫都具有優先權，並且任何子動畫都會被阻止。為了使子動畫執行，父動畫必須查詢每個包含子動畫的元素，並使用此函式執行它們。

請注意，此屬性旨在與 query() 一起使用，並且它僅適用於使用 Angular 動畫庫賦值的動畫。CSS 關鍵幀和轉場不由本 API 處理。
