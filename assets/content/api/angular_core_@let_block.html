@angular/core

@let

控制區塊
程式碼

@let 允許你定義一個區域性變數並在整個範本中複用它。

語法 

          
@let name = user.name;@let data = data$ | async;

描述

@let 物件宣告類似於 JavaScript 的 let，並且它們的值可以是任何有效的 Angular 表示式。每次執行範本時，這些表示式都會被重新求值。

與 JavaScript 變數類似，用 @let 宣告的變數可以用於：

  • 在它們被宣告之後
  • 在相同作用域和巢狀作用域的範圍內
          
@let user = user$ | async;@if (user) {  <h1>Hello, {{user.name}}</h1>  <user-avatar [photo]="user.photo"/>  <ul>    @for (snack of user.favoriteSnacks; track snack.id) {      <li>{{snack.name}}</li>    }  </ul>  <button (click)="update(user)">Update profile</button>}

語法定義

@let 語法的正式定義為：

  • @let 關鍵字。
  • 後面跟著一個或多個空格，不包括換行符。
  • 後面跟著一個有效的 JavaScript 名稱和零個或多個空格。
  • 後面跟著 = 符號和零個或多個空格。
  • 後面跟著一個 Angular 表示式，它可以是多行的。
  • 以 ; 符號結尾。

提示：關於此屬性的全面描述，請參見範本指南。
