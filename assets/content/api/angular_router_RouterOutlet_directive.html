一个占位符，Angular 会根据当前的路由器状态动态填充它。

每个出口可以具有唯一的名称，该 name 由可选的 name 属性确定。该名称不能动态设置或更改。如果未设置，则默认值为 “primary”。

          
            <router-outlet></router-outlet><router-outlet name='left'></router-outlet><router-outlet name='right'></router-outlet>

命名插座可以作为次级路由的目标。次级路由的 Route 对象有一个 outlet 属性用于标识目标插座：

{path: <base-path>, component: <component>, outlet: <target_outlet_name>}

使用命名插座和次级路由，你可以在同一个 RouterLink 指令中定位多个插座。

路由器在导航树中跟踪每个命名出口的单独分支，并在 URL 中生成该树的表示形式。辅助路由的 URL 使用以下语法同时指定主要路由和辅助路由：

http://base-path/primary-route-path(outlet-name:route-path)

路由器插座在实例化新组件时发出激活事件，在销毁组件时发出停用事件。当 RouteReuseStrategy 指示插座重新附着子树时，发出附着事件；当 RouteReuseStrategy 指示插座分离子树时，发出分离事件。

          
            <router-outlet  (activate)='onActivate($event)'  (deactivate)='onDeactivate($event)'  (attach)='onAttach($event)'  (detach)='onDetach($event)'></router-outlet>
