初始化一个内容子查询。

查询结果表示为包含所有匹配元素的只读集合信号。

通过声明一个类字段并使用 contentChildren() 函数初始化它，在组件中创建一个子查询。

          
            @Component({...})export class TestComponent {  headerEl = contentChildren<ElementRef>('h');   // Signal<ReadonlyArray<ElementRef>>}
