ExistingProvider

interface
code

配置 Injector 返回另一个 useExisting 标记的值。

interface ExistingProvider {
}

多值示例

provide

any

一个注入令牌。通常是 Type或 InjectionToken的实例，但也可以是 any。

multi

boolean

如果为 true，则注入器返回实例数组。这对于允许多个提供者散布在多个文件中以向公共令牌提供配置信息很有用。

useExisting

any

返回现有的 token。（等效于 injector.get(useExisting)）
