檢索與給定 DOM 元素關聯的元件實例。

getComponent(
  element: Element
): T;

給定以下 DOM 結構：

          
            <app-root>  <div>    <child-comp></child-comp>  </div></app-root>

呼叫 getComponent 方法獲取 <child-comp> 上的元件實例將回傳與該 DOM 元素關聯的 ChildComponent 實例。

在 <my-app> 上呼叫該函式將回傳 MyApp 實例。
