检索与给定 DOM 元素关联的组件实例。

getComponent(
  element: Element
): T;

给定以下 DOM 结构：

          
            <app-root>  <div>    <child-comp></child-comp>  </div></app-root>

调用 getComponent 方法获取 <child-comp> 上的组件实例将返回与该 DOM 元素关联的 ChildComponent 实例。

在 <my-app> 上调用该函数将返回 MyApp 实例。
