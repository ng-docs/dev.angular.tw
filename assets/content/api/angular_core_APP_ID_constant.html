@angular/core

APP_ID

常數
程式碼

一個 DI 令牌，表示字串 ID，主要用於在使用 ViewEncapsulation#Emulated 時為應用屬性和 CSS 樣式新增字首。

API

  
    const APP_ID: InjectionToken<string>;

說明

一個 DI 令牌，表示字串 ID，主要用於在使用 ViewEncapsulation#Emulated 時為應用屬性和 CSS 樣式新增字首。

在同一頁面上引導啟動多個應用（例如，使用 bootstrapApplication 呼叫）時，需要此令牌。在這種情況下，請確保這些應用具有不同的 APP_ID 值設定。例如：

          
bootstrapApplication(ComponentA, {  providers: [    { provide: APP_ID, useValue: 'app-a' },    // ... other providers ...  ]});bootstrapApplication(ComponentB, {  providers: [    { provide: APP_ID, useValue: 'app-b' },    // ... other providers ...  ]});

預設情況下，當只有一個應用被引導啟動時，你不需要提供 APP_ID 令牌（ng 將用作應用 ID）。
