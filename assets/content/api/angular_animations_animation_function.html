animation

AnimationReferenceMetadata

生成一個可複用的動畫，該動畫可以在另一個動畫或序列中透過呼叫 useAnimation() 函式來呼叫。

@paramstepsAnimationMetadata | AnimationMetadata[]

一個或多個動畫物件，由 animate() 或 sequence() 函式回傳，這些物件形成從一個狀態到另一個狀態的轉場。 當你傳遞一個陣列時，預設使用序列。

@paramoptionsAnimationOptions | null

一個選項物件，可以包含動畫啟動的延遲值，以及其他由開發者定義的引數。 為附加引數提供的值將用作預設值，並且重寫值可以在呼叫時傳遞給呼叫者。

@returnsAnimationReferenceMetadata

用法說明

以下範例定義了一個可複用的動畫，並提供了一些預設引數值。

          
var fadeAnimation = animation([  style({ opacity: '{{ start }}' }),  animate('{{ time }}',  style({ opacity: '{{ end }}'}))  ],  { params: { time: '1000ms', start: 0, end: 1 }});

以下程式碼透過呼叫 useAnimation() 來呼叫已定義的動畫，並傳入重寫的引數值。

          
useAnimation(fadeAnimation, {  params: {    time: '2s',    start: 1,    end: 0  }})

如果在此呼叫中缺少任何傳入的引數值，則會使用預設值。 如果在一個步驟動畫化之前缺少一個或多個引數值，則 useAnimation() 將丟擲一個錯誤。
