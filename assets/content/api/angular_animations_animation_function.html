通过调用 useAnimation() 函数生成可重用动画，可以在另一个动画或序列中调用。

animation(
  steps: AnimationMetadata | AnimationMetadata[],
  options?: AnimationOptions
): AnimationReferenceMetadata;

下面的例子定义了一个可复用的动画，提供了一些默认的参数值。

          
            var fadeAnimation = animation([  style({ opacity: '{{ start }}' }),  animate('{{ time }}',  style({ opacity: '{{ end }}'}))  ],  { params: { time: '1000ms', start: 0, end: 1 }});

以下调用 useAnimation() 所定义的动画，并传入覆盖参数值。

          
            useAnimation(fadeAnimation, {  params: {    time: '2s',    start: 1,    end: 0  }})

如果此调用中缺少任何传入的参数值，将使用默认值。如果在某个步骤动画之前缺少一个或多个参数值， useAnimation() 将抛出错误。

animation

AnimationReferenceMetadata

通过调用 useAnimation() 函数生成可重用动画，可以在另一个动画或序列中调用。

@paramstepsAnimationMetadata | AnimationMetadata[]

一个或多个动画对象（由 animate() 或 sequence() 函数返回），形成从一个状态到另一个状态的转换。默认情况下，当你传入一个数组时，会使用一个序列。

@paramoptionsAnimationOptions

一个选项对象，可以包含动画开始的延迟值，以及开发人员定义的参数的额外重写值。为额外参数提供的值作为默认值，重写值可以在调用时传递给调用者。

@returnsAnimationReferenceMetadata
