描述

一個介面，為開發 Router 的元件出口定義了契約。

出口充當佔位符，Angular 會根據當前的路由器狀態動態地填充它。

路由器出口應該透過 ChildrenOutletContexts#onChildOutletCreated 向 Router 註冊自己，並透過 ChildrenOutletContexts#onChildOutletDestroyed 登出。當 Router 識別到對應的 Route 時，它會在 ChildrenOutletContexts 中查詢已註冊的出口並激活它。
