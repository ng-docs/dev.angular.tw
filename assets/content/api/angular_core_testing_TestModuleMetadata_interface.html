errorOnUnknownElements

boolean | undefined

當元件的樣板中存在未知元素時，是否應該丟擲 NG0304 執行時錯誤。預設為 false，此時只會記錄錯誤。如果設定為 true，則會丟擲錯誤。

errorOnUnknownProperties

boolean | undefined

當元件的樣板中存在未知屬性（Property）時，是否應該丟擲錯誤。預設為 false，此時只會記錄錯誤。如果設定為 true，則會丟擲錯誤。

rethrowApplicationErrors

boolean | undefined

是否應該重新丟擲在應用變更檢測期間發生的錯誤。

當為 true 時，在應用變更檢測期間捕獲的錯誤將報告給 ErrorHandler 並重新丟擲，以防止它們在測試中被忽略。

當為 false 時，錯誤僅轉發到 ErrorHandler，預設情況下，它只會將錯誤記錄到控制台。

預設為 true。

deferBlockBehavior

DeferBlockBehavior | undefined

延遲控制區塊的行為方式應該是手動觸發還是正常播放。預設為 manual。
