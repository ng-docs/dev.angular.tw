@angular/core

AfterRenderOptions

interface
開發者預覽版
程式碼

傳遞給 afterRender 和 afterNextRender 的選項。

API

  
    interface AfterRenderOptions {}

injector

Injector | undefined

在建立期間使用的 Injector。

如果未提供，則將使用當前的注入上下文（透過 inject）。

manualCleanup

boolean | undefined

此 Hook 是否應要求手動清理。

如果這是 false（預設值），則此 Hook 將自動註冊自身，以便使用當前的 DestroyRef 進行清理。

phase

AfterRenderPhase | undefined
@deprecated

透過傳遞一個 spec-object 作為 afterRender 或 afterNextRender 的第一個引數（而不是函式），來指定回呼函式應在哪個階段執行。

回呼函式應在其中呼叫的階段。

預設為 AfterRenderPhase.MixedReadWrite。你應該選擇一個更具體的階段。有關更多訊息，請參閱 AfterRenderPhase。
