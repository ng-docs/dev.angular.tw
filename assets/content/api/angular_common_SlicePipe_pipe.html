code

从一个 Array 或 String 中创建其元素一个新子集（slice）。

class SlicePipe {
}

所有行为都基于 JavaScript API Array.prototype.slice() 和 String.prototype.slice() 的预期行为。

当操作 Array 时，返回的 Array 始终是一个副本 —— 即使返回了所有元素也是一样。

当操作空白值时，该管道也会返回空白值。

@paramvaluereadonly T[]

要进行切片的列表或字符串。

@paramstartnumber

要返回的子集的起始索引：

  • 正整数：返回 start 索引处的条目以及列表或字符串表达式中的所有后续条目。
  • 负整数：从末尾开始返回 start 索引处的条目以及列表或字符串表达式中的所有后续条目。
  • 如果为正且大于表达式的大小：返回空列表或字符串。
  • 如果为负且大于表达式的大小：返回整个列表或字符串。
@paramendnumber

要返回的子集的结束索引：

  • 省略：返回直到结尾的所有条目。
  • 如果为正数：返回列表或字符串中 end 索引之前的所有条目。
  • 如果为负数：返回从列表或字符串末尾的 end 索引之前的所有条目。
