NgStyle

指令
程式碼

一個屬性型指令，用於更新包含 HTML 元素的樣式。 設定一個或多個樣式屬性（Property），以冒號分隔的鍵值對形式指定。 鍵是樣式名，帶有可選的 .<unit> 字尾 （例如 'top.px'、'font-style.em'）。 值是要進行求值的表示式。 求值結果為非 null 值時，會以給定的單位表示，並賦值給給定的樣式屬性（Property）。 如果求值結果為 null，則會移除相應的樣式。

描述

一個屬性型指令，用於更新包含 HTML 元素的樣式。 設定一個或多個樣式屬性（Property），以冒號分隔的鍵值對形式指定。 鍵是樣式名，帶有可選的 .<unit> 字尾 （例如 'top.px'、'font-style.em'）。 值是要進行求值的表示式。 求值結果為非 null 值時，會以給定的單位表示，並賦值給給定的樣式屬性（Property）。 如果求值結果為 null，則會移除相應的樣式。

用法說明

將包含元素的寬度設定為表示式回傳的畫素值。

          
<some-element [ngStyle]="{'max-width.px': widthExp}">...</some-element>

使用回傳鍵值對的表示式設定樣式值的集合。

          
<some-element [ngStyle]="objExp">...</some-element>

對於更簡單的用例，你可以直接使用樣式繫結。 它不需要匯入指令。

將包含元素的字型設定為表示式的結果。

          
<some-element [style]="{'font-style': styleExp}">...</some-element>
