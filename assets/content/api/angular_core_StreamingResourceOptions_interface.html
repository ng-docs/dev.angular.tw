@angular/core

StreamingResourceOptions

interface
實驗性的
程式碼

用於建立資源時，傳遞給 resource 函式的選項。

API

  
    interface StreamingResourceOptions<T, R> extends BaseResourceOptions<T, R> {}

stream

ResourceStreamingLoader<T, R>

載入函式，它回傳一個 Promise，該 Promise 會求解為一個訊號，該訊號表示給定請求的資源值，該值會隨著從流中接收到新值而隨時間變化。

loader

undefined

不能同時指定 stream 和 loader。

request

(() => R) | undefined

一個回應式函式，用於確定要發出的請求。每當請求更改時，將觸發載入器以獲取資源的新值。

如果未提供請求函式，則除非重新載入資源，否則載入器不會重新執行。

defaultValue

NoInfer<T> | undefined

當服務器值不可用時（例如，當資源仍在載入中或處於錯誤狀態時），將從資源回傳的值。

equal

ValueEqualityFn<T> | undefined

用於比較載入器回傳值的相等性函式。

injector

Injector | undefined

改寫 resource 使用的 Injector。
