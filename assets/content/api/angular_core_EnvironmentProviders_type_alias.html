描述

封裝的 Provider，僅在建立 EnvironmentInjector 時（例如在 NgModule 中）被接受。

使用此套件裝器型別可以防止那些僅設計用於應用/環境注入器的提供者被意外地包含在 @Component.providers 中，並最終進入元件注入器。

此套件裝器型別可防止訪問內部的 Provider。
