CreateEffectOptions

interface
開發者預覽
程式碼

傳遞給 effect 函式的選項。

API

  
    interface CreateEffectOptions {}

injector

Injector | undefined

在其中建立副作用的 Injector。

如果未提供，則將使用當前的注入上下文（透過 inject）。

manualCleanup

boolean | undefined

effect 是否應要求手動清理。

如果為 false（預設值），則副作用將自動註冊自身，以便使用當前的 DestroyRef 進行清理。

forceRoot

true | undefined

始終建立一個根副作用（計劃為微任務），無論 effect 是否在元件內呼叫。

allowSignalWrites

boolean | undefined
@deprecated

不再需要，預設情況下允許訊號寫入。

debugName

string | undefined

副作用的除錯名稱。在 Angular DevTools 中用於標識副作用。
