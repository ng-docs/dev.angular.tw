传递给 effect 函数的选项。

interface CreateEffectOptions {
}

injector

Injector

用于创建效果的 Injector。

如果未提供，将使用当前的 注入上下文（通过 inject）。

manualCleanup

boolean

是否要求手动清理 effect。

如果此项为 false（默认值），效果会自动注册自身以便与当前的 DestroyRef 一起清理。

allowSignalWrites

boolean

此 effect 是否应该允许写入信号。

使用效果器通过写入信号来同步数据可能会导致混乱和潜在的不正确行为，应仅在必要时启用。
