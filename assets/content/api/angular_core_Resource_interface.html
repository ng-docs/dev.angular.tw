@angular/core

Resource

interface
實驗性的
程式碼

Resource 是一種非同步依賴項（例如，API 呼叫的結果），它透過訊號進行管理和傳遞。

API

  
    interface Resource<T> {}

value

Signal<T>

Resource 的當前值，如果沒有當前值，則為 undefined。

status

Signal<ResourceStatus>

Resource 的當前狀態，它描述了資源當前正在做什麼以及對其 value 的預期。

error

Signal<unknown>

當處於 error 狀態時，這將回傳來自 Resource 的最後一個已知錯誤。

isLoading

Signal<boolean>

此資源是否正在載入新值（或重新載入現有值）。

hasValue

boolean

此資源是否具有有效的當前值。

此函式是回應式的。

@returnsboolean

reload

boolean

指示資源重新載入它可能具有的任何非同步依賴項。

請注意，在發出實際後端請求之前，資源不會進入其重新載入狀態。

@returnsboolean

描述

Resource 是一種非同步依賴項（例如，API 呼叫的結果），它透過訊號進行管理和傳遞。

建立 Resource 的常用方法是透過 resource 函式，但各種其他 API 可能會提供 Resource 實例來描述它們自己的概念。
