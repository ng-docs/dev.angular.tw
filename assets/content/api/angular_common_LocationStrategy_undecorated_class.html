启用 Location 服务从浏览器的 URL 读取路由状态。Angular 提供了两种策略： HashLocationStrategy 和 PathLocationStrategy。

应用应使用 Router 或 Location 服务与应用的路由状态进行交互。

例如， HashLocationStrategy 生成类似 http://example.com#/foo 的 URL，而 PathLocationStrategy 生成等效的 URL http://example.com/foo。

有关更多信息，请参见这两个类。
