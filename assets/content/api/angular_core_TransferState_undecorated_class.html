描述

一個鍵值儲存，用於從伺服器端的應用傳輸到用戶端的應用。

TransferState 可用作可注入的令牌。 在用戶端，只需使用依賴注入（DI）注入此令牌並使用它，它將被延遲載入地初始化。 在服務器端，如果使用了 renderApplication 函式，則已包含它。否則，匯入 ServerTransferStateModule 模組以使 TransferState 可用。

儲存中的值使用 JSON.stringify/JSON.parse 進行序列化/反序列化。因此，只有布林值、數字、字串、null 和非類物件會以無損的方式進行序列化和反序列化。
