从服务端的应用程序传到客户端的应用程序的键值存储。

TransferState作为可注入的令牌可用。在客户端，只需使用 DI 注入此令牌并使用它，它将被延迟初始化。在服务器上，如果使用 renderApplication函数，它已经包含在内。否则，导入 ServerTransferStateModule模块以使 TransferState可用。

这里会使用 JSON.stringify/JSON.parse 对存储中的值进行序列化/反序列化。因此，仅布尔、数字、字符串、null 和非类对象能以无损的方式进行序列化和反序列化。
