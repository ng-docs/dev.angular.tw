定义 Component 装饰器的 encapsulation 选项的 CSS 样式封装策略。

参见 {@link Componentencapsulation encapsulation}。

仿真

通过向组件的宿主元素添加特定属性并将相同属性应用于所有通过 Componentstyles styles 或 {@link ComponentstyleUrls styleUrls} 提供的 CSS 选择器，仿真原生 Shadow DOM 封装行为。

这是默认选项。

无

不提供任何 CSS 样式封装，这意味着通过 Componentstyles styles 或 {@link ComponentstyleUrls styleUrls} 提供的所有样式都适用于应用的任何 HTML 元素，而不管其宿主组件。

ShadowDom

使用浏览器的原生 Shadow DOM API 来封装 CSS 样式，这意味着它会为组件的宿主元素创建一个 ShadowRoot，然后用该元素来封装所有组件的样式。
