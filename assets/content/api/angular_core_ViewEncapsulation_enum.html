ViewEncapsulation

列舉
程式碼

定義了 Component 裝飾器的 encapsulation 選項的 CSS 樣式封裝策略。

API

  
    enum ViewEncapsulation {}

已模擬

透過給元件的宿主元素新增一個特定的 Attribute，並將相同的 Attribute 應用於透過 styles 或 styleUrls 提供的所有 CSS 選擇器，來模擬原生的 Shadow DOM 封裝行為。

這是預設選項。

無

不提供任何型別的 CSS 樣式封裝，這意味著透過 styles 或 styleUrls 提供的所有樣式都適用於應用的任何 HTML 元素，而與其宿主元件無關。

ShadowDom

使用瀏覽器原生的 Shadow DOM API 來封裝 CSS 樣式，這意味著它會為元件的宿主元素建立一個 ShadowRoot，然後用它來封裝元件的所有樣式。

描述

定義了 Component 裝飾器的 encapsulation 選項的 CSS 樣式封裝策略。

請參閱 encapsulation。
