AnimationPlayer

interface
程式碼

提供對可複用動畫序列的程式設計式控制，此動畫序列使用 AnimationBuilder.build() 方法建立，此方法回傳一個 AnimationFactory，而 create() 方法會實例化並初始化此介面。
