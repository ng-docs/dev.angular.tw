@angular/core

SkipSelf

decorator
程式碼

要用在建構式函式引數上的引數裝飾器，它告訴依賴注入（DI）框架從父注入器開始解析依賴。解析過程會在注入器分層結構中向上進行，因此不會檢查本地注入器中是否有提供者。

依賴注入指南SelfOptional

API

  
    @SkipSelf ({})

用法說明

在以下範例中，當實例化子類別時可以解析依賴，但當實例化類本身時則不能。

          
class Dependency {}        @Injectable()        class NeedsDependency {          constructor(@SkipSelf() public dependency: Dependency) {}        }        const parent = Injector.create({providers: [{provide: Dependency, deps: []}]});        const child = Injector.create({          providers: [{provide: NeedsDependency, deps: [Dependency]}],          parent,        });        expect(child.get(NeedsDependency).dependency instanceof Dependency).toBe(true);        const inj = Injector.create({          providers: [{provide: NeedsDependency, deps: [[new Self(), Dependency]]}],        });        expect(() => inj.get(NeedsDependency)).toThrowError();
