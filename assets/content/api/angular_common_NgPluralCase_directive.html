@angular/common

NgPluralCase

directive
code

创建一个视图，当给定表达式根据CLDR规则匹配复数表达式时，它将被添加/移除到父 NgPlural中。

class NgPluralCase {
}
          
            <some-element [ngPlural]="value">  <ng-template ngPluralCase="=0">...</ng-template>  <ng-template ngPluralCase="other">...</ng-template></some-element>

有关更多详细信息和示例，请参见{@link NgPlural}。
