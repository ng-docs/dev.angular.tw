返回一组依赖注入提供者以在应用中启用动画。请参阅 动画指南 以了解更多关于 Angular 中动画的信息。

provideAnimations(
  
): Provider[];

当你希望在使用 bootstrapApplication 函数引导的应用中启用动画时，此函数很有用。在这种情况下，根本不需要导入 BrowserAnimationsModule NgModule，只需将此函数返回的提供者添加到 providers 列表中，如下所示。

          
            bootstrapApplication(RootComponent, {  providers: [    provideAnimations()  ]});
