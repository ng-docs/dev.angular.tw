API

  
    function provideAnimations(): Provider[];

provideAnimations

Provider[]

回傳一組依賴注入提供者，以便在應用中啟用動畫。請參閱動畫指南，以瞭解關於 Angular 動畫的更多訊息。

@returnsProvider[]

用法說明

當你想在透過 bootstrapApplication 函式引導的應用中啟用動畫時，此函式非常有用。在這種情況下，完全不需要匯入 BrowserAnimationsModule NgModule，只需將此函式回傳的提供者新增到 providers 清單中，如下所示。

          
bootstrapApplication(RootComponent, {  providers: [    provideAnimations()  ]});
