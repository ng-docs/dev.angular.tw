@angular/core

OnChanges

interface
程式碼

當指令的任何資料繫結屬性發生更改時，將呼叫此生命週期鉤子。 定義 ngOnChanges() 方法來處理這些更改。

DoCheckOnInit生命週期鉤子指南

API

  
    interface OnChanges {}

ngOnChanges

void

每當預設變更檢測器檢查完資料繫結屬性之後，如果檢測到至少一個屬性已更改，並且在檢查檢視和內容子項之前，就會立即呼叫此回呼方法。

@paramchangesSimpleChanges

已更改的屬性。

@returnsvoid

使用須知

以下程式碼片段展示了一個元件如何實現此介面，以便為輸入屬性定義一個 on-changes 處理程式。

          
@Component({        selector: 'my-cmp',        template: `...`,        standalone: false,      })      class MyComponent implements OnChanges {        @Input() prop: number = 0;        ngOnChanges(changes: SimpleChanges) {          // changes.prop contains the old and the new value...        }      }
