@angular/router

BaseRouteReuseStrategy

類別
程式碼

此基本路由複用策略僅在對應的路由器設定相同時才複用路由。這可以防止僅當路由引數、查詢引數或片段更改時元件被銷燬和重新建立（也就是說，現有元件被複用）。

描述

此基本路由複用策略僅在對應的路由器設定相同時才複用路由。這可以防止僅當路由引數、查詢引數或片段更改時元件被銷燬和重新建立（也就是說，現有元件被複用）。

此策略不儲存任何路由以供稍後複用。

Angular 預設使用此策略。

它可以用作自定義路由複用策略的基底類別，即你可以建立自己的類別來擴充套件 BaseRouteReuseStrategy 類別。
