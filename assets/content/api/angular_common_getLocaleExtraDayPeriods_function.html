getLocaleExtraDayPeriods

string[]

檢索特定於語言環境的日時段，這些時段大致指示了一天在不同語言中是如何劃分的。 例如，對於 en-US，時段為上午、中午、下午、傍晚和午夜。

僅當您載入了完整的語言環境資料後，此功能才可用。 請參閱『I18n 指南』。

@deprecated
