用于手动引导应用的钩子，而不是使用 @NgModule 注解中的 bootstrap 数组。此钩子仅在 bootstrap 数组为空或未提供时被调用。

当前应用的引用作为参数提供。

参见 《引导》。

下面的示例使用 ApplicationRef.bootstrap() 在页面上渲染 AppComponent。

          
            class AppModule implements DoBootstrap {  ngDoBootstrap(appRef: ApplicationRef) {    appRef.bootstrap(AppComponent); // Or some other component  }}
