描述

用於手動引導應用的鉤子，取代在 @NgModule 註解中使用 bootstrap 陣列的做法。本鉤子僅當 bootstrap 陣列為空或未提供時才會被呼叫。

當前應用的引用會作為引數傳入。

參見 『引導啟動』。

使用須知

下面的範例使用 ApplicationRef.bootstrap() 在頁面上渲染 AppComponent。

          
class AppModule implements DoBootstrap {  ngDoBootstrap(appRef: ApplicationRef) {    appRef.bootstrap(AppComponent); // Or some other component  }}
