StaticClassProvider

interface
code

配置 Injector 以返回某个令牌的 useClass 实例。

interface StaticClassProvider {
}

请注意，以下两个提供者并不相等：

多值示例

provide

any

一个注入令牌。通常是 Type或 InjectionToken的实例，但也可以是 any。

multi

boolean

如果为 true，则注入器返回实例数组。这对于允许多个提供者散布在多个文件中以向公共令牌提供配置信息很有用。

useClass

Type<any>

供 token 实例化的可选类。默认情况下，会把 provide 类实例化。

deps

any[]

由注入器解析的 token 列表。将这个值列表用作 useClass 构造函数的参数。
