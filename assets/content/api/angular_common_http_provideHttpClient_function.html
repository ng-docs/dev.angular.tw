provideHttpClient

EnvironmentProviders

設定 Angular 的 HttpClient 服務以使其可用於注入。

預設情況下，將設定 HttpClient 以進行注入，並使用其預設選項來保護出站請求的 XSRF。可以透過將屬性函式傳遞給 provideHttpClient 來提供其他設定選項。例如，可以使用 withInterceptors(...) 屬性新增 HTTP 攔截器。

強烈建議為使用伺服器端渲染的應用啟用 fetch，以獲得更好的效能和相容性。要啟用 fetch，請將 withFetch() 屬性新增到應用根部的 provideHttpClient() 呼叫中：

          
provideHttpClient(withFetch());
