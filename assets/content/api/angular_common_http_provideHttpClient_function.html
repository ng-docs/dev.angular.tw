配置 Angular 的 HttpClient 服务以供注入。

默认情况下， HttpClient 将使用其默认的 XSRF 保护选项进行注入。可以通过将特性函数传递给 provideHttpClient 来提供其他配置选项。例如，可以使用 withInterceptors(...) 特性添加 HTTP 拦截器。

强烈建议为使用服务器端渲染的应用启用 fetch 以获得更好的性能和兼容性。要启用 fetch，请在应用的根部将 withFetch() 特性添加到 provideHttpClient() 调用中：

          
            provideHttpClient(withFetch());
