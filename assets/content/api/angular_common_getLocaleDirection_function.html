getLocaleDirection

"ltr" | "rtl"

檢索指定語言環境的書寫方向

@deprecated

Angular 建議依賴 Intl API 進行 i18n。 對於日期和數字，讓 Intl.DateTimeFormat() 和 Intl.NumberFormat() 確定書寫方向。 Intl 的替代方案 getTextInfo。 僅有部分支援（Chromium M99 和 Safari 17）。 諸如 rtl-detect 之類的第三方替代方案可以解決此問題。

@paramlocalestring

用於指定語言環境格式規則的語言環境程式碼。

@returns"ltr" | "rtl"
