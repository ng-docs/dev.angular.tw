检索指定语言环境的书写方向

@deprecated

Angular 建议依赖 Intl API 进行国际化。对于日期和数字，让 Intl.DateTimeFormat() 和 Intl.NumberFormat() 决定书写方向。 Intl 的替代方案 getTextInfo 只有部分支持（Chromium M99 和 Safari 17）。第三方替代品如 rtl-detect 可以解决这个问题。

@paramlocalestring

用于要使用的语言环境格式规则的语言环境代码。

@returns"ltr" | "rtl"
