element

ElementRef<any>

用於指定此容器在所屬檢視中的位置的錨點元素。 每個檢視容器只能有一個錨點元素，並且每個錨點元素只能有一個檢視容器。

附著到此容器的檢視的根元素，在渲染後的檢視中會變成錨點元素的兄弟節點。

要訪問元素的 ViewContainerRef，方法是在元素上放置一個注入了 ViewContainerRef 的 Directive，或者使用 ViewChild 查詢。

描述

表示一個容器，其中一個或多個檢視可以附著到一個元件。

可以包含宿主檢視（透過使用 createComponent() 方法實例化元件來建立）和嵌入式檢視（透過使用 createEmbeddedView() 方法實例化 TemplateRef 來建立）。

檢視容器實例可以包含其他檢視容器，從而建立檢視分層結構。

用法說明

下面的範例示範了如何使用 createComponent 函式來動態建立 ComponentRef 的實例並將其附著到 ApplicationRef，以便將其包含在變更檢測週期中。

注意：該範例使用獨立元件，但該函式也可用於非獨立元件（在 NgModule 中宣告）。
