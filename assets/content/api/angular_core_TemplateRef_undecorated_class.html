elementRef

ElementRef<any>

此內嵌檢視在父檢視中的錨點元素。

由此 TemplateRef 建立的內嵌檢視的資料繫結和注入上下文繼承自此位置的上下文。

通常，新的內嵌檢視會附著到此位置的檢視容器，但在高階用例中，檢視可以附著到不同的容器，同時保留來自原始位置的資料繫結和注入上下文。

createEmbeddedView

EmbeddedViewRef<C>

基於此樣板實例化一個未附著的內嵌檢視。

@paramcontextC

內嵌檢視的資料繫結上下文，如 <ng-template> 用法中所宣告的那樣。

@paraminjectorInjector | undefined

要在內嵌檢視中使用的注入器。

@returnsEmbeddedViewRef<C>

描述

表示一個內嵌樣板，可用於實例化內嵌檢視。要基於樣板實例化內嵌檢視，請使用 ViewContainerRef 方法 createEmbeddedView()。

透過將指令放在 <ng-template> 元素（或帶有 * 字首的指令）上來訪問 TemplateRef 實例。內嵌檢視的 TemplateRef 會被注入到指令的建構式函式中，使用的令牌是 TemplateRef 令牌。

你也可以使用 Query 來查詢與元件或指令關聯的 TemplateRef。
