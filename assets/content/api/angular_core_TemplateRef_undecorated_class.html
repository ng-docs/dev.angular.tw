abstract class TemplateRef {
}

表示一个嵌入模板，可以用于实例化嵌入视图。要基于模板实例化嵌入视图，请使用 ViewContainerRef 方法 createEmbeddedView()。

通过在 <ng-template> 元素（或带有 * 前缀的指令）上放置指令来访问 TemplateRef 实例。嵌入视图的 TemplateRef 被注入到指令的构造函数中，使用 TemplateRef 令牌。

你也可以使用 Query 查找与组件或指令关联的 TemplateRef。

elementRef

ElementRef<any>

内嵌视图在其所属视图中的位置。

从此 TemplateRef 创建的嵌入视图的数据绑定和 注入上下文 继承自此位置的上下文。

通常，新的内嵌视图会被附加到当前位置的视图容器中，但是在一些高级用例中，该视图可能被附加到别的容器中， 同时还保留原位置的数据绑定和依赖注入上下文。
