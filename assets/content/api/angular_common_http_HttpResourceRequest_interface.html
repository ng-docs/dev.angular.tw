@angular/common/http

HttpResourceRequest

interface
實驗性的
程式碼

將要傳送到後端的 httpResource 請求的結構。

API

  
    interface HttpResourceRequest {}

url

string

請求的 URL。

此 URL 不應包含查詢引數。而是透過 params 欄位指定查詢引數。

method

string | undefined

請求的 HTTP 方法，如果未指定，則預設為 GET。

body

unknown

要隨請求傳送的正文（如果有）。

如果使用者未指定 Content-Type 標頭，Angular 將嘗試根據 body 的型別設定一個。

引數

HttpParams | Record<string, string | number | boolean | readonly (string | number | boolean)[]> | undefined

將附加到請求 URL 的查詢引數字典。

headers

HttpHeaders | Record<string, string | readonly string[]> | undefined

要包含在傳出請求中的標頭字典。

context

HttpContext | undefined

儲存在鍵值對字典中的請求上下文。

reportProgress

boolean | undefined

如果為 true，則將為請求啟用進度事件，並透過 HttpResource.progress Signal 傳遞。

withCredentials

boolean | undefined

指定是否應在傳出請求上設定 withCredentials 標誌。

此標誌使瀏覽器隨請求一起傳送 Cookie 和其他身份驗證訊息。

transferCache

boolean | { includeHeaders?: string[] | undefined; } | undefined

為此請求設定服務器端渲染傳輸快取。

有關更多訊息，請參閱有關傳輸快取的文件。
