alias

string

输入的可选公共名称。默认情况下，使用类字段名称。

transform

(v: TransformT) => T

每当绑定新值时运行的可选转换。可用于在更新输入之前转换输入值。

转换函数可以扩展输入的类型。例如，考虑一个 disabled 输入。实际上，作为组件作者，你只想处理布尔值，但用户可能希望绑定字符串，如果他们只是使用属性形式通过 <my-dir input> 绑定到输入。转换函数可以处理这些字符串值并将其转换为 boolean。参见： booleanAttribute。
