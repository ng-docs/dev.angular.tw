alias

string | undefined

輸入的可選公共名稱。預設情況下，使用類別欄位名稱。

transform

((v: TransformT) => T) | undefined

可選的轉換，每當繫結新值時運行。可用於在更新輸入之前轉換輸入值。

轉換函式可以拓寬輸入的型別。例如，考慮 disabled 的輸入。實際上，作為元件作者，你只想處理布林值，但如果使用者僅使用屬性形式透過 <my-dir input> 繫結到輸入，他們可能希望繫結字串。然後，轉換可以處理此類別字串值並將其轉換為 boolean。請參閱：booleanAttribute。

debugName

string | undefined

輸入訊號的除錯名稱。在 Angular DevTools 中用於標識訊號。
