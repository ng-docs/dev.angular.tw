将现有的 FormGroup或 FormRecord绑定到DOM元素。

此指令接受现有的 FormGroup实例。然后，它将使用此 FormGroup实例将任何子 FormControl、 FormGroup/ FormRecord和 FormArray实例与子 FormControlName、 FormGroupName和 FormArrayName指令匹配。

注册表单组

以下示例注册了一个带有名字和姓氏控件的 FormGroup，并在点击按钮时监听 ngSubmit事件。

@parampathstring | (string | number)[]

控件名称列表，用于指定如何从当前控件移至要查询错误的控件。

@returnsboolean
Usage notes

例如，对于以下 FormGroup：

          
            form = new FormGroup({  address: new FormGroup({ street: new FormControl() })});

从根表单到 'street' 控件的路径是 'address' -> 'street'。

调用此方法有两种形式：

  1. 控件名称的字符串数组，如 ['address', 'street']
  2. 以一个字符串表示的句号分隔的控件名称列表，如 'address.street'

如果没有给出路径，则此方法检查当前控件上的错误。
