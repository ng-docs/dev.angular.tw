从所有 NgModule 和独立组件（包括可传递导入的组件）收集提供程序。

importProvidersFrom(
  ...sources: ImportProvidersSource[]
): EnvironmentProviders;

从所有 NgModule 和独立组件（包括可传递导入的组件）收集提供程序。

通过 importProvidersFrom 提取的提供者只能在应用注入器或其他环境注入器（例如路由注入器）中使用。它们不应在组件提供者中使用。

关于独立组件的更多信息，请参见 本指南。

importProvidersFrom 调用的结果可以在 bootstrapApplication 调用中使用：

          
            await bootstrapApplication(RootComponent, {  providers: [    importProvidersFrom(NgModuleOne, NgModuleTwo)  ]});

当使用独立组件时，你也可以在路由的 providers 字段中使用 importProvidersFrom 的结果：

          
            export const ROUTES: Route[] = [  {    path: 'foo',    providers: [      importProvidersFrom(NgModuleOne, NgModuleTwo)    ],    component: YourStandaloneComponent  }];
