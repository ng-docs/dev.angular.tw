MinLengthValidator

directive
code

一种指令，添加最小长度验证到标记了 minlength 属性的控件上。此指令由 NG_VALIDATORS 多提供者列表提供。

表单验证
class MinLengthValidator {
}

添加最小长度验证器

下面的例子演示了如何为带有 ngModel 绑定的输入框添加最小长度验证器。

          
            <input name="firstName" ngModel minlength="4">
