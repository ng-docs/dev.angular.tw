禁用初始导航。

如果由于某些复杂的初始化逻辑而有理由对路由器何时开始其初始导航有更多控制权，可以使用。

如何禁用初始导航的基本示例：

          
            const appRoutes: Routes = [];bootstrapApplication(AppComponent,  {    providers: [      provideRouter(appRoutes, withDisabledInitialNavigation())    ]  });
