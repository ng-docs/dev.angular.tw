withDisabledInitialNavigation

DisabledInitialNavigationFeature

停用初始導向。

如果出於某些複雜的初始化邏輯，需要更好地控制路由器何時啟動其初始導向，請使用此選項。

@returnsDisabledInitialNavigationFeature

說明

停用初始導向。

如果出於某些複雜的初始化邏輯，需要更好地控制路由器何時啟動其初始導向，請使用此選項。

用法說明

如何停用初始導向的基本範例：

          
const appRoutes: Routes = [];bootstrapApplication(AppComponent,  {    providers: [      provideRouter(appRoutes, withDisabledInitialNavigation())    ]  });
