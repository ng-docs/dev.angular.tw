formatDate

string

基于区域规则格式化日期。

@paramvaluestring | number | Date

要格式化的日期，可以是一个 Date 对象、一个数字（自 UTC 纪元以来的毫秒数）或一个 ISO 日期时间字符串。

@paramformatstring

要包含的日期-时间组件。参见 DatePipe 了解详情。

@paramlocalestring

用于要使用的语言环境格式规则的语言环境代码。

@paramtimezonestring

时区。一个从 GMT 起的时区偏移（例如 '+0430'），或标准的 UTC/GMT 或美国大陆时区缩写。如果未指定，则使用宿主系统设置。

@returnsstring
