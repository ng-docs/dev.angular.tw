formatDate

string

根據語言環境規則格式化日期。

@paramvaluestring | number | Date

要格式化的日期，可以是 Date，也可以是數字（自 UTC epoch 以來的毫秒數）， 或者 ISO 日期-時間字串。

@paramformatstring

要包含的日期-時間元件。 詳情參閱 DatePipe。

@paramlocalestring

用於語言環境格式規則的語言環境程式碼。

@paramtimezonestring | undefined

時區。 與 GMT 的時區偏移量（例如 '+0430'）。 如果未指定，則使用宿主系統設定。

@returnsstring
