描述

當應用的狀態發生變化時，Angular 保持更新的條目的不可修改清單。

ViewChildren、ContentChildren 和 QueryList 提供的物件型別。

實現了可迭代介面，因此它既可以用在 ES6 javascript 的 for (var i of items) 迴圈中，也可以用在 Angular 範本的 *ngFor="let i of myList" 中。

透過訂閱變更 Observable 可以觀察到這些變更。

注意：將來此類別會實現 Observable 介面。
