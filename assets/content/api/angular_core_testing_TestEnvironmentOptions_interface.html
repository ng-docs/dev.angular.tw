teardown

ModuleTeardownOptions

配置 TestBed 中测试模块的拆卸行为。

errorOnUnknownElements

boolean

当组件的模板中存在未知元素时是否应抛出错误。默认为 false，仅记录错误。如果设置为 true ，则抛出错误。

errorOnUnknownProperties

boolean

当组件的模板中存在未知属性时是否应抛出错误。默认为 false，仅记录错误。如果设置为 true ，则抛出错误。
