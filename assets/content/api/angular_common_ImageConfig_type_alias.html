@angular/common

ImageConfig

型別別名
程式碼

用於影象相關選項的設定物件。包含：

  • breakpoints：用於為回應式影象生成 srcset 的整數斷點陣列。
  • disableImageSizeWarning：一個布林值。設定為 true 將停用關於超大影象的控制台警告。
  • disableImageLazyLoadWarning：一個布林值。設定為 true 將停用關於設定了 loading="lazy" 的 LCP 影象的控制台警告。 在 NgOptimizedImage 指南中瞭解有關回應式影象設定的更多訊息。 在 相關的錯誤頁面中瞭解有關影象警告選項的更多訊息。

API

  
    type ImageConfig = {  breakpoints?: number[];  placeholderResolution?: number;  disableImageSizeWarning?: boolean;  disableImageLazyLoadWarning?: boolean;}
