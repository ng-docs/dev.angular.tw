@angular/common

ImageConfig

Type Alias
code

图像相关选项的配置对象。包含：

  • breakpoints：用于生成响应式图像 srcsets 的整数断点数组。
  • disableImageSizeWarning：一个布尔值。将其设置为 true 将禁用关于图像过大问题的控制台警告。
  • disableImageLazyLoadWarning：一个布尔值。将其设置为 true 将禁用关于配置为 loading="lazy" 的 LCP 图像的控制台警告。了解更多响应式图像配置，请参阅 NgOptimizedImage 指南。了解更多图像警告选项，请参阅 相关错误页面。

type ImageConfig = {
  breakpoints?: number[];
  placeholderResolution?: number;
  disableImageSizeWarning?: boolean;
  disableImageLazyLoadWarning?: boolean;
}
