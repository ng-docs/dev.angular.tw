API

  
    const REQUEST: InjectionToken<Request | null>;

描述

表示當前 HTTP 請求物件的注入令牌。

在處理伺服器端渲染 (SSR) 時，使用此令牌來訪問當前請求。

用法說明

在以下場景中，此令牌可能為 null：

  • 在建立過程中。
  • 當應用在瀏覽器中渲染時（用戶端渲染）。
  • 當執行靜態站點生成 (SSG) 時。
  • 在開發期間的路由提取過程中（在請求時）。
