@angular/forms

FormControlOptions

interface
程式碼

提供給 FormControl 的選項的介面。

API

  
    interface FormControlOptions extends AbstractControlOptions {}

nonNullable

boolean | undefined

是否將用於構造 FormControl 的初始值也用作其預設值。如果此選項為 false 或未提供，則 FormControl 的預設值為 null。 當 FormControl 在沒有顯式值的情況下重置時，其值將恢復為其預設值。

initialValueIsDefault

boolean | undefined
@deprecated

請改用 nonNullable。

validators

ValidatorFn | ValidatorFn[] | null | undefined

應用於控制元件的驗證器清單。

asyncValidators

AsyncValidatorFn | AsyncValidatorFn[] | null | undefined

應用於控制元件的非同步驗證器清單。

updateOn

"change" | "blur" | "submit" | undefined

控制元件更新時所依據的事件名稱。

描述

提供給 FormControl 的選項的介面。

此介面擴充套件了 AbstractControlOptions 中的所有選項，以及 FormControl 特有的一些選項。
