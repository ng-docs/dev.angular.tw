interface FormControlOptions {
}

提供给 FormControl的选项的接口。

此接口扩展了所有来自 AbstractControlOptions 的选项，以及一些特定于 FormControl的选项。

nonNullable

boolean

是否使用用来构造 FormControl 的初始值作为其默认值。如果此选项为 false 或未提供，FormControl 的默认值为 null。当 FormControl 在没有显式值的情况下重置时，其值会恢复为默认值。

initialValueIsDefault

boolean

validators

ValidatorFn | ValidatorFn[]

应用于该控件的验证器列表。

asyncValidators

AsyncValidatorFn | AsyncValidatorFn[]

应用于该控件的异步验证器列表。

updateOn

"change" | "blur" | "submit"

会导致更新控件的事件名称。
