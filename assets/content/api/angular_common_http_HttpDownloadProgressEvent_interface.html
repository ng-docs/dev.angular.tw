下载进度事件。

interface HttpDownloadProgressEvent {
}

type

HttpEventType.DownloadProgress

partialText

string

到目前为止已经下载的那部分响应体。

只有当 responseType 是 text 时才会出现。

loaded

number

已经上传或下载的字节数。

total

number

要上传或下载的总字节数。它可能是无法计算出来的，因此也就可能不存在，取决于是请求还是响应。
