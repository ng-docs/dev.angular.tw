UrlSerializer 的默认实现。

范例网址：

          
            /inbox/33(popup:compose)/inbox/33;open=true/messages/44

DefaultUrlSerializer 使用圆括号序列化辅助段（比如，popup:compose），使用冒号语法指定出口，并使用';parameter=value' 语法（比如 open=true）来指定路由的特有参数。
