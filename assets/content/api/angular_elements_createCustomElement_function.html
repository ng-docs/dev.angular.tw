createCustomElement

NgElementConstructor<P>

基於 Angular 元件建立一個自定義元素類。

建立一個類，它封裝了所提供的元件的功能，並使用設定訊息為該類別提供更多上下文。 獲取元件工廠的輸入和輸出，以將它們轉換為正確的自定義元素 API，並新增鉤子來處理輸入更改。

設定的注入器是類別上設定的初始注入器，預設情況下用於每個建立的實例。此行為可以透過靜態屬性來重寫，以影響所有新創建的實例，也可以作為一次性建立的建構式函式引數來重寫。
