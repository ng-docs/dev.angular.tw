API

  
    type ImportedNgModuleProviders = EnvironmentProviders

說明

透過 importProvidersFrom 函式從 NgModule 匯入的提供者。

這些提供者旨在用於應用注入器（或其他環境注入器），不應在元件注入器中使用。

此類型無法直接實現。它從 importProvidersFrom 函式回傳，用於防止提取的 NgModule 提供者在錯誤的上下文中使用。
