Angular 平台是网页上 Angular 的入口点。每个页面只有一个平台。服务（如反射）在其作用域内绑定，这些服务是每个在页面上运行的 Angular 应用共有的。当使用类似 PlatformBrowser 的平台工厂创建平台时，页面的平台会被隐式地初始化，或者通过调用 createPlatform() 函数显式地初始化。
