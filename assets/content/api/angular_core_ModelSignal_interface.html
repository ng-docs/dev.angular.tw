ModelSignal 表示指令/组件模型字段的一个特殊 Signal。

模型信号是一个可写的信号，可以作为输出公开。 每当其值更新时，它会向输出发出信号。

@paramvalueT
@returnsvoid

void

根据信号的当前值更新信号值，并通知任何依赖者。

@paramupdateFn(value: T) => T

asReadonly

Signal<T>

返回此信号的只读版本。只读信号可以访问以读取其值，但不能使用 set 或 update 方法更改。只读信号 没有任何内置机制来防止其值的深层变更。

@returnsSignal<T>
