withConfig

ModuleWithProviders<ReactiveFormsModule>

提供了一些选项，供配置响应式表单模块。

@paramopts{ warnOnNgModelWithFormControl?: "always" | "never" | "once"; callSetDisabledState?: SetDisabledStateOption; }

一个配置选项对象

  • warnOnNgModelWithFormControl 配置何时在使用响应式表单指令的情况下发出 ngModel 绑定的警告。
  • callSetDisabledState 配置是 always 调用 setDisabledState，这更正确，还是仅调用它 whenDisabled，这是旧版行为。
