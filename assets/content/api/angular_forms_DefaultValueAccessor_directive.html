用法說明

使用預設值訪問器

以下範例展示了如何使用啟用預設值訪問器的輸入元素（在本例中，是一個文字欄位）。

          
const firstNameControl = new FormControl();

        
          
<input type="text" [formControl]="firstNameControl">

預設情況下，此值訪問器用於 <input type="text"> 和 <textarea> 元素，但你也可以將其用於具有類似行為且不需要特殊處理的自定義元件。為了將預設值訪問器附著到自定義元素，請新增 ngDefaultControl 屬性，如下所示。

          
<custom-input-component ngDefaultControl [(ngModel)]="value"></custom-input-component>
