class DefaultValueAccessor {

用于在输入元素上写入值并监听更改的默认 ControlValueAccessor。该访问器被 FormControlDirective、 FormControlName 和 NgModel 指令使用。

{@searchKeywords ngDefaultControl}

使用默认值访问器

以下示例演示了如何使用输入元素激活默认值访问器（在这种情况下为文本字段）。

          
            const firstNameControl = new FormControl();
          
        
          
            <input type="text" [formControl]="firstNameControl">

此值访问器默认用于 <input type="text">和 <textarea>元素，但也可用于具有类似行为且不需要特殊处理的自定义组件。要将默认值访问器附加到自定义元素，请添加如下所示的 ngDefaultControl属性。

          
            <custom-input-component ngDefaultControl [(ngModel)]="value"></custom-input-component>
