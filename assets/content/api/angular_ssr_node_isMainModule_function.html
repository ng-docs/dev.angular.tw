isMainModule

boolean

確定提供的 URL 是否表示主入口點模組。

此函式檢查提供的 URL 是否對應於直接執行的主 ESM 模組。 它對於有條件地執行僅應在模組是入口點時運行的程式碼很有用，例如啟動伺服器或初始化應用。

它執行兩個關鍵檢查：

  1. 驗證 URL 是否以 'file:' 開頭，確保它是一個本地檔案。
  2. 將 URL 的已求解檔案路徑與第一個指令行引數 (process.argv[1]) 進行比較， 該引數指向正在執行的檔案。
@paramurlstring

要檢查的模組的 URL。這通常應該是 import.meta.url。
