@angular/core

HOST_TAG_NAME

constant
code

一个可以用来注入宿主节点标签名的令牌。

const HOST_TAG_NAME: InjectionToken<string>;

注入一个已知存在的标签名

          
            @Directive()class MyDir {  tagName: string = inject(HOST_TAG_NAME);}

可选地注入一个标签名

          
            @Directive()class MyDir {  tagName: string | null = inject(HOST_TAG_NAME, {optional: true});}
