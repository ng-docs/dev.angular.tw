API

  
    type UrlMatcher = (  segments: UrlSegment[],  group: UrlSegmentGroup,  route: Route,) => UrlMatchResult | null

描述

一個用於將路由與 URL 進行對應的函式。當 path 和 pathMatch 的組合 不足以表達時，請為 Route.matcher 實現自定義 URL 對應器。不能與 path 和 pathMatch 一起使用。

該函式接受以下引數並回傳一個 UrlMatchResult 物件。

  • segments：URL 段的陣列。
  • group：一個段組。
  • route：要對應的路由。

以下範例實現對應 HTML 檔案。

          
export function htmlFiles(url: UrlSegment[]) {  return url.length === 1 && url[0].path.endsWith('.html') ? ({consumed: url}) : null;}export const routes = [{ matcher: htmlFiles, component: AnyComponent }];
