描述

ControlValueAccessor 用於寫入多選控制元件值和監聽多選控制元件的更改。值訪問器被 FormControlDirective、FormControlName 和 NgModel 指令使用。

匯出自

  • ReactiveFormsModule
  • FormsModule

用法說明

使用多選控制元件

以下範例展示了如何將多選控制元件與回應式表單一起使用。

          
const countryControl = new FormControl();

        
          
<select multiple name="countries" [formControl]="countryControl">  <option *ngFor="let country of countries" [ngValue]="country">    {{ country.name }}  </option></select>

自定義選項選擇

要自定義預設選項比較演算法，<select> 支援 compareWith 輸入。 請參閱 SelectControlValueAccessor 以瞭解用法。
