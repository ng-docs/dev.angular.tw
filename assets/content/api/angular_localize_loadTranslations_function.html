loadTranslations

void

如果執行執行時翻譯，則載入翻譯以供 $localize 使用。

如果 $localize 標記的字串不會在編譯時被替換，則可以載入一組翻譯，這些翻譯將在執行時在瀏覽器中應用於 $localize 標記的字串。

如果新的翻譯與之前的翻譯具有相同的 MessageId，則載入新的翻譯將覆蓋之前的翻譯。

請注意，$localize 訊息僅在首次遇到標記的字串時處理一次，並且不提供在不重新整理瀏覽器的情況下動態更改語言的功能。在應用生命週期後期載入新的翻譯不會更改已翻譯訊息的翻譯文字。

訊息 ID 和翻譯的格式與提取訊息時渲染到『簡單 JSON』翻譯檔案中的格式相同。特別是，訊息中的佔位符使用 {$PLACEHOLDER_NAME} 語法渲染。例如，來自以下樣板的訊息：

          
<div i18n>pre<span>inner-pre<b>bold</b>inner-post</span>post</div>

在 translations 對映表中將具有以下形式：

          
{  "2932901491976224757":     "pre{$START_TAG_SPAN}inner-pre{$START_BOLD_TEXT}bold{$CLOSE_BOLD_TEXT}inner-post{$CLOSE_TAG_SPAN}post"}
@paramtranslationsRecord<string, string>

從訊息 ID 到翻譯後訊息的對映表。

這些訊息會被處理，並根據其 MessageId 新增到查詢表中。
