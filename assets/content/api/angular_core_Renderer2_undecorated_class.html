Renderer2

程式碼

擴充套件此基底類別以實現自定義渲染。預設情況下，Angular 會將樣板渲染到 DOM 中。你可以使用自定義渲染來攔截渲染呼叫，或者渲染到 DOM 以外的其他內容。

描述

擴充套件此基底類別以實現自定義渲染。預設情況下，Angular 會將樣板渲染到 DOM 中。你可以使用自定義渲染來攔截渲染呼叫，或者渲染到 DOM 以外的其他內容。

使用 RendererFactory2 建立你的自定義渲染器。

使用自定義渲染器可以繞過 Angular 的樣板，並進行無法以宣告方式表達的自定義 UI 更改。例如，如果你需要設定一個名字不是靜態已知的屬性（Property）或屬性（Attribute），請使用 setProperty() 或 setAttribute() 方法。
