@angular/common/http

httpResource

初始化器 API
實驗性的
程式碼

httpResource 發起一個回應式 HTTP 請求，並將請求狀態和回應值公開為 WritableResource。預設情況下，它假定後端將回傳 JSON 資料。要發起一個需要不同型別資料的請求，你可以使用 httpResource 的子建構式函式， 例如 httpResource.text。

API

  
    
  
  
function httpResource<TResult = unknown>(url: string | (() => string | undefined), options: HttpResourceOptions<TResult, unknown> & { defaultValue: NoInfer<TResult>; }): HttpResourceRef<TResult>;

建立一個 Resource，它透過向給定 URL 傳送 HTTP GET 請求來獲取資料。

如果為 URL 傳入一個回應式函式，則當 URL 透過訊號發生變化時，該資源將更新。

使用 HttpClient 發起請求，並支援攔截器、測試以及 HttpClient API 的其他屬性。預設情況下，資料會被解析為 JSON - 使用 httpResource 的子函式（例如 httpResource.text()）以不同的方式解析回應。
