getOwningComponent(
  elementOrDir: {} | Element

检索其视图中包含此 DOM 元素的组件实例。

例如，如果 <child-comp> 在 <app-comp> 的模板中使用（即 <app-comp> 的一个 ViewChild），调用 getOwningComponent 方法获取 <child-comp> 上的组件将返回 <app-comp>。
