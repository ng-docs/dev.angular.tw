謹慎使用

當需要直接訪問 DOM 時，請將此 API 作為最後的手段使用。請改用 Angular 提供的樣板和資料繫結。或者，你可以看看 Renderer2，它提供了一個可以安全使用的 API。

描述

View 內部對原生元素的一個套件裝器。

ElementRef 由渲染器特定的元素提供支援。在瀏覽器中，這通常是一個 DOM 元素。
