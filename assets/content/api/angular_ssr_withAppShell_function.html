withAppShell

ServerRoutesFeature<ServerRoutesFeatureKind>

使用提供的元件設定 app shell 路由。

App shell 充當應用的主要入口點，通常用於啟用應用外殼的伺服器端渲染 (SSR)。它處理與任何特定伺服器路由都不對應的請求，提供了一種回退機制，並提高了導向期間的感知效能。

這種設定在利用漸進式 Web 應用 (PWA) 模式（例如 Service Worker）的應用中尤其有用，以提供無縫的使用者體驗。
