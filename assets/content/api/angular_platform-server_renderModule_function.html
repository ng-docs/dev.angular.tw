renderModule

Promise<string>

使用提供的 NgModule 引导应用程序，并将页面内容序列化为字符串。

@parammoduleTypeType<T>

对应该用于引导的 NgModule 的引用。

@paramoptions{ document?: string | Document; url?: string; extraProviders?: StaticProvider[]; }

渲染操作的附加配置：

  • document - 要渲染的页面文档，可以是 HTML 字符串或对 document 实例的引用。
  • url - 当前呈现请求的 URL。
  • extraProviders - 当前渲染请求的平台级提供程序集。
@returnsPromise<string>
