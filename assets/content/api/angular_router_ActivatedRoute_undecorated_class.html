class ActivatedRoute {
}

提供对与在某个出口中加载的组件关联的路由信息的访问。用于遍历 RouterState 树并从节点中提取信息。

以下示例演示如何使用当前激活的路由中的信息来构造组件。

注意：此类中的可观察者仅在当前值和先前值基于浅比较不同的时候发出。例如，更改已求解的 data 中深层嵌套的属性不会导致 ActivatedRoute.data Observable 发出新值。

title

url

Observable<UrlSegment[]>

此路由范围内的矩阵参数的可观察值。

所有路由共享的查询参数的可观察值。

fragment

Observable<string>

所有路由共享的 URL 片段的可观察者。

Observable<Data>

此路由的静态和已解析数据的可观察者。

Route

用于匹配本路由的配置项。

ActivatedRoute

children

ActivatedRoute[]

在路由器状态树中，当前路由的所有子路由。

pathFromRoot

ActivatedRoute[]

在路由器状态树中从根节点开始到当前路由的完整路径。

paramMap

Observable<ParamMap>

一个 Observable，其中包含特定于路由的必要和可选参数的映射。该映射支持从同一参数中检索单个和多个值。

queryParamMap

Observable<ParamMap>

一个 Observable，其中包含可用于所有路由的查询参数的映射。该映射支持从查询参数中检索单个和多个值。
