ActivatedRoute

類別
程式碼

提供對與 outlet 中載入的元件關聯的路由訊息的訪問。 用於遍歷 RouterState 樹並從節點中提取訊息。

描述

提供對與 outlet 中載入的元件關聯的路由訊息的訪問。 用於遍歷 RouterState 樹並從節點中提取訊息。

以下範例展示了如何使用來自當前啟用路由的訊息來建立元件。

注意：此class中的可觀察物件僅在當前值和先前值基於淺相等性而不同時發出。例如，更改已求解的 data 中的深層巢狀屬性不會導致 ActivatedRoute.data Observable 發出新值。
