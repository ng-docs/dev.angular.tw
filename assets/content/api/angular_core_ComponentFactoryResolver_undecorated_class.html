一个简单的注册表，将 Components 映射到生成的 ComponentFactory 类，这些类可用于创建组件实例。用于获取给定组件类型的工厂，然后使用工厂的 create() 方法创建该类型的组件。

注意：从 v13 开始，通过ViewContainerRef.createComponent创建动态组件不需要解析组件工厂：组件类可以直接使用。
