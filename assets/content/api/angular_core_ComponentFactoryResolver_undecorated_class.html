ComponentFactoryResolver

程式碼

一個簡單的登錄檔，用於將 Components 對映到生成的 ComponentFactory 類別，這些類別可用於建立元件的實例。 用於獲取給定元件類別型的工廠，然後使用該工廠的 create() 方法來建立該類別型的元件。

ComponentFactoryResolver

描述

一個簡單的登錄檔，用於將 Components 對映到生成的 ComponentFactory 類別，這些類別可用於建立元件的實例。 用於獲取給定元件類別型的工廠，然後使用該工廠的 create() 方法來建立該類別型的元件。

注意：從 v13 版本開始，透過 ViewContainerRef.createComponent 進行動態元件建立，不 需要求解元件工廠：可以直接使用元件類別。
