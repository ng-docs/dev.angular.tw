說明

Testability 服務提供了可以從瀏覽器訪問的測試鉤子。

使用 NgModule 引導的 Angular 應用（透過 @NgModule.bootstrap 欄位）也將預設實例化 Testability（在開發和生產模式下）。

對於使用 bootstrapApplication 函式引導的應用，預設情況下不包含 Testability。你可以透過使用 provideProtractorTestingSupport() 函式獲取必要的提供者清單並將它們新增到 options.providers 陣列中，將其包含到你的應用中。例如：

          
import {provideProtractorTestingSupport} from '@angular/platform-browser';await bootstrapApplication(RootComponent, providers: [provideProtractorTestingSupport()]);
