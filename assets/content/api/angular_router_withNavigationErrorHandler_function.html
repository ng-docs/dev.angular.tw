withNavigationErrorHandler(
  handler: (error: NavigationError) => unknown
): NavigationErrorHandlerFeature;

提供在导航错误发生时调用的函数。

此函数在应用的 注入上下文 中运行，因此可以使用 inject 函数。

此函数可以返回一个 RedirectCommand 将错误转换为重定向，类似于从守卫返回 UrlTree 或 RedirectCommand。这也会阻止 Router 发出 NavigationError；它会改为发出带有代码 NavigationCancellationCode.Redirect 的 NavigationCancel。返回除了 RedirectCommand 以外的值将被忽略，不会改变 Router 处理错误的行为。

如何使用错误处理程序选项的基本示例：

          
            const appRoutes: Routes = [];bootstrapApplication(AppComponent,  {    providers: [      provideRouter(appRoutes, withNavigationErrorHandler((e: NavigationError) =>inject(MyErrorTracker).trackError(e)))    ]  });
