@angular/router

ResolveData

型別別名
程式碼

表示與特定路由關聯的已求解資料。

Route#resolve

API

  
    type ResolveData = {  [key: string | symbol]: ResolveFn<unknown> | DeprecatedGuard;}

說明

表示與特定路由關聯的已求解資料。

回傳 RedirectCommand 會指示路由器取消當前導向並重定向到 RedirectCommand 中提供的位置。請注意，當解析器執行時，沒有排序保證。如果多個解析器都回傳 RedirectCommand，則只會使用回傳的第一個。
