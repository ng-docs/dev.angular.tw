class HttpErrorResponse {
}

一个用于表示错误或失败的响应对象，或者来自执行请求时发生的错误给出的失败的 HTTP 状态码，或者来自在解析响应对象期间发生的其它错误。

在 Observable响应流中返回的任何错误都会被包装在 HttpErrorResponse中，以提供该错误发生时 HTTP 层状态的额外上下文信息。error 属性将包含一个被包装的 Error 对象或从服务器返回的错误响应。

name

"HttpErrorResponse"

message

string

error

any

ok

false

错误永远不可接受，即使状态码在 2xx 成功范围内。

headers

HttpHeaders

所有响应头。

status

number

响应的状态码。

statusText

string

响应状态码的文本描述，默认为 OK。

请不要在代码中依赖它。

url

string

检索到资源的 URL，如果不可用则为 null。

type

HttpEventType.ResponseHeader | HttpEventType.Response

响应对象的类型，窄化为完整的响应对象或只有响应头。
