@angular/core

output

初始化器 API
程式碼

output 函式允許在指令和元件中宣告 Angular 輸出。

你可以使用輸出向父指令和元件傳送值。 父元件可以透過以下方式訂閱更改：

  • 範本事件繫結。例如，(myOutput)="doSomething($event)"
  • 透過使用 OutputRef#subscribe 進行程式設計方式訂閱。

API

output

OutputEmitterRef<T>
@paramoptsOutputOptions | undefined
@returnsOutputEmitterRef<T>

用法說明

要使用 output()，請從 @angular/core 匯入該函式。

          
import {output} from '@angular/core';

在你的元件內部，引入一個新的類別成員，並透過呼叫 output 來初始化它。

          
@Directive({  ...})export class MyDir {  nameChange = output<string>();    // OutputEmitterRef<string>  onClick    = output();            // OutputEmitterRef<void>}

你可以透過使用 OutputEmitterRef 中的 emit 方法，向指令的消費者傳送值。

          
updateName(newName: string): void {  this.nameChange.emit(newName);}
