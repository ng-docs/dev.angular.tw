output 函数允许在指令和组件中声明 Angular 的输出。

你可以使用输出向父指令和组件发送值。父组件可以通过以下方式订阅更改：

  • 模板事件绑定。例如， (myOutput)="doSomething($event)"
  • 通过使用 OutputRef#subscribe 进行编程订阅。

要使用 output()，请从 @angular/core 导入该函数。

          
            import {output} from '@angular/core`;

在你的组件中，引入一个新的类成员，并用 output 调用进行初始化。

          
            @Directive({  ...})export class MyDir {  nameChange = output<string>();    // OutputEmitterRef<string>  onClick    = output();            // OutputEmitterRef<void>}

你可以使用 OutputEmitterRef 的 emit 方法向指令的使用者发送值。

          
            updateName(newName: string): void {  this.nameChange.emit(newName);}
