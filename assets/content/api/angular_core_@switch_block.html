@switch 控制块的灵感来自 JavaScript 的 switch 语句：

语法

          
            @switch (condition) {  @case (caseA) {    Case A.  }  @case (caseB) {    Case B.  }  @default {    Default case.  }}

说明

@switch 控制块显示与条件表达式匹配的某个 case 选择的内容。条件表达式的值与 case 表达式使用 === 运算符进行比较。

@default 控制块是可选的，可以省略。如果没有 @case 与表达式匹配，并且没有 @default 控制块，则不显示任何内容。

@switch 没有贯穿，因此你不需要类似 break 或 return 语句的等价物。
