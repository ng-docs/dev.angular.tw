@angular/core

@switch

控制區塊
程式碼

@switch 控制區塊的靈感來源於 JavaScript 的 switch 語句：

語法 

          
@switch (condition) {  @case (caseA) {    Case A.  }  @case (caseB) {    Case B.  }  @default {    Default case.  }}

描述

@switch 控制區塊顯示的內容是由與條件表示式對應的 case 語句選擇的。條件表示式的值會使用 === 運算子與 case 表示式進行比較。

@default 控制區塊是可選的，可以省略。如果沒有 @case 對應表示式，並且沒有 @default 控制區塊，則不會顯示任何內容。

@switch 沒有 fallthrough 行為，因此你不需要等效於 break 或 return 語句的寫法。
