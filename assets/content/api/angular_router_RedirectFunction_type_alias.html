描述

當路徑與 Route 設定對應時，可用於處理重導向的函式型別。

RedirectFunction 確實可以訪問完整的 ActivatedRouteSnapshot 介面。在路由對應階段，某些資料無法準確得知。例如，解析器要稍後才會執行，因此任何已解析的標題都不會被填充。延遲載入的元件也是如此。對於直到根的所有快照來說，這也是成立的，因此也排除了包含父級（根、父級、pathFromRoot）的屬性。當然，完整的路由對應尚未發生，因此 firstChild 和 children 也都不可用。
