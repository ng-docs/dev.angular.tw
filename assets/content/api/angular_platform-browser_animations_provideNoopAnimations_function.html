返回一组依赖注入提供者以在应用中禁用动画。请参阅 动画指南 以了解更多关于 Angular 中动画的信息。

provideNoopAnimations(
  
): Provider[];

当你希望使用 bootstrapApplication 函数引导应用，但需要禁用动画时（例如，在运行测试时），此函数很有用。

          
            bootstrapApplication(RootComponent, {  providers: [    provideNoopAnimations()  ]});
