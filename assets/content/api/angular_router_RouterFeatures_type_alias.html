RouterFeatures

型別別名
程式碼

表示所有可與 provideRouter 一起使用的 Router 屬性的型別別名。 可以透過將特殊函式新增到 provideRouter 呼叫來啟用屬性。 請參閱每個符號的文件以查詢相應的函式名稱。另請參閱 provideRouter 文件，以瞭解如何使用這些函式。
