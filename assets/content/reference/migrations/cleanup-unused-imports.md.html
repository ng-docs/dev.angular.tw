清理未使用的匯入

edit

從版本 19 開始，當元件的 imports 陣列包含範本中未使用的符號時，Angular 會發出報告。

執行此原理圖將清理專案內所有未使用的匯入。

使用以下指令執行原理圖：

      
ng generate @angular/core:cleanup-unused-imports

之前 

      
import { Component } from '@angular/core';import { UnusedDirective } from './unused';@Component({  template: 'Hello',  imports: [UnusedDirective],})export class MyComp {}

之後 

      
import { Component } from '@angular/core';@Component({  template: 'Hello',  imports: [],})export class MyComp {}
