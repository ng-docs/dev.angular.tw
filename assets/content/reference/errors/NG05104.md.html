未找到根元素

edit

引导的组件定义在 @NgModule 装饰器的 bootstrap 属性中，或作为独立组件的 bootstrapApplication 的第一个参数。

当 Angular 尝试引导这些组件之一但无法在 DOM 中找到其对应节点时，会发生此错误。

调试此错误

此问题发生在选择器与标签不匹配时

      
@Component({
  selector: 'my-app',
  ...
})
class AppComponent {}

    
      
<html>
    <app-root></app-root> <!-- Doesn't match the selector -->
</html>

将标签替换为正确的：

      
<html>
    <my-app></my-app> <!-- OK -->
</html>
