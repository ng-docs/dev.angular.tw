找不到根元素

edit

以獨立元件形式啟動的元件定義於 @NgModule 裝飾器的 bootstrap 屬性中，或定義於 bootstrapApplication 的第一個參數中。

當 Angular 嘗試啟動其中一個元件，但無法在 DOM 中找到其對應的節點時，會發生此錯誤。

偵錯錯誤

當選擇器與標籤不匹配時會發生此問題

      
@Component({
  selector: 'my-app',
  ...
})
class AppComponent {}

    
      
<html>
    <app-root></app-root> <!-- Doesn't match the selector -->
</html>

用正確的標籤取代標籤：

      
<html>
    <my-app></my-app> <!-- OK -->
</html>
