未找到根元素

edit

已引導元件在 @NgModule 裝飾器的 bootstrap 屬性中定義，或者作為獨立元件的 bootstrapApplication 的第一個引數。

當 Angular 嘗試引導這些元件之一，但在 DOM 中找不到其對應的節點時，會發生此錯誤。

除錯錯誤

當選擇器與標籤不對應時，會發生此問題

      
@Component({  selector: 'my-app',  ...})class AppComponent {}

    
      
<html>    <app-root></app-root> <!-- Doesn't match the selector --></html>

用正確的標籤替換該標籤：

      
<html>    <my-app></my-app> <!-- OK --></html>
