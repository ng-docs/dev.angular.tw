參數非文字

edit

為了讓 Angular 編譯器中的元數據萃取速度更快，以下裝飾器 @NgModule、 @Pipe、@Component、@Directive 和 @Injectable 只接受物件字面量(object literal)作為參數。

這是一個 Ivy 中的故意變更，它對裝飾器的參數要求比 View Engine 更嚴格。 Ivy 採用這種方法是因為它會編譯裝飾器，將表達式移至類別輸出中的其他位置。

偵錯錯誤

移動所有宣告：

      
const moduleDefinition = {…}
@NgModule(moduleDefinition)
export class AppModule {
    constructor() {}
}

into the decorator:

      
@NgModule({…})
export class AppModule {
    constructor() {}
}
