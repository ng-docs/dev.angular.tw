参数不是字面量

edit

为了加快 Angular 编译器中的元数据提取，装饰器 @NgModule、 @Pipe、 @Component、 @Directive 和 @Injectable 仅接受对象字面量作为参数。

这是 Ivy 中 有意的更改，它对装饰器的参数要求比 View Engine 更严格。Ivy 需要这种方法，因为它通过将表达式移动到类输出的其他位置来编译装饰器。

调试此错误

把所有声明：

      
const moduleDefinition = {…}
@NgModule(moduleDefinition)
export class AppModule {
    constructor() {}
}

移入装饰器：

      
@NgModule({…})
export class AppModule {
    constructor() {}
}
