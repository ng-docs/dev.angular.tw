选择器冲突

edit

两个或更多 组件使用相同的元素选择器。因为一个元素只能关联一个组件，所以选择器必须是唯一的字符串，以防止 Angular 的歧义。

调试此错误

使用错误消息中的元素名称搜索代码库中使用相同选择器声明的地方：

      
@Component({
  selector: 'YOUR_STRING',
  …
})

确保每个组件都有一个唯一的 CSS 选择器。这将确保 Angular 渲染你所期望的组件。

如果你在查找具有此选择器标签名的多个组件时遇到问题，请检查导入的组件库（如 Angular Material）中的组件。确保你遵循 最佳实践，以防止选择器冲突。
