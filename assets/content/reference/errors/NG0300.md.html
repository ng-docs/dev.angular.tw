選擇器衝突

edit

兩個或多個 元件 使用相同的元素選取器。因為一個元素只能關聯到一個元件，所以選取器必須是唯一的字串，以防止 Angular 產生歧義。

偵錯錯誤

使用錯誤訊息中的元素名稱，在你的程式碼庫中搜尋你使用相同選擇器宣告的地方：

      
@Component({
  selector: 'YOUR_STRING',
  …
})

確保每個元件都有唯一的 CSS 選擇器。這將保證 Angular 呈現您預期的元件。

如果您使用這個選擇器標記名稱時遇到尋找多個元件的問題，請檢查來自匯入元件庫（例如 Angular Material）的元件。請務必遵循選擇器的 最佳實務以避免衝突。
