無效的 Attribute

edit

在編譯期間，無法求解屬性（Attribute）或屬性（Property）。

當嘗試繫結到不存在的屬性（Property）時，會發生此錯誤。 任何屬性（Property）繫結都必須對應於以下兩者之一：

  • HTML 元素上的原生屬性（Property），或
  • 應用於該元素的元件或指令的 @Input() 屬性（Property）。

此問題的執行時錯誤是 NG0304: '${tagName}' 不是已知元素: …'。

除錯此錯誤

檢視所用特定綁定語法的文件。這通常是筆誤或匯入不正確。 也可能存在缺少方向的屬性（Property）選擇器 'name' 或缺少輸入。
