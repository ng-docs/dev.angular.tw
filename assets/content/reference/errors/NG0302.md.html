管道找不到

edit

Angular 找不到具有此名稱的管道。

範本中參照的 pipe 尚未命名或正確宣告。

要使用管道：

  • 確保在範本中使用的名稱與在 pipe 裝飾器中定義的名稱相符。
  • 透過將 standalone: true 旗標新增到 pipe 的裝飾器，將其標記為獨立，或透過將其新增至該模組的宣告陣列，將其宣告為 NgModule 的一部分。
  • 在獨立元件和/或需要它的 NgModules 中匯入它。

偵錯錯誤

使用管線名稱追蹤管線在何處宣告和使用。

要解決這個錯誤：

  • 如果管線是 NgModule 的本機，請在管線的裝飾器中給予它一個唯一名稱，並在 NgModule 中宣告它。
  • 如果管線是獨立的或是在另一個 NgModule 中宣告，請將它加入獨立元件或目前 NgModule 的 imports 欄位中。

如果您最近新增一個匯入或宣告，您可能需要重新啟動您的伺服器來檢視這些變更。
