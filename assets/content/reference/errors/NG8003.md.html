缺少引用目标

edit

Angular 找不到带有 {{ PLACEHOLDER }} 导出名称的指令。这通常是因为指令上缺少导入或缺少 exportAs。

帮助： 这是一个常见运行时错误的编译器等价物 NG0301: 未找到导出。

调试此错误

使用找不到的导出字符串名称来跟踪使用此导出的模板或模块。

确保所有依赖项都正确导入并在我们的模块中声明。 例如，如果未找到的导出是 ngForm，我们需要导入 FormsModule 并在 *.module.ts 中的导入列表中声明它，以解决缺少的导出错误。

      
import { FormsModule } from '@angular/forms';
@NgModule({
  …
  imports: [
    FormsModule,
    …

如果最近添加了导入，则需要重新启动服务器以查看这些更改。
