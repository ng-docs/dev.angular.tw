缺少引用目標

edit

Angular 找不到帶有 {{ PLACEHOLDER }} 匯出名稱的指令。 這通常是由於缺少匯入或指令上缺少 exportAs 導致的。

提示： 這相當於編譯器中常見的執行時錯誤 NG0301：找不到匯出。

除錯錯誤

使用找不到的匯出的字串名稱來追蹤使用此匯出的樣板或模組。

確保所有依賴項都已正確匯入並在我們的模組中宣告。 例如，如果找不到的匯出是 ngForm，我們將需要匯入 FormsModule 並在 *.module.ts 的匯入清單中宣告它，以解決找不到匯出的錯誤。

      
import { FormsModule } from '@angular/forms';@NgModule({  …  imports: [    FormsModule,    …

如果您最近新增了匯入，則需要重啟伺服器才能看到這些更改。
