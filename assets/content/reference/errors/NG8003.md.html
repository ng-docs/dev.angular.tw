遺失的參考目標

edit

Angular 無法找到具有 {{ PLACEHOLDER }} 匯出名稱的指令。 這通常是指令中遺漏匯入或遺漏 exportAs 所導致。

HELPFUL: 這是編譯器等同於常見執行時期錯誤 NG0301: 輸出找不到。

偵錯錯誤

使用找不到的匯出之字串名稱來追蹤使用此匯出的範本或模組。

確保所有相依性都正確地匯入和宣告在我們的模組中。 例如，如果找不到的匯出是 ngForm，我們需要匯入 FormsModule 並將它宣告在 *.module.ts 的匯入清單中，以解決遺失的匯出錯誤。

      
import { FormsModule } from '@angular/forms';
@NgModule({
  …
  imports: [
    FormsModule,
    …

如果您最近添加了一個匯入，您將需要重新啟動伺服器才能看到這些變更。
