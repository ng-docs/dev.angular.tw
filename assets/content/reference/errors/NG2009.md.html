无效的 Shadow DOM 选择器

edit

使用 ViewEncapsulation.ShadowDom 的组件的选择器不符合自定义元素标签名称要求。

为了使标签名称被视为有效的自定义元素名称，它必须：

  • 是小写的。
  • 包含连字符。
  • 以字母（a-z）开头。

调试此错误

重命名组件的选择器，使其符合要求。

之前：

      
@Component({
  selector: 'comp',
  encapsulation: ViewEncapsulation.ShadowDom
  …
})

之后：

      
@Component({
  selector: 'app-comp',
  encapsulation: ViewEncapsulation.ShadowDom
  …
})
