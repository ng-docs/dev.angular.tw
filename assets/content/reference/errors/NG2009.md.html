無效 Shadow DOM 選擇器

edit

使用 ViewEncapsulation.ShadowDom 的元件選取器不符合自訂元素標記名稱需求。

要讓標籤名稱被視為有效的自訂元素名稱，它必須：

  • 以小寫字母表示。
  • 包含連接字元。
  • 以字母 (a-z) 開頭。

偵錯錯誤

重命名元件選擇器，讓它符合需求。

之前：

      
@Component({
  selector: 'comp',
  encapsulation: ViewEncapsulation.ShadowDom
  …
})

之後：

      
@Component({
  selector: 'app-comp',
  encapsulation: ViewEncapsulation.ShadowDom
  …
})
