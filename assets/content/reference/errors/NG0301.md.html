未找到导出

edit

Angular 找不到具有 {{ PLACEHOLDER }}导出名称的指令。导出名称在指令装饰器的 exportAs属性中指定。这在模板中使用 FormsModule 或 Material 模块而忘记导入对应模块时很常见。

提示：这是常见编译器错误 NG8003: No directive found with export的运行时等效错误。

调试此错误

根据此导出名来追踪使用了此导出的模板或模块。

确保所有依赖项都已正确导入并在你的 NgModules 中声明。例如，如果找不到的导出是 ngForm，我们需要导入 FormsModule 并在 *.module.ts 的导入列表中声明它以解决此错误。

      
import { FormsModule } from '@angular/forms';
@NgModule({
  …
  imports: [
    FormsModule,
    …

如果你最近添加了导入，则可能需要重新启动服务器以查看这些更改。
