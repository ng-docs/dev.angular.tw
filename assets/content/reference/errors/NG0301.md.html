未找到匯出

edit

Angular 找不到帶有 {{ PLACEHOLDER }} 匯出名的指令。匯出名是在指令裝飾器的 exportAs 屬性中指定的。當在樣板中使用 FormsModule 或 Material 模組，但忘記匯入相應的模組時，這很常見。

提示： 這在執行時等效於常見的編譯器錯誤 NG8003：找不到帶有匯出的指令。

除錯錯誤

使用匯出名來追蹤使用此匯出的樣板或模組。

確保所有依賴都已在你的 NgModule 中正確匯入和聲明。例如，如果未找到的匯出是 ngForm，我們需要匯入 FormsModule 並在 *.module.ts 的匯入清單中宣告它以解決此錯誤。

      
import { FormsModule } from '@angular/forms';@NgModule({  …  imports: [    FormsModule,    …

如果你最近新增了匯入，你可能需要重啟伺服器才能看到這些更改。
