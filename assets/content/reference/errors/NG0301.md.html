匯出未找到

edit

Angular 無法找到具有 {{ PLACEHOLDER }} 匯出名稱的指令。匯出名稱是在指令裝飾器的 exportAs 屬性中指定。當在範本中使用 FormsModule 或 Material 模組，而且您忘記匯入對應的模組時，通常會發生這種情況。

HELPFUL: 這是常見編譯器錯誤 [NG8003: 找不到具有 export 的指令] 的執行時期等效項目 (errors/NG8003)。

偵錯錯誤

使用 export 名稱來追蹤使用此 export 的範本或模組。

請確保所有相依性都正確地匯入並在您的 NgModules 中宣告。例如，如果找不到的匯出是 ngForm，我們需要匯入 FormsModule 並在 *.module.ts 中的匯入清單中宣告它來解決錯誤。

      
import { FormsModule } from '@angular/forms';
@NgModule({
  …
  imports: [
    FormsModule,
    …

如果您最近添加了匯入，可能需要重新啟動伺服器才能看到這些變更。
