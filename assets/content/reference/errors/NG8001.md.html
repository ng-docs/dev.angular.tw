無效元素

edit

在編譯期間，一個或多個元素無法被求解，原因是該元素未由 HTML 規範定義，或者沒有元件或指令具有這樣的元素選擇器。

提示： 這相當於編譯器中常見的執行時錯誤 NG0304: '${tagName}' 不是已知元素：...。

除錯錯誤

使用錯誤訊息中的元素名稱來查詢正在使用該元素的檔案。

檢查名稱和選擇器是否正確。

透過檢查元件是否出現在 imports 欄位中，確保該組件已正確匯入到你的 NgModule 或獨立元件中。如果該組件是在 NgModule 中宣告的（意味著它不是獨立的），請透過檢查元件是否出現在 exports 欄位中，確保它已從中正確匯出。

當使用自定義元素或 Web Components 時，請確保將 CUSTOM_ELEMENTS_SCHEMA 新增到應用模組中。

如果這樣做未能求解此錯誤，請檢查匯入的函式庫，看看你正在使用的匯出和屬性是否有任何最新更改，並重啟你的伺服器。
