CLI 参考

edit
命令 别名 说明
add 为你的项目添加对外部库的支持。
analytics 配置 Angular CLI 使用指标的收集。
build b 将 Angular 应用或库编译到指定输出路径中的 dist/ 目录。
cache 配置持久磁盘缓存并检索缓存统计信息。
completion 为你的终端设置 Angular CLI 自动补全。
config 检索或设置 angular.json 文件中的 Angular 配置值。
deploy 调用指定项目或工作区中的默认项目的部署构建器。
e2e e 构建并启动 Angular 应用，然后运行端到端测试。
extract-i18n 从源代码中提取 i18n 消息。
generate g 基于一个原理图生成和/或修改文件。
lint 在给定项目文件夹中对 Angular 应用代码运行代码检查工具。
new n 创建一个新的 Angular 工作区。
run 运行带有可选自定义构建器配置的 Architect 目标，定义在你的项目中。
serve s, dev 构建并提供你的应用程序，在文件更改时重新构建。
test t 在项目中运行单元测试。
update 更新你的工作区及其依赖项。详见 https://angular.dev/update-guide/。
version v 输出 Angular CLI 版本。
