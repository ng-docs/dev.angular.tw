CLI 參考

edit
指令 別名 描述
add 為你的專案新增對外部函式庫的支援。
analytics 設定 Angular CLI 使用情況指標的收集。
build b 將 Angular 應用或函式庫編譯到給定輸出路徑的名為 dist/ 的輸出目錄中。
cache 設定持久磁碟快取並檢索快取統計訊息。
completion 為你的終端建立 Angular CLI 自動補全。
config 檢索或設定工作區 angular.json 檔案中的 Angular 設定值。
deploy 為指定專案或工作區中的預設專案呼叫部署建構式。
e2e e 建立併為 Angular 應用啟動伺服器，然後執行端到端測試。
extract-i18n 從原始碼中提取 i18n 訊息。
generate g 基於原理圖生成和/或修改檔案。
lint 在給定的專案資料夾中對 Angular 應用程式碼執行 lint 工具。
new n 建立一個新的 Angular 工作區。
run 使用在你的專案中定義的可選自定義建構式設定來執行 Architect target。
serve s、dev 建立你的應用併為其啟動伺服器，並在檔案更改時重新建立。
test t 在一個專案中執行單元測試。
update 更新你的工作區及其依賴項。請參閱 https://angular.dev/update-guide/。
version v 輸出 Angular CLI 版本。
