信號必須在範本插補中呼叫。

edit

此診斷偵測範本插補中未呼叫的訊號。

      
import {Component, signal, Signal} from '@angular/core';
@Component({
  template: `<div>{{ mySignal }}/div>`,
})
class MyComponent {
    mySignal: Signal<number> = signal(0);
}

有什麼問題？

Angular Signals 是零參數函式 (() => T)。執行時，會傳回訊號的目前值。 這表示它們應該在用於範本內插時呼叫，以呈現它們的值。

我該怎麼做？

在範本內插中使用信號以呈現其值時，請務必呼叫該信號。

      
import {Component, signal, Signal} from '@angular/core';
@Component({
  template: `<div>{{ mySignal() }}/div>`,
})
class MyComponent {
  mySignal: Signal<number> = signal(0)
}

組態需求

必須啟用 strictTemplates 才能發出任何延伸診斷。 interpolatedSignalNotInvoked 除了 strictTemplates 之外沒有其他額外需求。

如果我無法避免這件事怎麼辦？

這個診斷可以透過編輯專案的 tsconfig.json 檔案停用：

      
{
  "angularCompilerOptions": {
    "extendedDiagnostics": {
      "checks": {
        "interpolatedSignalNotInvoked": "suppress"
      }
    }
  }
}
