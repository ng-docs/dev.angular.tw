後綴不支援

edit

此診斷偵測何時將 .px、.% 和 .em 字尾與屬性繫結一起使用。

      
<img [attr.width.px]="5">

有什麼問題？

這些字尾僅可使用於樣式繫結。在繫結至屬性時，它們沒有任何意義。

我該怎麼做？

不要使用僅在樣式繫結中支援的 .px、.% 或 .em 後綴， 將其移至繫結的值分配。

      
<img [attr.width]="'5px'">

組態需求

必須啟用 strictTemplates 才能發出任何擴充診斷。 invalidBananaInBox 除了 strictTemplates 之外沒有其他額外需求。

如果我無法避免這件事怎麼辦？

這個診斷可以透過編輯專案的 tsconfig.json 檔案停用：

      
{
  "angularCompilerOptions": {
    "extendedDiagnostics": {
      "checks": {
        "suffixNotSupported": "suppress"
      }
    }
  }
}
