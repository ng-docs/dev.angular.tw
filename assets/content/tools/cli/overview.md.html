Angular CLI

edit

Angular CLI 是一个命令行界面工具，允许你直接从命令行界面中搭建、开发、测试、部署和维护 Angular 应用程序。

Angular CLI 作为 @angular/cli 包发布在 npm 上，包含一个名为 ng 的二进制文件。使用 ng 命令调用 Angular CLI。

无需本地环境，学习 Angular

如果你是 Angular 新手，你可能想从 立即尝试开始，该教程将在一个现成的基本在线商店应用程序的背景下，介绍 Angular 的基本知识，供你查看和修改。这个独立教程利用交互式的 StackBlitz 环境进行在线开发。你不需要在准备好之前设置本地环境。

CLI 命令语法

Angular CLI 大致遵循 Unix/POSIX 惯例的选项语法。

布尔型选项

布尔选项有两种形式： --this-option 将标志设置为 true， --no-this-option 将其设置为 false。你也可以使用 --this-option=false 或 --this-option=true。如果未提供任何选项，则标志保持其默认状态，如参考文档中所列。

数组选项

数组选项可以用两种形式提供：--option value1 value2 或 --option value1 --option value2。

相对路径

用来指定文件的选项可以用绝对路径，也可以用相对于当前目录的相对路径，当前目录通常是工作区或项目的根目录。
