什麼是 Angular？

edit

Angular 是一個 Web 框架，旨在幫助開發者建立快速、可靠的應用。

Angular 由 Google 的專業團隊維護，提供了一整套廣泛的工具、API 和函式庫，以簡化和最佳化你的開發工作流程。Angular 為你提供了一個堅實的平台，使你能夠建立快速、可靠的應用，並能隨著你的團隊規模和程式碼庫規模的擴大而擴充套件。

想看一些程式碼範例嗎？ 請跳轉到我們的 入門指南 以快速瞭解 Angular 的使用方式，或者如果你喜歡按照步驟操作，請從教程開始。

為你的開發賦能的屬性

使用具有統一規範的元件模型和靈活的依賴注入體系，保持程式碼庫的井井有條

Angular 元件使你可以輕鬆地將程式碼拆分為良好封裝的部分。

通用的依賴注入有助於你保持程式碼的模組化、松耦合和可測試性。

元件入門

透過基於 Signal 的細粒度回應式系統，獲得快速的狀態更新

我們細粒度的回應式模型，結合編譯時最佳化，簡化了開發，並有助於預設建立更快的應用。

細粒度地跟蹤狀態在整個應用中的使用方式和位置，使框架能夠透過高度最佳化的指令渲染快速更新。

探索 Angular Signal

透過 SSR、SSG、水合和下一代延遲載入，實現你的效能目標

Angular 同時支援伺服器端渲染 (SSR) 和靜態站點生成 (SSG) 以及完整的 DOM 水合。樣板中的 @defer 控制區塊可以輕鬆地將樣板宣告式地劃分為可延遲載入的部分。

閱讀關於 SSR 的內容

使用 Angular 的表單、路由等第一方模組，保證一切協同工作

Angular 的路由器 提供了一個功能豐富的導向工具套件，包括對路由守衛、資料求解、延遲載入等的支援。

Angular 的表單模組 提供了一個用於表單參與和驗證的標準化系統。

以超乎以往的速度開發應用

使用 Angular CLI，輕鬆建立、啟動伺服器、測試和部署

Angular CLI 可以在不到一分鐘的時間內啟動你的專案，並提供你所需的指令，以便將其發展為已部署的生產應用。

Angular CLI

使用 Angular DevTools 瀏覽器擴充套件，以視覺化方式除錯、分析和最佳化你的程式碼

Angular DevTools 與你的瀏覽器開發者工具並排就坐。它可以幫助除錯和分析你的應用，包括元件樹檢查器、依賴注入樹檢視和自定義效能分析火焰圖。

Angular DevTools

使用 ng update，絕不會錯過任何版本

Angular CLI 的 ng update 執行自動化程式碼轉換，自動處理例行的破壞性變更，從而大大簡化了主版本更新。保持使用最新版本可使你的應用儘可能快速和安全。

ng update

透過在你最喜歡的編輯器中整合 IDE，保持高效

Angular 的 IDE 語言服務在你最喜歡的編輯器中支援程式碼補全、導向、重構和實時診斷。

語言服務

充滿信心地交付

在 Google 龐大的單一程式碼倉庫中，逐個送出地進行驗證

每個 Angular 送出都會在 Google 的內部程式碼倉庫中針對成千上萬的測試進行檢查，代表著無數真實的場景。

Angular 致力於為 Google 的一些最大型產品（包括 Google Cloud）提供穩定性。這一承諾確保變更都經過充分測試、向後相容，並在可能的情況下包含遷移工具。

瞭解關於 Google 的單一程式碼倉庫

清晰的支援策略和可預測的發布時間表

Angular 可預測的、基於時間的發布時間表，使你的組織對框架的穩定性和向後相容性充滿信心。長期支援 (LTS) 視窗確保你在需要時獲得關鍵的安全修復。第一方更新工具、指南和自動化遷移原理圖有助於使你的應用與框架和 Web 平台的最新進展保持同步。

版本控制與發布

適用於任何規模

透過國際化支援，覆蓋全球使用者

Angular 的國際化屬性處理訊息翻譯和格式化，包括對 Unicode 標準 ICU 語法的支援。

國際化

透過預設安全保護你的使用者

透過與 Google 世界一流的安全工程師合作，Angular 旨在使開發預設情況下是安全的。內建的安全屬性，包括 HTML 無害化和可信型別支援，有助於保護你的使用者免受常見的漏洞攻擊，如跨站指令碼和跨站請求偽造。

安全性

使用 Vite 和 esbuild，保持大型團隊的高效

Angular CLI 包含一個使用 Vite 和 ESBuild 的快速、現代化的建立流水線。開發者報告說，建立包含數十萬行程式碼的專案只需不到一分鐘的時間。

ESBuild 和 Vite

已在 Google 的一些最大的 Web 應用中得到驗證

大型 Google 產品基於 Angular 的架構建立，並幫助開發新的屬性，從而進一步提高 Angular 的延展性，從 Google Fonts 到 Google Cloud。
