什么是 Angular？

edit

Angular 是一个 Web 框架，能够帮助开发者构建快速、可靠的应用。

由 Google 的专业团队维护，Angular 提供了广泛的工具、API 和库，简化和优化你的开发工作流程。Angular 为你提供了一个坚实的平台，可用于构建快速、可靠、能够随着团队规模和代码库规模扩展的应用程序。

想看一些代码吗？转到我们的 基础知识快速了解使用 Angular 的情况，或者在 教程中开始，如果你更喜欢按步骤的说明。

支持你开发的功能

通过具有明确定义的组件模型和灵活的依赖注入系统保持代码库组织良好

Angular 组件使将代码拆分为封装良好的部分变得简单。

多功能的依赖注入帮助你保持代码模块化、松耦合和易于测试。

从组件开始

通过基于信号的细粒度反应性实现快速状态更新

我们的细粒度反应性模型，结合编译时优化，简化了开发，通过高度优化的指令默认构建更快的应用程序。

通过细粒度跟踪应用程序中状态的使用方式和位置，使框架能够通过高度优化的指令快速更新。

探索 Angular Signals

通过 SSR、SSG、水合和下一代延迟加载实现性能目标

Angular 支持服务器端渲染（SSR）和静态网站生成（SSG），以及完整的DOM水合。模板中的 @defer块使得将模板分割为可延迟加载部分变得简单。

阅读关于 SSR 的内容

通过 Angular 的官方模块，保证所有模块能够协同工作，包括表单、路由等

Angular 的路由器提供了功能丰富的导航工具包，包括路由守卫、数据解析、延迟加载等功能。

Angular 的表单模块提供了标准化的表单参与和验证系统。

比以往更快地开发应用程序

使用 Angular CLI 轻松构建、运行、测试和部署

Angular CLI 通过你需要的命令，在不到一分钟的时间内让你的项目运行起来，以便发展成为一个部署的生产应用程序。

Angular CLI

使用 Angular DevTools 浏览器扩展，可视化调试、分析和优化你的代码。

Angular DevTools 与你的浏览器开发工具并列。它帮助调试和分析你的应用，包括组件树检查器、依赖注入树视图和自定义性能分析火焰图。

Angular 开发者工具

使用 ng update 永不遗漏任何版本

Angular CLI 的 ng update 运行自动化代码转换，自动处理常规破坏性更改，极大简化主要版本更新。保持最新版本，使你的应用尽可能快速和安全。

ng update

在你喜爱的编辑器中使用 IDE 集成保持高效

Angular 的 IDE 语言服务支持代码补全、导航、重构和实时诊断在你喜爱的编辑器中。

语言服务

自信地部署

逐一验证提交到谷歌庞大的单一代码仓库

每次 Angular 提交都会针对 Google 内部代码库中的 数十万个测试进行检查，这些测试代表了无数的真实场景。

Angular 致力于为谷歌的一些最大产品提供稳定性。这一承诺确保更改经过充分测试、向后兼容，并在可能的情况下包含迁移工具。

了解 Google 的单一代码仓

明确的支持政策和可预测的发布计划

Angular 的可预测、基于时间的发布计划使你的组织对框架的稳定性和向后兼容性充满信心。长期支持（LTS）窗口确保在需要时获得关键的安全修复。官方更新工具、指南和自动迁移原理图帮助保持你的应用与框架和网络平台的最新进展同步。

版本控制和发布

适用于任何规模

通过国际化支持，覆盖到各个用户

Angular 的国际化功能处理消息翻译和格式化，包括支持 Unicode 标准 ICU 语法。

国际化

默认安全保护用户

与谷歌世界一流的安全工程师合作，Angular 旨在通过默认使开发变得更加安全。内置安全功能，包括 HTML 消毒和可信类型支持，有助于保护用户免受常见漏洞，如跨站脚本和跨站请求伪造。

安全

使用 Vite 和 esbuild 使大型团队保持高效

Angular CLI 包含一个快速、现代的构建流水线，使用 Vite 和 ESBuild。开发人员报告在不到一分钟内构建数十万行代码的项目。

ESBuild 和 Vite

在谷歌一些最大的 Web 应用中经过验证

基于 Angular 架构构建的大型谷歌产品，帮助开发新功能，进一步改善 Angular 的可伸缩性，从 Google Fonts 到 Google Cloud。
