管理动态数据

edit

定义组件的状态和行为来管理动态数据。

现在我们已经学习了组件的基本结构，让我们学习如何定义组件的数据（即状态）和行为。

什么是状态？

组件可以让你整洁地封装应用某些离散部分的职责。例如，一个 SignUpForm 组件可能需要在允许用户执行特定动作之前跟踪表单是否有效。因此，组件需要跟踪的各种属性通常被称为「状态」。

定义状态

要定义状态，你可以在组件内使用 类字段语法。

例如，使用 TodoListItem 组件，创建两个你想要跟踪的属性：

  1. taskTitle — 任务的标题
  2. isComplete — 任务是否完成
      
// todo-list-item.component.ts
@Component({ ... })
export class TodoListItem {
  taskTitle = '';
  isComplete = false;
}

更新状态

当你想更新状态时，通常通过在组件类中定义方法来实现，这些方法可以使用 this 关键字访问各种类字段。

      
// todo-list-item.component.ts
@Component({ ... })
export class TodoListItem {
  taskTitle = '';
  isComplete = false;
  completeTask() {
    this.isComplete = true;
  }
  updateTitle(newTitle: string) {
    this.taskTitle = newTitle;
  }
}
