屬性型指令

edit

使用屬性型指令更改 DOM 元素和 Angular 元件的外觀或行為。

建立屬性型指令

本節將引導你建立一個高亮指令，該指令將宿主元素的背景顏色設定為黃色。

  1. 要建立指令，請使用 CLI 指令 ng generate directive。

          
    ng generate directive highlight

    CLI 會建立 src/app/highlight.directive.ts 和一個對應的測試檔案 src/app/highlight.directive.spec.ts。

    src/app/highlight.directive.ts

          
    import {Directive} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {}

    @Directive() 裝飾器的設定屬性指定了該指令的 CSS 屬性 Attribute 選擇器 [appHighlight]。

  2. 從 @angular/core 匯入 ElementRef。 ElementRef 允許透過其 nativeElement 屬性直接訪問宿主 DOM 元素。

  3. 在指令的 constructor() 中新增 ElementRef，以便注入對宿主 DOM 元素的引用，即你應用 appHighlight 的元素。

  4. 向 HighlightDirective 類新增邏輯，以將背景設定為黃色。

    src/app/highlight.directive.ts

          
    import {Directive, ElementRef, inject} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  constructor() {    this.el.nativeElement.style.backgroundColor = 'yellow';  }}

提示： 指令不支援命名空間。

src/app/app.component.avoid.html (不支援)

      
<p app:Highlight>This is invalid</p>

應用屬性型指令

  1. 要使用 HighlightDirective，請將帶有該指令的 <p> 元素作為屬性新增到 HTML 樣板中。

    src/app/app.component.html

          
    <h1>My First Attribute Directive</h1><p appHighlight>Highlight me!</p><p appHighlight="yellow">Highlighted in yellow</p><p [appHighlight]="'orange'">Highlighted in orange</p><p [appHighlight]="color">Highlighted with parent component's color</p>

Angular 建立 HighlightDirective 類別的實例，並將對 <p> 元素的引用注入到該指令的建構式函式中，建構式函式會將 <p> 元素的背景樣式設定為黃色。

處理使用者事件

本節向你展示如何檢測使用者何時將滑鼠移入或移出元素，並透過設定或清除高亮顏色來做出回應。

  1. 從『@angular/core』匯入 HostListener。

    src/app/highlight.directive.ts (匯入)

          
    import {Directive, ElementRef, HostListener, inject} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @HostListener('mouseenter') onMouseEnter() {    this.highlight('yellow');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

  2. 新增兩個事件處理程式，以便在滑鼠進入或離開時做出回應，每個處理程式都帶有 @HostListener() 裝飾器。

    src/app/highlight.directive.ts (滑鼠方法)

          
    import {Directive, ElementRef, HostListener, inject} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @HostListener('mouseenter') onMouseEnter() {    this.highlight('yellow');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

使用 @HostListener() 裝飾器訂閱宿主屬性型指令的 DOM 元素的事件，在本例中為 <p>。

提示： 這些處理程式會委託給一個輔助函式 highlight()，該函式在宿主 DOM 元素 el 上設定顏色。

完整的指令如下所示：

src/app/highlight.directive.ts

      
import {Directive, ElementRef, HostListener, inject} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @HostListener('mouseenter') onMouseEnter() {    this.highlight('yellow');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

當指標懸停在段落元素上時，背景顏色出現；當指標移開時，背景顏色消失。

將值傳入屬性型指令

本節將引導你瞭解在應用 HighlightDirective 時如何設定高亮顏色。

  1. 在 highlight.directive.ts 中，從 @angular/core 匯入 Input。

    src/app/highlight.directive.ts (匯入)

          
    import {Directive, ElementRef, HostListener, inject, Input} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @Input() appHighlight = '';  @HostListener('mouseenter') onMouseEnter() {    this.highlight(this.appHighlight || 'red');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

  2. 新增一個 appHighlight @Input() 屬性（Property）。

    src/app/highlight.directive.ts

          
    import {Directive, ElementRef, HostListener, inject, Input} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @Input() appHighlight = '';  @HostListener('mouseenter') onMouseEnter() {    this.highlight(this.appHighlight || 'red');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

    @Input() 裝飾器向類新增元資料，使指令的 appHighlight 屬性（Property）可用於繫結。

  3. 在 app.component.ts 中，向 AppComponent 新增一個 color 屬性（property）。

    src/app/app.component.ts (class)

          
    import {Component} from '@angular/core';import {HighlightDirective} from './highlight.directive';@Component({  selector: 'app-root',  templateUrl: './app.component.1.html',  imports: [HighlightDirective],})export class AppComponent {  color = 'yellow';}

  4. 要同時應用指令和顏色，請將屬性（Property）繫結與 appHighlight 指令選擇器一起使用，並將其設定為等於 color。

    src/app/app.component.html (color)

          
    <h1>My First Attribute Directive</h1><h2>Pick a highlight color</h2><div>  <input type="radio" name="colors" (click)="color='lightgreen'">Green  <input type="radio" name="colors" (click)="color='yellow'">Yellow  <input type="radio" name="colors" (click)="color='cyan'">Cyan</div><p [appHighlight]="color">Highlight me!</p><p [appHighlight]="color" defaultColor="violet">  Highlight me too!</p><hr><h2>Mouse over the following lines to see fixed highlights</h2><p [appHighlight]="'yellow'">Highlighted in yellow</p><p appHighlight="orange">Highlighted in orange</p><hr><h2>ngNonBindable</h2><p>Use ngNonBindable to stop evaluation.</p><p ngNonBindable>This should not evaluate: {{ 1 + 1 }}</p><h3>ngNonBindable with a directive</h3><div ngNonBindable [appHighlight]="'yellow'">This should not evaluate: {{ 1 +1 }}, but will highlight yellow.</div>

    [appHighlight] 屬性（Attribute）繫結執行兩個任務：

    • 將高亮指令應用於 <p> 元素
    • 使用屬性（Property）繫結設定指令的高亮顏色

使用使用者輸入設定值

本節將指導你新增單選按鈕，以便將你的顏色選擇繫結到 appHighlight 指令。

  1. 按如下所示向 app.component.html 新增用於選擇顏色的標記：

    src/app/app.component.html (v2)

          
    <h1>My First Attribute Directive</h1><h2>Pick a highlight color</h2><div>  <input type="radio" name="colors" (click)="color='lightgreen'">Green  <input type="radio" name="colors" (click)="color='yellow'">Yellow  <input type="radio" name="colors" (click)="color='cyan'">Cyan</div><p [appHighlight]="color">Highlight me!</p><p [appHighlight]="color" defaultColor="violet">  Highlight me too!</p><hr><h2>Mouse over the following lines to see fixed highlights</h2><p [appHighlight]="'yellow'">Highlighted in yellow</p><p appHighlight="orange">Highlighted in orange</p><hr><h2>ngNonBindable</h2><p>Use ngNonBindable to stop evaluation.</p><p ngNonBindable>This should not evaluate: {{ 1 + 1 }}</p><h3>ngNonBindable with a directive</h3><div ngNonBindable [appHighlight]="'yellow'">This should not evaluate: {{ 1 +1 }}, but will highlight yellow.</div>

  2. 修改 AppComponent.color，使其沒有初始值。

    src/app/app.component.ts (class)

          
    import {Component} from '@angular/core';import {HighlightDirective} from './highlight.directive';@Component({  selector: 'app-root',  templateUrl: './app.component.html',  imports: [HighlightDirective],})export class AppComponent {  color = '';}

  3. 在 highlight.directive.ts 中，修改 onMouseEnter 方法，以便它首先嘗試使用 appHighlight 進行高亮顯示，如果 appHighlight 為 undefined，則回退到 red。

    src/app/highlight.directive.ts (mouse-enter)

          
    import {Directive, ElementRef, HostListener, inject, Input} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @Input() appHighlight = '';  @HostListener('mouseenter') onMouseEnter() {    this.highlight(this.appHighlight || 'red');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

  4. 為你的應用啟動伺服器，以驗證使用者可以使用單選按鈕選擇顏色。

繫結到第二個屬性（Property）

本節將指導你完成應用的設定，以便開發人員可以設定預設顏色。

  1. 向 HighlightDirective 新增第二個名為 defaultColor 的 Input() 屬性（Property）。

    src/app/highlight.directive.ts (defaultColor)

          
    import {Directive, ElementRef, HostListener, inject, Input} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @Input() defaultColor = '';  @Input() appHighlight = '';  @HostListener('mouseenter') onMouseEnter() {    this.highlight(this.appHighlight || this.defaultColor || 'red');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

  2. 修改指令的 onMouseEnter，以便它首先嘗試使用 appHighlight 進行高亮顯示，然後使用 defaultColor，如果這兩個屬性（Property）都為 undefined，則回退到 red。

    src/app/highlight.directive.ts (mouse-enter)

          
    import {Directive, ElementRef, HostListener, inject, Input} from '@angular/core';@Directive({  selector: '[appHighlight]',})export class HighlightDirective {  private el = inject(ElementRef);  @Input() defaultColor = '';  @Input() appHighlight = '';  @HostListener('mouseenter') onMouseEnter() {    this.highlight(this.appHighlight || this.defaultColor || 'red');  }  @HostListener('mouseleave') onMouseLeave() {    this.highlight('');  }  private highlight(color: string) {    this.el.nativeElement.style.backgroundColor = color;  }}

  3. 要繫結到 AppComponent.color 並回退到 『violet』 作為預設顏色，請新增以下 HTML。 在這種情況下，defaultColor 繫結不使用方括號 []，因為它是一個靜態值。

    src/app/app.component.html (defaultColor)

          
    <h1>My First Attribute Directive</h1><h2>Pick a highlight color</h2><div>  <input type="radio" name="colors" (click)="color='lightgreen'">Green  <input type="radio" name="colors" (click)="color='yellow'">Yellow  <input type="radio" name="colors" (click)="color='cyan'">Cyan</div><p [appHighlight]="color">Highlight me!</p><p [appHighlight]="color" defaultColor="violet">  Highlight me too!</p><hr><h2>Mouse over the following lines to see fixed highlights</h2><p [appHighlight]="'yellow'">Highlighted in yellow</p><p appHighlight="orange">Highlighted in orange</p><hr><h2>ngNonBindable</h2><p>Use ngNonBindable to stop evaluation.</p><p ngNonBindable>This should not evaluate: {{ 1 + 1 }}</p><h3>ngNonBindable with a directive</h3><div ngNonBindable [appHighlight]="'yellow'">This should not evaluate: {{ 1 +1 }}, but will highlight yellow.</div>

    與元件一樣，你可以向宿主元素新增多個指令屬性（Property）繫結。

如果沒有預設顏色繫結，則預設顏色為紅色。 當用戶選擇顏色時，所選顏色將成為活動的高亮顏色。

使用 NgNonBindable 停用 Angular 處理

要防止在瀏覽器中進行表示式求值，請將 ngNonBindable 新增到宿主元素。 ngNonBindable 會停用樣板中的內嵌、指令和繫結。

在以下範例中，表示式 {{ 1 + 1 }} 的渲染效果與在程式碼編輯器中一樣，並且不顯示 2。

src/app/app.component.html

      
<h1>My First Attribute Directive</h1><h2>Pick a highlight color</h2><div>  <input type="radio" name="colors" (click)="color='lightgreen'">Green  <input type="radio" name="colors" (click)="color='yellow'">Yellow  <input type="radio" name="colors" (click)="color='cyan'">Cyan</div><p [appHighlight]="color">Highlight me!</p><p [appHighlight]="color" defaultColor="violet">  Highlight me too!</p><hr><h2>Mouse over the following lines to see fixed highlights</h2><p [appHighlight]="'yellow'">Highlighted in yellow</p><p appHighlight="orange">Highlighted in orange</p><hr><h2>ngNonBindable</h2><p>Use ngNonBindable to stop evaluation.</p><p ngNonBindable>This should not evaluate: {{ 1 + 1 }}</p><h3>ngNonBindable with a directive</h3><div ngNonBindable [appHighlight]="'yellow'">This should not evaluate: {{ 1 +1 }}, but will highlight yellow.</div>

將 ngNonBindable 應用於元素會停止對該元素的子元素進行繫結。 但是，ngNonBindable 仍然允許指令在你應用 ngNonBindable 的元素上工作。 在以下範例中，appHighlight 指令仍然處於活動狀態，但 Angular 不會對錶達式 {{ 1 + 1 }} 求值。

src/app/app.component.html

      
<h1>My First Attribute Directive</h1><h2>Pick a highlight color</h2><div>  <input type="radio" name="colors" (click)="color='lightgreen'">Green  <input type="radio" name="colors" (click)="color='yellow'">Yellow  <input type="radio" name="colors" (click)="color='cyan'">Cyan</div><p [appHighlight]="color">Highlight me!</p><p [appHighlight]="color" defaultColor="violet">  Highlight me too!</p><hr><h2>Mouse over the following lines to see fixed highlights</h2><p [appHighlight]="'yellow'">Highlighted in yellow</p><p appHighlight="orange">Highlighted in orange</p><hr><h2>ngNonBindable</h2><p>Use ngNonBindable to stop evaluation.</p><p ngNonBindable>This should not evaluate: {{ 1 + 1 }}</p><h3>ngNonBindable with a directive</h3><div ngNonBindable [appHighlight]="'yellow'">This should not evaluate: {{ 1 +1 }}, but will highlight yellow.</div>

如果將 ngNonBindable 應用於父元素，則 Angular 會停用該元素子元素的任何型別的內嵌和繫結，例如屬性（Property）繫結或事件繫結。
