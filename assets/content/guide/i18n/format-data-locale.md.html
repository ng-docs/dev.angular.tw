基於語言環境格式化資料

edit

Angular 提供了下列內建的資料轉換通道。資料轉換通道使用 LOCALE_ID 令牌，以根據每個語言環境的規則格式化資料。

資料轉換通道 詳情
DatePipe 格式化日期值。
CurrencyPipe 將數字轉換為貨幣字串。
DecimalPipe 將數字轉換為十進位數字字串。
PercentPipe 將數字轉換為百分比字串。

使用 DatePipe 顯示當前日期

要以當前語言環境的格式顯示當前日期，請為 DatePipe 使用以下格式。

      
{{ today | date }}

為 CurrencyPipe 覆蓋當前語言環境

將 locale 引數新增到通道以覆蓋 LOCALE_ID 令牌的當前值。

要強制貨幣使用美式英語 (en-US)，請為 CurrencyPipe 使用以下格式

      
{{ amount | currency : 'en-US' }}

提示： 為 CurrencyPipe 指定的語言環境會覆蓋應用的全域 LOCALE_ID 令牌。
