類別繫結與樣式繫結

edit

使用類別和樣式繫結從元素的 class 屬性中新增和刪除 CSS 類別名稱，以及動態設定樣式。

繫結到單個 CSS class

要建立單個類別繫結，請鍵入以下內容：

[class.sale]="onSale"

當繫結表示式 onSale 為真時，Angular 新增該類別，當表示式為假時移除該類別，但 undefined 除外。當元素上存在來自不同指令的多個繫結到同一類別名稱時， undefined 的行為特別有用。

繫結到多個 CSS 類

要繫結到多個類，請鍵入以下內容：

[class]="classExpression"

表示式可以是以下之一：

  • 用空格分隔的類別名稱字串。
  • 以類別名稱作為鍵名並將真或假表示式作為值的物件。
  • 類別名稱的陣列。

對於物件格式，Angular 會在其關聯的值為真時才新增類。

重要：對於任何類似物件的表示式，比如 object、 Array、 Map 或 Set，物件的標識必須改變才能使 Angular 更新類別清單。更新屬性而不改變物件標識沒有效果。

如果存在多個繫結到相同類別名稱的繫結，Angular 使用樣式優先級確定要使用的繫結。

下表是各種類別繫結語法的小結。

繫結型別 語法 輸入型別 範例輸入值
單類別繫結 [class.sale]="onSale" boolean undefined
多類別繫結 [class]="classExpression" string "my-class-1 my-class-2 my-class-3"
多類別繫結 [class]="classExpression" Record undefined
多類別繫結 [class]="classExpression" Array ['foo', 'bar']

繫結到單個樣式

要建立單個樣式繫結，請使用字首 style，後跟點和CSS樣式的名稱。

例如，要設定 width 樣式，請輸入如下內容： [style.width]="width"

Angular 將該屬性設定為繫結表示式的值，這通常是一個字串。（可選）你可以新增單位副檔名，比如 em 或 %，這需要數字型別。

  1. 要以中線格式（dash-case）編寫樣式，請鍵入以下內容：

          
    <nav [style.background-color]="expression"></nav>

  2. 要以駝峰格式（camelCase）編寫樣式，請鍵入以下內容：

          
    <nav [style.backgroundColor]="expression"></nav>

繫結到多個樣式

要切換多個樣式，請繫結到 [style]屬性，例如， [style]="styleExpression"。 styleExpression可以是以下之一：

  • 樣式的字串清單，比如 "width: 100px; height: 100px; background-color: cornflowerblue;"。
  • 一個物件，其鍵名是樣式名，其值是樣式值，比如 {width: '100px', height: '100px', backgroundColor: 'cornflowerblue'}。

請注意，不支援將陣列繫結到 [style]。

重要：當將 [style]繫結到物件表示式時，物件的身份必須發生變化，Angular才會更新類別清單。 更新屬性而不改變物件身份沒有效果。

單樣式和多樣式繫結範例

nav-bar.component.ts

      
import {Component} from '@angular/core';
@Component({
  standalone: true,
  selector: 'app-nav-bar',
  template: ` <nav [style]="navStyle">
    <a [style.text-decoration]="activeLinkStyle">Home Page</a>
    <a [style.text-decoration]="linkStyle">Login</a>
  </nav>`,
})
export class NavBarComponent {
  navStyle = 'font-size: 1.2rem; color: cornflowerblue;';
  linkStyle = 'underline';
  activeLinkStyle = 'overline';
  /* . . . */
}

如果對同一樣式屬性有多個繫結，Angular會使用樣式優先級來確定要使用的繫結。

下表是各種樣式綁定語法的小結。

繫結型別 語法 輸入型別 範例輸入值
單樣式繫結 [style.width]="width" string undefined
帶單位的單樣式繫結 [style.width.px]="width" number undefined
多樣式繫結 [style]="styleExpression" string "width: 100px; height: 100px"
多樣式繫結 [style]="styleExpression" Record undefined

樣式優先級

一個 HTML 元素可以將其 CSS 類別清單和樣式值繫結到多個源（比如，來自多個指令的宿主繫結）。
