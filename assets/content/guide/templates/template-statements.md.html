範本語句

edit

範本語句是可在 HTML 中用於回應使用者事件的方法或屬性。使用範本語句，你的應用可以透過諸如顯示動態內容或送出表單之類別的動作吸引使用者。

在以下範例中，範本語句 deleteHero() 出現在 = 號右側的引號中，(event)="statement"。

當用戶單擊 Delete hero 按鈕時，Angular 就會呼叫元件類別中 deleteHero() 方法。

可以將範本語句與元素、元件或指令一起使用以回應事件。

有幫助的: 回應事件是 Angular 單向資料流的一個方面。你可以在單個事件迴圈中更改應用中的任何內容。

語法

與 範本表示式類似，範本語句使用一種看起來像 JavaScript 的語言。然而，範本語句的解析器與範本表示式的解析器不同。此外，範本語句解析器特別支援基本賦值（ =）和用分號（ ;）的鏈式呼叫。

不允許使用以下 JavaScript 和範本表示式語法：

  • new
  • 遞增和遞減運算符 ++ 和 --
  • 賦值運算符，比如 += 和 -=
  • 按位運算符，比如 | 和 &
  • 此 通道運算子

語句上下文

語句具有上下文 - 也就是語句所屬應用中的特定部分。

語句只能引用語句上下文中的內容，通常是元件實例。比如，(click)="deleteHero()" 中的 deleteHero() 就是下面程式碼段落中的元件方法之一。

實踐 詳情
簡潔性 使用方法呼叫或基本屬性賦值以保持範本語句的簡潔。
在上下文中工作 範本語句的上下文可以是元件類別實例或範本。因此，範本語句不能引用全域命名空間中的任何內容，如 window或 document。例如，範本語句不能呼叫 console.log()或 Math.max()。
