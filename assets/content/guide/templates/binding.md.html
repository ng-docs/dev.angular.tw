繫結動態文字、屬性（Property）和屬性（Attribute）

edit

在 Angular 中，繫結在元件的範本和資料之間建立動態連線。這種連線確保了元件資料的更改會自動更新渲染後的範本。

使用文字內嵌渲染動態文字

你可以在範本中使用雙花括號繫結動態文字，這告訴 Angular 它負責處理花括號內的表示式，並確保它被正確更新。這被稱為文字內嵌。

      
@Component({  template: `    <p>Your color preference is {{ theme }}.</p>  `,  ...})export class AppComponent {  theme = 'dark';}

在這個範例中，當代碼片段渲染到頁面上時，Angular 會將 {{ theme }} 替換為 dark。

      
<!-- Rendered Output --><p>Your color preference is dark.</p>

除了在首次渲染時求值表示式之外，當表示式的值發生變化時，Angular 還會更新渲染後的內容。

繼續以主題範例為例，如果使用者在頁面載入後點擊一個按鈕，將 theme 的值更改為 'light'，則頁面會相應地更新為：

      
<!-- Rendered Output --><p>Your color preference is light.</p>

你可以在通常在 HTML 中編寫文字的任何地方使用文字內嵌。

所有表示式的值都會被轉換為字串。物件和數組會使用值的 toString 方法進行轉換。

繫結動態屬性（Property）和屬性（Attribute）

Angular 支援使用方括號將動態值繫結到物件屬性（Property）和 HTML 屬性（Attribute）。

你可以繫結到 HTML 元素的 DOM 實例、元件實例或 指令實例上的屬性（Property）。

原生元素屬性（Property）

每個 HTML 元素都有一個對應的 DOM 表示形式。例如，每個 <button> HTML 元素都對應於 DOM 中 HTMLButtonElement 的一個實例。在 Angular 中，你可以使用屬性（Property）繫結將值直接設定到元素的 DOM 表示形式。

      
<!-- Bind the `disabled` property on the button element's DOM object --><button [disabled]="isFormValid">Save</button>

在這個範例中，每次 isFormValid 更改時，Angular 都會自動設定 HTMLButtonElement 實例的 disabled 屬性（Property）。

元件和指令屬性（Property）

當元素是 Angular 元件時，你可以使用屬性（Property）繫結，使用相同的方括號語法來設定元件的輸入屬性（Property）。

      
<!-- Bind the `value` property on the `MyListbox` component instance. --><my-listbox [value]="mySelection" />

在這個範例中，每次 mySelection 更改時，Angular 都會自動設定 MyListbox 實例的 value 屬性（Property）。

你也可以繫結到指令屬性（Property）。

      
<!-- Bind to the `ngSrc` property of the `NgOptimizedImage` directive  --><img [ngSrc]="profilePhotoUrl" alt="The current user's profile photo">

屬性（Attribute）

當你需要設定沒有對應 DOM 屬性（Property）的 HTML 屬性（Attribute）時（例如 ARIA 屬性（Attribute）或 SVG 屬性（Attribute）），你可以使用 attr. 字首將屬性（Attribute）繫結到範本中的元素。

      
<!-- Bind the `role` attribute on the `<ul>` element to the component's `listRole` property. --><ul [attr.role]="listRole">

在這個範例中，每次 listRole 更改時，Angular 都會透過呼叫 setAttribute 自動設定 <ul> 元素的 role 屬性（Attribute）。

如果屬性（Attribute）繫結的值為 null，Angular 會通過呼叫 removeAttribute 刪除該屬性（Attribute）。

在屬性（Property）和屬性（Attribute）中使用文字內嵌

你還可以在屬性（Property）和屬性（Attribute）中使用文字內嵌語法，方法是使用雙花括號語法而不是屬性（Property）或屬性（Attribute）名稱周圍的方括號。當使用此語法時，Angular 會將賦值視為屬性（Property）繫結。

      
<!-- Binds a value to the `alt` property of the image element's DOM object. --><img src="profile-photo.jpg" alt="Profile photo of {{ firstName }}" >

要使用文字內嵌語法繫結到 Attribute，請在 Attribute 名稱前加上 attr. 字首

      
<button attr.aria-label="Save changes to {{ objectType }}">

CSS 類別和樣式屬性（Property）繫結

Angular 支援將 CSS 類別和 CSS 樣式屬性（Property）繫結到元素的附加屬性。

CSS 類別

你可以建立一個 CSS 類別繫結，以根據繫結值是 真值與否，有條件地在元素上新增或刪除 CSS 類別。

      
<!-- When `isExpanded` is truthy, add the `expanded` CSS class. --><ul [class.expanded]="isExpanded">

你也可以直接繫結到 class 屬性（Property）。Angular 接受三種型別的值：

`class` 值的描述 TypeScript 型別
一個包含一個或多個 CSS 類別（以空格分隔）的字串 string
CSS 類別字串的陣列 string[]
一個物件，其中每個屬性（Property）名稱都是一個 CSS 類別名稱，每個對應的值根據真值與否決定是否將該類別應用於元素。 Record<string, any> 
      
@Component({  template: `    <ul [class]="listClasses"> ... </ul>    <section [class]="sectionClasses"> ... </section>    <button [class]="buttonClasses"> ... </button>  `,  ...})export class UserProfile {  listClasses = 'full-width outlined';  sectionClasses = ['expandable', 'elevated'];  buttonClasses = {    highlighted: true,    embiggened: false,  };}

上面的範例渲染以下 DOM：

      
<ul class="full-width outlined"> ... </ul><section class="expandable elevated"> ... </section><button class="highlighted"> ... </button>

Angular 忽略任何不是有效 CSS 類別名稱的字串值。

當使用靜態 CSS 類別、直接繫結 class 以及繫結特定類別時，Angular 會智慧地將所有類別組合在渲染結果中。

      
@Component({  template: `<ul class="list" [class]="listType" [class.expanded]="isExpanded"> ...`,  ...})export class Listbox {  listType = 'box';  isExpanded = true;}

在上面的範例中，Angular 使用所有三個 CSS 類別渲染 ul 元素。

      
<ul class="list box expanded">

Angular 不保證渲染元素上 CSS 類別的任何特定順序。

當將 class 繫結到陣列或物件時，Angular 使用三等號運算符 (===) 將先前的值與當前值進行比較。當你修改這些值時，必須建立一個新的物件或陣列實例，以便 Angular 應用任何更新。

如果一個元素對同一個 CSS 類別有多個繫結，Angular 會按照其樣式優先級順序來求解衝突。

CSS 樣式屬性（Property）

你也可以直接繫結到元素上的 CSS 樣式屬性（Property）。

      
<!-- Set the CSS `display` property based on the `isExpanded` property. --><section [style.display]="isExpanded ? 'block' : 'none'">

你可以進一步為接受單位的 CSS 屬性（Property）指定單位。

      
<!-- Set the CSS `height` property to a pixel value based on the `sectionHeightInPixels` property. --><section [style.height.px]="sectionHeightInPixels">

你還可以在一個繫結中設定多個樣式值。Angular 接受以下型別的值：

`style` 值的描述 TypeScript 型別
一個包含零個或多個 CSS 宣告的字串，例如 "display: flex; margin: 8px"。 string
一個物件，其中每個屬性（Property）名稱都是一個 CSS 屬性（Property）名稱，每個對應的值都是該 CSS 屬性（Property）的值。 Record<string, any> 
      
@Component({  template: `    <ul [style]="listStyles"> ... </ul>    <section [style]="sectionStyles"> ... </section>  `,  ...})export class UserProfile {  listStyles = 'display: flex; padding: 8px';  sectionStyles = {    border: '1px solid black',    'font-weight': 'bold',  };}

上面的範例渲染出如下 DOM。

      
<ul style="display: flex; padding: 8px"> ... </ul><section style="border: 1px solid black; font-weight: bold"> ... </section>

當把 style 繫結到一個物件時，Angular 會用三等號運算符 (===) 來比較前一個值和當前值。當你要修改這些值以便 Angular 應用任何更新時，必須建立一個新的物件實例。

如果一個元素針對同一個 style 屬性有多個繫結，Angular 會按照其 style 優先級順序來解決衝突。
