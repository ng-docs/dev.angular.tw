调试测试代码

edit

如果你的测试没能如预期般工作，可以在浏览器中查看和调试它们。

在浏览器中调试这些测试规约的方式与调试应用时相同。

  1. 显示 Karma 浏览器窗口。 如果你需要帮助，请参阅 建立测试环境。

  2. 单击DEBUG按钮以打开新的浏览器选项卡并重新运行测试。

  3. 打开浏览器的开发者工具。在 Windows 上，按 Ctrl-Shift-I。在 macOS 上，按 Command-Option-I。

  4. 选择Sources部分。

  5. 按 Control/Command-P，然后开始键入测试文件的名称以打开它。

  6. 在测试中设置一个断点。

  7. 刷新浏览器，注意它会在这个断点处停下来。
