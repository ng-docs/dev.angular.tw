除錯測試

edit

如果你的測試沒有像預期的那樣工作，你可以在瀏覽器中檢查和除錯它們。

在瀏覽器中除錯規範，就像除錯應用一樣。

  1. 顯示 Karma 瀏覽器視窗。 如果你在這一步需要幫助，請參閱 建立測試環境。

  2. 點選 DEBUG 按鈕以開啟新的瀏覽器標籤頁並重新執行測試。

  3. 開啟瀏覽器的 開發者工具。在 Windows 上，按 Ctrl-Shift-I。在 macOS 上，按 Command-Option-I。

  4. 選擇 Sources 部分。

  5. 按 Control/Command-P，然後開始輸入你的測試檔案的名稱以開啟它。

  6. 在測試中設定一個斷點。

  7. 重新整理瀏覽器，並注意它是如何在斷點處停止的。
