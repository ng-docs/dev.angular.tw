在範本中使用通道

edit

要應用一個 pipe，請在範本表示式中使用 pipe 運算子（ |），如下列程式碼範例所示。

app.component.html

      
<p>The hero's birthday is {{ birthday | date }}</p>

元件的 birthday 值透過 pipe 運算子（ |）傳遞給 DatePipe，其 pipe 名稱為 date。此 pipe 以預設格式渲染日期，如 2023 年 4 月 07 日。

app.component.ts

      
import { Component } from '@angular/core';
import { DatePipe } from '@angular/common';
@Component({
  standalone: true,
  templateUrl: './app.component.html',
  imports: [DatePipe],
})
export class AppComponent {
  birthday = new Date();
}

Pipe 的附加引數

Pipe 可以接收附加引數來設定轉換。引數可以是可選的或必要的。

例如， date pipe 接受控制日期顯示格式的可選引數。要指定引數，請在 pipe 名稱後加上冒號（ :）和引數值（格式）。

app.component.html

      
<p>The hero's birthday is in {{ birthday | date:'yyyy' }}</p>

Pipe 也可以接收多個引數。你可以透過冒號（ :）分隔這些引數來傳遞多個引數。例如， date pipe 接受第二個可選引數來控制時區。

app.component.html

      
<p>The current time is: {{ currentTime | date:'hh:mm':'UTC' }}</p>

這將以 UTC 時區顯示當前時間（如 10:53）。

鏈式呼叫 pipe

你可以連線多個 pipe，使一個 pipe 的輸出成為下一個 pipe 的輸入。

以下範例將日期傳遞給 DatePipe，然後將結果傳遞給 UpperCasePipe pipe。

app.component.html

      
<p>The hero's birthday is {{ birthday | date }}</p>
<p>The hero's birthday is in {{ birthday | date:'yyyy' | uppercase }}</p>
