在模板中使用管道

edit

要应用一个 pipe，请在模板表达式中使用 pipe 操作符（ |），如下列代码示例所示。

app.component.html

      
<p>The hero's birthday is {{ birthday | date }}</p>

组件的 birthday 值通过 pipe 操作符（ |）传递给 DatePipe，其 pipe 名称为 date。此 pipe 以默认格式渲染日期，如 2023 年 4 月 07 日。

app.component.ts

      
import { Component } from '@angular/core';
import { DatePipe } from '@angular/common';
@Component({
  standalone: true,
  templateUrl: './app.component.html',
  imports: [DatePipe],
})
export class AppComponent {
  birthday = new Date();
}

Pipe 的附加参数

Pipe 可以接收附加参数来配置转换。参数可以是可选的或必要的。

例如， date pipe 接受控制日期显示格式的可选参数。要指定参数，请在 pipe 名称后加上冒号（ :）和参数值（格式）。

app.component.html

      
<p>The hero's birthday is in {{ birthday | date:'yyyy' }}</p>

Pipe 也可以接收多个参数。你可以通过冒号（ :）分隔这些参数来传递多个参数。例如， date pipe 接受第二个可选参数来控制时区。

app.component.html

      
<p>The current time is: {{ currentTime | date:'hh:mm':'UTC' }}</p>

这将以 UTC 时区显示当前时间（如 10:53）。

链式调用 pipe

你可以连接多个 pipe，使一个 pipe 的输出成为下一个 pipe 的输入。

以下示例将日期传递给 DatePipe，然后将结果传递给 UpperCasePipe pipe。

app.component.html

      
<p>The hero's birthday is {{ birthday | date }}</p>
<p>The hero's birthday is in {{ birthday | date:'yyyy' | uppercase }}</p>
