瞭解通道

edit

使用通道來轉換字串、貨幣金額、日期和其他資料以便顯示。

什麼是通道

通道是在範本中使用的簡單函式，用於接受一個輸入值並回傳一個轉換後的值。通道很有用，因為你可以在整個應用中使用它們，而每個通道只需宣告一次。 例如，你可以使用通道將日期顯示為 1988年4月15日，而不是原始的字串格式。

你可以建立自己的自定義通道，以在範本中暴露可複用的轉換。

內建通道

Angular 為典型的資料轉換提供了內建的通道，包括國際化的轉換（i18n），它使用本地化訊息來格式化資料。資料格式化常用的內建通道如下：

  • DatePipe：根據本地環境中的規則格式化日期值。
  • UpperCasePipe：把文字全部轉換成大寫。
  • LowerCasePipe：把文字全部轉換成小寫。
  • CurrencyPipe：把數字轉換成貨幣字串，根據本地環境中的規則進行格式化。
  • DecimalPipe：根據語言環境規則，將一個數字轉換為帶小數點的字串。
  • PercentPipe：把數字轉換成百分比字串，根據本地環境中的規則進行格式化。
  • AsyncPipe：訂閱和取消訂閱一個非同步源，如可觀察者。
  • JsonPipe：將元件物件屬性以 JSON 格式顯示到螢幕上以便除錯。

注意：有關內建通道的完整清單，請參閱 通道 API 文件。 要了解更多關於通道在國際化（i18n）工作中的使用，請參閱 基於語言環境格式化資料。
