模板表达式中的管道优先级

edit

有时你需要在将某个选择传递给管道之前，根据某个条件在两个值之间进行选择。 你可以在模板中使用 JavaScript 三元运算符 ( ?:) 来做出选择。

注意！管道操作符的优先级高于 JavaScript 三元运算符 ( ?:)。

如果你像从左到右求值那样简单地编写表达式，可能会对结果感到惊讶。例如：

      
condition ? a : b | pipe

解析为

      
condition ? a : (b | pipe)

b 的值通过 pipe； a 的值 不会。

如果你希望 pipe 应用于三元表达式的结果，请将整个表达式包裹在括号中。例如：

      
(condition ? a : b) | pipe

一般来说，你应该始终使用括号，确保 Angular 按照你的意图求值表达式。
