樣板表示式中的通道優先級

edit

有時你需要在將某個選擇傳遞給通道之前，根據某個條件在兩個值之間進行選擇。 你可以在樣板中使用 JavaScript 三元運算符 ( ?:) 來做出選擇。

注意！通道運算子的優先級高於 JavaScript 三元運算符 ( ?:)。

如果你像從左到右求值那樣簡單地編寫表示式，可能會對結果感到驚訝。例如：

      
condition ? a : b | pipe

解析為

      
condition ? a : (b | pipe)

b 的值透過 pipe； a 的值 不會。

如果你希望 pipe 應用於三元表示式的結果，請將整個表示式包裹在括號中。例如：

      
(condition ? a : b) | pipe

一般來說，你應該始終使用括號，確保 Angular 按照你的意圖求值表示式。
