导入和使用组件

edit

提示：本指南假定你已经阅读了 基本指南。如果你是 Angular 的新手，请先阅读该指南。

Angular支持两种方式将组件提供给其他组件：作为独立的组件或在 NgModule中。

独立组件

一个 独立组件是在其组件元数据中设置 standalone: true的组件。独立组件直接导入在其模板中使用的其他组件、指令和管道：

      
@Component({
  standalone: true,
  selector: 'profile-photo',
})
export class ProfilePhoto { }
@Component({
  standalone: true,
  imports: [ProfilePhoto],
  template: `<profile-photo />`
})
export class UserProfile { }

独立组件可以直接导入到其他独立组件中。

Angular团队建议所有新开发都使用独立组件。

NgModules

在使用独立组件之前，Angular 代码使用 NgModule 作为导入和使用其他组件的机制。详情请参见完整的 NgModule 指南。
