匯入和使用元件

edit

提示：本指南假定你已經閱讀了 基本指南。如果你是 Angular 的新手，請先閱讀該指南。

Angular支援兩種方式將元件提供給其他元件：作為獨立的元件或在 NgModule中。

獨立元件

一個 獨立元件是在其元件元資料中設定 standalone: true的元件。獨立元件直接匯入在其範本中使用的其他元件、指令和通道：

      
@Component({
  standalone: true,
  selector: 'profile-photo',
})
export class ProfilePhoto { }
@Component({
  standalone: true,
  imports: [ProfilePhoto],
  template: `<profile-photo />`
})
export class UserProfile { }

獨立元件可以直接匯入到其他獨立元件中。

Angular團隊建議所有新開發都使用獨立元件。

NgModules

在使用獨立元件之前，Angular 程式碼使用 NgModule 作為匯入和使用其他元件的機制。詳情請參見完整的 NgModule 指南。
