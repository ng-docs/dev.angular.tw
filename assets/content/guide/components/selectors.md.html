组件选择器

edit

提示：本指南假定你已经阅读了 基本指南。如果你是 Angular 的新手，请先阅读该指南。

每个组件都定义了一个 CSS 选择器，用于确定如何使用该组件：

      
@Component({
  selector: 'profile-photo',
  ...
})
export class ProfilePhoto { }

通过在 其他组件的模板中创建匹配的HTML元素来使用组件：

      
@Component({
  template: `
    <profile-photo />
    <button>Upload a new profile photo</button>`,
  ...,
})
export class UserProfile { }

Angular在编译时静态匹配选择器。在运行时改变DOM，无论是通过Angular绑定还是DOM API，都不会影响渲染的组件。

一个元素只能匹配一个组件选择器。如果多个组件选择器匹配一个元素，Angular会报错。

组件选择器是区分大小写的。

选择器类型

Angular 在组件选择器中支持一组 基本 CSS 选择器类型：

选择器类型 说明 例子
类型选择器 根据HTML标签名称或节点名称匹配元素。 profile-photo
属性选择器 根据HTML属性的存在以及可选的属性确切值匹配元素。 [dropzone] [type="reset"]
类选择器 根据CSS类的存在匹配元素。 .menu-item

对于属性值，Angular 支持使用等号（ =）运算符匹配确切的属性值。Angular 不支持其他属性值运算符。

Angular 组件选择器不支持组合器，包括 后代组合器或 子组合器。

Angular 组件选择器不支持指定 命名空间。

:not 伪类

Angular 支持 伪类 :not。你可以将这个伪类附加到任何其他选择器上，以缩小组件选择器匹配的元素范围。例如，你可以定义一个 [dropzone]属性选择器并阻止匹配 textarea元素：

      
@Component({
  selector: '[dropzone]:not(textarea)',
  ...
})
export class DropZone { }

Angular 不支持组件选择器中的其他伪类或伪元素。

组合选择器

你可以通过连接它们来组合多个选择器。例如，你可以匹配指定了 type="reset"的 <button>元素：

      
@Component({
  selector: 'button[type="reset"]',
  ...
})
export class ResetButton { }

你也可以使用逗号分隔的列表定义多个选择器：

      
@Component({
  selector: 'drop-zone, [dropzone]',
  ...
})
export class DropZone { }

Angular 为匹配列表中 任何选择器的每个元素创建一个组件。

选择选择器

绝大多数组件应使用自定义元素名称作为其选择器。所有自定义元素名称应包含一个连字符（如 HTML 规范 所述）。默认情况下，如果遇到不匹配任何可用组件的自定义标签名称，Angular 会报告错误，从而防止由于组件名称拼写错误导致的 bug。

详情见 高级组件配置，了解在 Angular 模板中使用 原生自定义元素。

选择器前缀

Angular 团队建议为项目内定义的所有自定义组件使用简短、一致的前缀。例如，如果你使用 Angular 构建 YouTube，可以为组件添加前缀 yt-，比如 yt-menu、 yt-player 等。这样命名空间化选择器可以立即明确某个组件的来源。默认情况下，Angular CLI 使用 app-。

Angular 使用 ng 选择器前缀用于其自身框架 API。切勿将 ng 用作自定义组件的选择器前缀。

何时使用属性选择器

当你希望在标准原生元素上创建组件时，可以考虑使用属性选择器。例如，如果你想创建一个自定义按钮组件，可以通过使用属性选择器利用标准 <button> 元素：

      
@Component({
  selector: 'button[yt-upload]',
   ...
})
export class YouTubeUploadButton { }

这种方法允许组件使用者直接使用该元素的所有标准 API，无需额外工作。这对于 ARIA 属性（如 aria-label）尤其有价值。

当遇到不匹配任何可用组件的自定义属性时，Angular 不会报告错误。使用属性选择器的组件时，使用者可能会忘记导入组件或其 NgModule，导致组件无法渲染。详情见 导入和使用组件。

定义属性选择器的组件应使用小写的、短横线连接的属性。你可以遵循上述相同的前缀建议。
